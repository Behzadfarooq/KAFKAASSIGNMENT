BBEdit Lite
Developer: Bare Bones Software
Latest release: 6.1.2 / 2001-10-02
OS: Mac OS, Mac OS X
Use: text editor
License: freeware
Website: ftp.barebones.com/pub/freeware/

BBEdit Lite is a text editor by Bare Bones Software that is a stripped down or "lightweight" version of their flagship product BBEdit. BBEdit Lite has many of the same basic features as BBEdit such as regular expressions, plug-in architecture and the same text engine. Lite lacked the power user tools such as syntax highlighting, command line shell, or FTP access. Several years after development stopped BBEdit Lite was replaced by TextWrangler after Bare Bones made it free.
