ISO 3166-2:BB

ISO 3166-2:BB is an ISO standard which defines geocodes. It is the subset of ISO 3166-2 that applies to Barbados. Each two-part code consists of the ISO 3166-1 code for Barbados (BB) and a single character, hyphenated together.

[edit] Codes

BB-A Saint Andrew
BB-C Christ Church
BB-E Saint Peter
BB-G Saint George
BB-J Saint John
BB-L Saint Lucy
BB-M Saint Michael
BB-O Saint Joseph
BB-P Saint Philip
BB-S Saint James
BB-T Saint Thomas
