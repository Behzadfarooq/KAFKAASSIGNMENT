[edit] RfA

Hi Bbatsell - I'm glad to hear you are considering the prospects of adminship. In my experience, objections to RfA can arise for lots of different reasons. Some of the common objections are:

  1. User has too few edits - This is rarely a problem at 2000+ edits, so you should be fine
  2. User has been here too little - You've been here a year so that should be fine. There could be a question as to why you become so much more active recently - so you should be prepared to answer that question
  3. User has too few edits to _____ space. In your case, you seem to focus mainly on the articles, and some people like to see administrators be involved in policy discussions
  4. User has done horrible things or been controversial - I haven't seen any problems here for you, but I've not gone through your entire history
  5. User has not done dirty work, such as cleanup of vandalism - clearly not a problem for you since you are a committed vandal fighter.
  6. User has nominated themselves, or campaigned to get votes for adminship - easy enough to avoid doing
  7. User gives poor or evasive answers to the questions they are asked in the RfA process
  8. User has not set an e-mail address
  9. User does not have a User page, or there user page is offensive for some reason - You user page looks great, unless someone complains you have too many user boxes - but you can't please everyone!

I think you would be an excellent admin. You probably have already done this, but you should look over the administrators' reading list and how-to guide, as well as the guide to requests for adminship. If you decide you would like to go forward, I would be honored to nominate you. Johntex\talk 18:16, 30 January 2006 (UTC)
