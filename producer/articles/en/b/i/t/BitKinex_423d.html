[edit] Overview

Bitkinex is a multiprotocol FTP client, supporting FTP, SFTP, HTTP (WebDAV). The FTP features on one-click commands list, multipart download and multi thread browse. The WebDAV client features messenger-like user interface, fast multithreaded and multipart downloads, server-to-server transfers and SSL.

[edit] Development

Bitkinex was originally a freeware named morgul, with the same logo but in grayscale, featuring on suspendable ftp upload.
