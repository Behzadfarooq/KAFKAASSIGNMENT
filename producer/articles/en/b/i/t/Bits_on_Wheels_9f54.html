Bits on Wheels
Developer: bitsonwheels.com
Latest release: 1.0.6 / November 19th, 2005
OS: Mac OS X
Use: BitTorrent client
License: Freeware
Website: bitsonwheels.com

Bits on Wheels is a Mac OS X BitTorrent client. It features a live 3D view of the active swarm, similar to the "Swarm" view in Azureus. Bits on Wheels was the first native Mac OS X BitTorrent client. It is written in Objective-C and Cocoa. It is currently freeware. The latest version is 1.0.6. The project is presumed dead.
