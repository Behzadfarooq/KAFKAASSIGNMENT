This code is in the trash can - it is not flexible, and in the template call it is hard to remember what comes after what. -- Boris 06:42, 30 December 2005 (UTC)

This is a template that i (for sure), and probably others, are going to use for the articles for the small biological molecules. To call the template i need to write the following code:

{{BioChemInfoBox|v1|v2|v3|v4|v5|v6|v7|v8|v9|v10|v11|v12|v13|v14|v15|v16}}

where, v1,v2...v16 are the values for the template parameters.

v1 - the name of the image file including the extension

v2 - the title of the image

v3 - the background color of the header. Any color format can be used [color_name, #XXYYZZ or rgb(XXX,YYY,ZZZ)].

v4 - the border color. Any color format can be used [color_name, #XXYYZZ or rgb(XXX,YYY,ZZZ)].

v5 - the width of image area - any size format can be used, i preffer to use the "px" format as the sizes of my images are in pixels. Also i use a standard "320" size as most of the images i have are with this width, the extra "6" in the width of the table - 326px - are to compensate for the "3px" padding space that i have on both side of the image.

v6 - the complete UIPAC name, if not know then "?". Be caefull with the UIPAC names, some of them might have several [[, {{, [[[, {{{, ]], }}, }}}, or ]]] brakes that will mess things up, use ((, (((, )), or ))) instead.

v7 - the abbreviations. If a compound has two of them a short one and a long one, the short comes first followed by "," and then the long one - eaxh α-amino acid has a one-letter and three-letter abbreviations so for Alanine the value of v7 would be "A,Ala". If not know then "?".

v8 - the synonyms. If there more than one they shoud be orginized alphabetically (the Greek alphabet comes before the Latin one) and separated by "<br />". If not know then "?".

v9 - the chemical formula. The following format should be use - Cn1Hn2Nn3On4Pn5Sn6. If not know then "?".

v10 - Molecular mass. If not know then "?".

v11 - the number of the H-bonds in which the molecule acts as a donor. If not know then "?".

v12 - the number of the H-bonds in which the molecule acts as an acceptor. If not know then "?".

v13 - the number of the tautomer forms. If not know then "?".

v14 - the values for "Acid dissociation constant", if there is more than one then separated by "<br />". If not know then "?".

v15 - The CAS registry number. If not know then "?".

v16 - The EINECS number. If not know then "?".
