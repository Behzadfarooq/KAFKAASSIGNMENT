This page is a copy of many of the religion-related userboxes as they existed on Saturday May 13. The hope has to preserve the most non-offensive of them for those users who would like to use them.

If you would like to create one for a denomination that I don't have here, please feel free to do so, HOWEVER, please make it positive. In other words, "This user is a member of denomination X" is good, but "This user left denomination X" or "This user does not agree with denomination X" isn't good. This is not a place for engaging in a religious debate.
