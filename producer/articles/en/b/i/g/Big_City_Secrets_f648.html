Big City Secrets is the first studio album by Joseph Arthur, released by Real World Records on March 11, 1997.

[edit] Artists

  • Joseph Arthur: Guitars, Vocals - all tracks, Harmonica - track 2, Highly-Strung bass - track 5.
  • Ron Aslan: Drum loop - track 7.
  • Roger Beaujolais: Vibes - track 1.
  • Martyn Barker: Treated Tubular Kit - track 1, Drums - track 2, 3, 5, 6, 9, 10, 11, Caxixi - track 4, Military Snare - track 7, Bondage Snare - track 8, Scaffolphoneum - track 9, Birimbau - track 11, Cow Gong - track 11, Percussion - track 12.
  • Markus Dravs: Keyboard instruments - track 1, Desert Storm - track 1, Gravel Percussion - track 1, Corn Horn - track 2, Drum Programming - track 4, 7, 9, Lesley Accordion - track 7, Low Drum - track 8, Sheet Percussion - track 8.
  • Nigel Eaton: Hurdy gurdy - track 3.
  • Simon Edwards: bass - all tracks, Piano - track 4, Coal Bass - track 7, Additional Guitars - track 7, String Fossil Bass - track 8, Maze Bass - track 9, Sas - track 9, Birimbau - track 11, Mellotron - track 11, Persian Fretless - track 11, Filter Bass - track 12, Harmonic Missiles - track 12.
  • Brian Eno: Backing vocals - track 2.
  • Peter Gabriel: Backing vocals - track 2.
  • Nick Plytas: Mellotron - track 7.
  • Ashley Slater: Trombone - track 6, 8.
