The code to the left of this text should be copied and pasted to an article where it is needed. This template is designed for seasons of Big Brother that have ended. For current seasons, please use {{Big Brother housemates}}.

What is written after each "=" affects what is displayed on an article when it is viewed.

seriesname
The name of the Big Brother series should be typed here and linked to the relevant article, i.e. [[Big Brother (Australia)|Big Brother Australia]].
seriesnum
The number of the season should be put here, i.e. 2001, or Series 1 (2001).
image
The season's logo should go here. The image name should be typed here, with Image:, but without square brackets or other image modifications, i.e. Image:BB06.JPG. Images are automatically sized to 137px.
imagebg
A background colour for the image can be defined here, but should only be used if the image has a solid background colour.
hmxx
This is where a housemate's name should be written. Names are not automatically linked to anything, so these need to be created manually.
hmxx-enter
The day that a housemate entered the Big Brother House should be here, written as Day x. If the housemate previously left and re-entered the House, it should be written as Day xx<br />Day z or Days x/z.
hmxx-exit
The day that a housemate left the House should be written here as Day x. If the housemate previously left and re-entered the House, it should be written as Day xx<br />Day z or Days x/z.
hmxx-stat
The current status of a housemate goes here.
  • winner - Winner
  • evic - Evicted
  • three - The third colour, this should be used for housemates that voluntarily left the House before the season ended.
  • five - The fifth colour, this should be used for housemates that involuntarily left the House, most likely due to being removed by Big Brother.
  • none - The housemate is not affected by anything. It is imperative that something is in the -stat field, otherwise the box will appear as white.
legendwinner
By default, this appears as "Winner" on an article. This can be changed if necessary.
legendrunner
This is for any runners-up, and should generally say "Runner-up".
legendfive
This is for housemates that involuntarily left the House before the season ended, and should say "Removed" or "Ejected", unless another term is used by Big Brother.
legend3
For housemates that voluntarily left the House before the season ended.
