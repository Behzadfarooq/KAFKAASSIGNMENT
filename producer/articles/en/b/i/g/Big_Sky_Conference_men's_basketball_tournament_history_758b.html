Big Sky Conference: year location champions

  • 1976: Wildcat Gym, Ogden, Utah - Boise State
  • 1977: ISU Minidome, Pocatello, Idaho - Idaho State
  • 1978: Dahlberg Arena, Missoula, Montana - Weber State
  • 1979: Dee Events Center, Ogden - Weber State
  • 1980: same - Weber State
  • 1981: Kibbie Dome, Moscow, Idaho - Idaho
  • 1982: Kibbie Dome, Moscow, Idaho - Idaho
  • 1983: Centennial Coliseum, Reno, Nevada - Weber State
  • 1984: Dee Events Center, Ogden, Utah - Nevada
  • 1985: BSU Pavilion, Boise, Idaho - Nevada
  • 1986: Lawlor Events Center, Reno, Nevada - Montana State
  • 1987: Walkup Skydome, Flagstaff, Arizona - Idaho State
  • 1988: Brick Breeden Fieldhouse, Bozeman, Montana - Boise State
  • 1989: BSU Pavilion Boise, Idaho - Idaho
  • 1990: BSU Pavilion Boise, Idaho- Idaho
  • 1991: Dahlberg Arena - Montana
  • 1992: same - Montana
  • 1993: Kibbie Dome - Boise State
  • 1994: BSU Pavilion - Boise State
  • 1995: Dee Events Center - Weber State
  • 1996: Brick Breeden Fieldhouse - Montana State
  • 1997: Walkup Skydome - Montana
  • 1998: Walkup Skydome - Northern Arizona
  • 1999: Dee Events Center - Weber State
  • 2000: Dahlberg Arena - Northern Arizona
  • 2001: Matadome, Northridge, California - Cal State Northridge
  • 2002: Brick Breeden Fieldhouse - Montana
  • 2003: first round games at campus sites; rest at Dee Events Center - Weber State
  • 2004: first round games at campus sites; rest at Reese Court, Cheney, Washington- Eastern Washington
  • 2005: first round games at campus sites; rest at Memorial Coliseum, Portland, Oregon -Montana
  • 2006: first round games at campus sites; rest at Walkup Skydome - Montana
  • 2007: campus sites; championship at Dee Events Center - Weber State
