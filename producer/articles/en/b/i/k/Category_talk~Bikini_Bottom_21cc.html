The Category:Bikini Bottom page is part of WikiProject SpongeBob SquarePants, an attempt to build a more detailed guide to the SpongeBob SquarePants universe. You may edit this page, or visit the project page, where you can join and contribute to the project and participate in discussion for the project. Below, are the objectives and guidelines for the project and SpongeBob SquarePants articles.

  • Episode Pages:
  • Infoboxes need to be completed on each episode article. All infoboxes have been completed on the season one and two articles. See here.
  • A screenshot of each title card for each episode needs to be complete on each episode article to improve it. Only the season 5 episodes (except for Rise and Shine and Fungus Among Us) are missing title cards.
  • Please try to help incorporate all trivia into the rest of the articles.
  • Please incorperate all SpongeBob SquarePants episode and character quotes to Wikiquote.
  • Expansion - Many articles need expanding, please try to help us with these. These include game articles (See games section on template and book information here.
  • Classify all SpongeBob articles on the Assessment scale.
  • See unassessed importance and unassessed quality categories.

Guidelines - Click "Show" to display.
Project Guidelines are listed here.  
view • talk • edit
  • Layouts - Please use the following layouts for SpongeBob pages:
  • Episode article layout
  • Character article layout
  • Images - Make sure all SpongeBob-related images contain copyright tags and are categorized with this category: [[Category:SpongeBob SquarePants images]]
  • Categorization - Make sure all SpongeBob-related articles are categorized.
  • Episode Names - When creating a new episode, page, if the article exists as something else, name the article Episode Name (SpongeBob SquarePants)
  • Moving Pages - When moving a page, if the page is already created, please do not copy and paste. Please fill out a form at Wikipedia:Requested moves.
  • Tags - If you see any tags on any SpongeBob article, (i.e. cleanup, wikify, etc.) please notify the WikiProject here. This will make it easier for project members to fix whatever problem there may be.
  • Trivia - Per WP:TRIVIA, trivia is not reccommended. Please try to incorporate relevent trivia into the rest of the article.
  • When describing something, please do not use words like "you will notice" or "if you..." etc. Also see Wikipedia:Avoid weasel words.
  • When writing a new article or making changes to an existing one, please adhere to the manual of style. For example, formal writing on Wikipedia is preferred. Therefore, the use of contractions should be avoided.
  • Writing about fiction - In-universe perspective should not be used when writing about fiction. In-universe perspective includes plots framed as biographies and the birth and death dates of fictional characters. Instead, use perspective of the real world.
