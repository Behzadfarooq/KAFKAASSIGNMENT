[edit] Userbox deletion

You can delete WP:GUS'd userboxes once transclusion is done via speedy deletion instead of "clogging" up the templates for deletion. There is a specific criteria that allows this for uncontroversial housekeeping (G6). I personally use the following code to speedy delete templates:

{{Db-g6}} Reason: Orphaned due to [[Wikipedia:Userbox migration]], all instances of transclusion moved to new location.

This helps speed up the process. Harryboyles 11:19, 14 October 2006 (UTC)

Hi, I'm already using {{db-g6}} (since a few days) for userfied ubx. However, there are some ubx that were up for deletion before or that have transclusion links on protected pages. I thought they might not fall under criteria for speedy deletion. --Bisco 11:26, 14 October 2006 (UTC)
