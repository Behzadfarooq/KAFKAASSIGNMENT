[edit] Suggestion

Can we create a template to use in edit summaries instead of having to type "Bad link repair ([[Wikipedia:Bad links|You can help!]])". TheLimbicOne(talk/cntrb) 02:35, 4 January 2006 (UTC)

I agree. It will only save a little time on each article, but it adds up. Usually I put it on my clipboard, but often while editing I end up having it replaced so it becomes a hassle. -- PS2pcGAMER 03:00, 4 January 2006 (UTC)
As a suggestion in the meantime, I use a text expander program when I'm doing bad link or disambiguation link repair. I only type 'blr' and it's replaced with the above edit summary. I'm a Mac user, but I'm pretty sure there are such programs for other operating systems. - ddlamb 21:51, 8 January 2006 (UTC)
I don't think the Mediawiki software expands templates in edit summaries...that's probably intentional, and I actually use double curly braces there all the time, to mean the literal text, not the template expansion. -- Beland 03:55, 3 April 2006 (UTC)
