[edit] PfD Footer section

To list a person for deletion: (This creates a separate debate page for each discussion, which prevents edit conflicts.)

  1. Go to the person you want to delete and insert {{subst:pfd}} at the top of that person's page.
  2. If the page is an advertisement, enclose any links in nowiki tags like this:

    <nowiki>[http://www.example.org Cheap Examples]</nowiki>

  3. Come back to this page and copy the following text:

    ====[[User:Person name|Person name]] &mdash; [[/Person name|Add to this discussion]]====
    {{Wikipedia:People for deletion/Person name}}

  4. Edit this page and paste the text below the last heading.
  5. Replace Person name in each line with the name of the person that you are proposing for deletion.
  6. Save page.
  7. Click the Discussion link and write why, according to our deletion policy, the page should be deleted.
  8. Save your edit.

Alternate Method (This adds the discussion to the bottom of the main page.)

  1. Go to the page you want to delete and insert {{subst:pfd}} at the top of that person's page.
  2. If the page is an advertisement, enclose any links in nowiki tags like this:

    <nowiki>[http://www.example.org Cheap Examples]</nowiki>

  3. Come back to this page and copy the following text:

    ====[[User:Person name|Person name]]====

  4. Edit this page and paste the text below the last heading.
  5. Replace Person name with the exact name of the person.
  6. Below the section header, write why, according to our deletion policy, the page should be deleted.
  7. Save your edit.
