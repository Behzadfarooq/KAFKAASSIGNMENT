[edit] Using

Backup Premium is an independent application compatible with Microsoft Windows 98/Me/NT/2000/XP/2003 Server. Once being installed and set up the program is automatically launched at the beginning of every Windows session and backups files according to a user-defined schedule.

[edit] Supported backup storages

Backup Premium supports almost all types of widely used data storages: external and internal hard drives, network drives (including SAN, NAS, and LAN locations), all types of flash memory (including USB flash drives), removable media devices (floppy disks, ZIP disks, JAZ, optical etc), remote FTP servers, secure SFTP and FTP SSL servers.

[edit] Main backup options

Automatic Backup Often backup process for a large amount of data requires additional system resources and can take a lot of time. In this case some applications could run slower than usually and the user would be forced to interrupt his work. The automatic scheduler allows avoiding such problems by launching backup process during the time when the computer is idle. Also automatic backup scheduling is a way to ensure that the data is being backed up on a regular basis without user interference, which is especially important for critical and frequently changing data.

Incremental Backup The program backs up only new or modified files and doesn't overwrite the existing files if they haven't been changed since the last back up session. Using this option helps to reduce significantly the back up time and still keep all backed up files up-to-date.

ZIP compression Using built-in ZIP compression increases backup speed and saves storage disk space. For data compression Backup Premium uses non-proprietary Zip format, which means that compressed files can be manually restored using any zip-compatible utility.

Data integrity testing Backup Premium can test the integrity of backed up data. At the end of each backup session the saved files are checked for possible errors to ensure that the data can be reliably restored afterwards. E-mail notification enables Backup Premium to inform the user about the status of the backup task upon its completion via email. The feature can be especially useful for network administrators or anybody who have to back up tons of data and need to control this process anytime and anywhere.

Backup of Windows Profile, Outlook, Outlook Express and ICQ user data Mail clients and messengers very often deal with some important information but their data folders are usually hidden and therefore are not backed up. Backup Premium can back up data for various versions of Outlook Express, Microsoft Outlook and ICQ. It is also possible to back up .reg files from the registry.

[edit] Security of backup process

In recent versions Backup Premium offers the possibility to back up data on secure servers that work according to cryptographic protected file transfer protocols: SFTP and FTP SSL/TLS. In both protocols passwords and files are encrypted before transferring, which guarantees that the transferred data will not be intercepted. Another security feature of Backup Premium is blowfish encryption that ensures that nobody except you will be able to access your data. Blowfish is a symmetric block cipher that takes a variable-length password key, from 4 to 56 characters in length, making it ideal for both domestic and exportable use. Blowfish was designed in 1993 by Bruce Schneier as a fast, free alternative to existing encryption algorithms.
