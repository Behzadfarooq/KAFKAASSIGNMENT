 This article is part of WikiProject Backpacking, an attempt to better organize information in articles related to backpacking. If you would like to participate, you can edit the article attached to this page, or visit the project page, where you can join the project and/or contribute to the discussion.


This template is part of WikiProject Backpacking.

Please note: This template should be transcluded as {{BackpackingProject}} and not substituted {{subst:}}. Transclusion allows easy updating of all the Project's talk pages without having to edit thousands of pages.

Please add to relevant articles' talk pages. This template adds articles to Category:WikiProject Backpacking and is part of Category:WikiProject Backpacking Templates
