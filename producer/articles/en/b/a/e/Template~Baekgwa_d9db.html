This template allows standardized linking to various common Korean encyclopedias from the "External links" section of articles. Depending on the situation, it may be more advantageous to add this to the Talk page.

[edit] Parameters

The currently supported parameters are:

  • empas_b (for the Korean language Britannica)
  • empas_k (for the Encyclopedia of Korean National Culture)
  • empas_m (extension for maps from Korean language Britannica; defaults to 001)
    • An image link is produced automatically unless empas_m=no
    • empas_desc is the label on the image link; it defaults to "Map"
  • Dusan mirrors:
    • encyber
    • naver
  • Pascal mirrors:
    • yahoo
    • nate

Set the variable to the document ID for the specific entry.

[edit] Usage

These encyclopedias should be linked only in cases where they provide information that is not duplicated by similarly accessible English-language sources. The purpose of these links is to assist in the verification, expansion, and assessment of Wikipedia articles on Korea-related topics. If links are unlikely to serve that purpose in a particular article, they should not be inserted. See Wikipedia:External links.

Note that these are non-English tertiary sources. Under Wikipedia:Reliable sources, sources of this kind should not be cited as support for information in an article, unless no better alternative is available.

This template should not be used to link identical entries. Individual entries should only be listed if they are *distinct* from the others and provide unique information. In many cases (such as Encyber and Naver) the encyclopedias are simply parallel mirrors, and only one of them should be linked. Both parameters are available, in case one mirror does not host a given article.
