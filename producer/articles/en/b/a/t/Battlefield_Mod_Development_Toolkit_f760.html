Battlefield Mod Development Toolkit

From Wikipedia, the free encyclopedia

Battlefield Mod Development Toolkit (MDT) is a collection of software utilities that help creating mods for the computer games Battlefield 1942 (2002) and Battlefield Vietnam (2004).

The latest version, 2.75, includes:

  • archiving tools for the proprietary file format .rfa
  • plugins to import and export models in Gmax and 3D Studio Max
  • special editors for menu (binary) and sound (ascii) files
  • wizards that help with the first basic steps in creating a new mod including an Installer
