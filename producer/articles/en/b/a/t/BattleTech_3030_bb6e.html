BattleTech 3030

BattleTech 3030 is a mux class telnet game. It is based off the popular BattleTech board game, but instead of using turn based rules for combat, action is instant. BattleTech 3030 is similar to BattleTech 3065, but is not a sim site. On 3065 you buy mechs and can use them again after every sim. In BattleTech 3030 if your unit is damged it must be repaired and is not reusable if the unit is destroyed.
