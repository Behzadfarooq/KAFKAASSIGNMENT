[edit] Add "emblem" to uniform

I have not yet added this to Infobox baseball team, but note that there is a new set of parameters on this template, to include what I'm tentatively calling an "emblem" for the shirt. This "emblem" can be stylized text, a seal, etc. This allows you to build a basic shirt but then add the team name/logo/emblem across the front, when available. Rolando (talk) 15:20, 3 November 2006 (UTC)

[edit] Problem with transparent emblems

I did not promote the emblem thing to {{Infobox baseball team}} because it still has some problems--not the least of which is that the {{Superimpose}} template it uses does not work on IE with transparent PNGs.

If you're going to use the "emblem", make sure it's 38px wide.
