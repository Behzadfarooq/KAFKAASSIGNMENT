Category:Base-dependent integer sequences

From Wikipedia, the free encyclopedia

You have new messages (last change).

This is a category for integer sequences whose definition is dependent on the numeral system (normally base 10) used to write the numbers rather than on the numbers themselves: for example, a sequence which is restricted to numbers containing particular digits.
