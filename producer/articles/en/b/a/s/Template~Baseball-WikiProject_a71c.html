[edit] Usage

edit the instructions

Please note: This template should be transcluded ({{Baseball-WikiProject}}) and not substituted (subst) because it employs conditional code. Transclusion also allows easy updating of all the Project's talk pages without having to edit thousands of pages.

{{Baseball-WikiProject
|small = 
|class = 
|importance = 
|attention = 
|image =
|Oldtimetaskforce = 
|littleleague = 
|player =
|past-collaboration = 
|peer-review = 
|old-peer-review = 
}}


{{Baseball-WikiProject

  • small: Answer yes to make the template small and along the right side of the page (as opposed to at the top.)
  • class: Options are FA, A, GA, B, Start, Stub, NA. If blank, this will default as Unassessed. Descriptions of the options can be found at Wikipedia:WikiProject_Baseball/Assessment#Quality_scale.
  • attention: Answer yes if the page needs immediate attention, otherwise leave blank.
  • image: Answer yes if the page needs images, otherwise leave blank.
  • Oldtimetaskforce: Answer yes if the page is part of the Old-time Base Ball task force.
  • littleleague: Answer yes if the page is part of the Little League task force.
  • player: Answer yes if the page is about a baseball player and is part of the Baseball Players task force.
  • past-collaboration: If the page is a past collaboration of the month enter there month and year that it was.
  • past-selected: Answer yes if the page is a past article of the month
  • peer-review: Answer yes if the page is currently nominated for Peer review by WikiProject Baseball.
  • old-peer-review: Answer yes if the article has received a peer review from WikiProject Baseball per the option above.
