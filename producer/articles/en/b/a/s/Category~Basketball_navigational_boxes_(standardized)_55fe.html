Category:Basketball navigational boxes (standardized)

From Wikipedia, the free encyclopedia

This category is for Basketball navigational boxes that are using {{bb start}} and {{bb end}}. To add a template to this category, place the following code:

{{using bb}}

So that the end of the navigational template code looks like this:

[...]Template code[...]
{{bb end}}
{{using bb}}</noinclude>

For more information and discussion about the standardized basketball box templates, see Wikipedia:WikiProject Basketball/Templates.
