Category:Band templates

From Wikipedia, the free encyclopedia

These are footer templates for bands. For footer templates for solo musicians, see Cat:Musician templates.

There is a standard format for band-related templates. Please copy/modify the code from an existing band template if you wish to create a new one to keep them all relatively uniform.

To add a template, use the following code:

<noinclude>
[[Category:Band templates|{{PAGENAME}}]]
</noinclude>

To place a template in an article, type the template name between {{}}. Do not just copy-and-paste the template code.
