[edit] Hax Lolz

In order to use a picture as a link, the following must be done:

  1. Upload the image.
  2. Edit the image page with: 
    #REDIRECT[[TargetPage]]
  3. Link to the image to link to the page to which the image page redirects.
  • Note! It may take some time before the redirect works on the link, but it will work (at least on Wikibooks).

[edit] Citations

Inline citations work as follows:

Statement of fact.<ref>http://www.link.to/source</ref>

Appears as: Statement of fact.[1]

To list the citations at the end of the page, use:

<references />

Appears as:

  1. ^ http://www.link.to/source

[edit] Template Code

Template code is as follows:

Contents
of
Template
<noinclude>
  Information on template page that will not be included in the actual template.
</noinclude>

To include the template:

{{Location:Of/Template}}

[edit] Prefix Index Pages

To list all the pages with a particular prefix, particularly useful on Wikibooks or for User subspace, the following code can be used:

[[Special:Prefixindex/prefixtobeindexed|NameofLink]]
