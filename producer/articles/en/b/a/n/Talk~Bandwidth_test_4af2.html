[edit] Software for maximum throughput measurements

  • TPtest [1], which is developed on initiative by Swedish authorities, and measures the UDP and TCP maximum throughput from a central reference test server in Sweden to the user computer.
  • TPspeed [2] which includes a client similar to TPtest, and a reference test server.
  • Netfor2 RxTX [3], which measures the maximum throughput by simultaneously transferring files between the computer and multiple web servers.
  • IPNet Tuner [4], which can measure the link data rate with any destination IP adress, for example the nearest router.
  • Du Meter [5] which is logging the throughput (but not necessarily maximum throughput)
  • BBMonitor Bandwidth speed test software.
  • IPTab Bandwidthtest Accurate Bandwidth Upload and Download speed test.
  • Speakeasy Download and upload speed test. Simple, no hassle.
  • MySpeed Bandwidth Test Download and upload broadband speed test. May be down.
  • Bandwidth Tests Download and upload broadband speed test
  • InternetFrog JavaScript Speed Test Download and upload broadband speed test
  • www.evaluationengineering.com Making Accurate Bandwidth Measurements
  • Lawrence Berkeley National Laboratory Available Bandwidth Measuring and Sampling
  • VivilProject SpeedTest A multi-thread Windows speed test software (last about 15 minutes but give real result even with 20Mbit/s DSL lines...)
  • Toast Internet Speed Test Download and upload tests. Combination of text and/or image downloads. Simple, no hassle.
  • Testmy.netBroadband speed test and broadband community
  • Verizon speedtest
