[edit] Deleting content from pages?

Beyondthislife wrote:

Hi, in regards to the comment you left on my talk page about blanking content - I appear to be having an issue that you've pointed out where all the content in the edit box gets cut-off at a certain point (therefore, clicking submit will post the page with everything after the cutoff missing). Is this an issue with Wikipedia or is it something on my end? Further investigation shows me it's happening wherever I click edit and the page if over a certain length. Regards. Dan 21:15, 26 June 2006 (UTC)

This is a known issue if you are using Mozilla Firefox and have the Google Toolbar extension installed. Usually it is caused by switching tabs while editing. If this is the case, you may want to try disabling the toolbar and/or opening pages in a new window, or being careful to stay on the same tab until the page is saved. It also happens with some old versions of certain web browsers, in which case it's best to upgrade to a newer browser if possible. Hope this helps – Gurch 21:25, 26 June 2006 (UTC)
