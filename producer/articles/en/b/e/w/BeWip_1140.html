[edit] Presentation

BeWip is a proprietary softphone which unifies VoIP (based on SIP) and Cellular communications GSM on mobile GSM/WIFI mobile phone based on the Microsoft Windows Mobile operating system like PDAs / Pocket PCs / smartphones.

The address book present in BeWIp is based on Pocket Outlook address book. Thus, contacts synchronisation from his/her computer can be easily done with Microsoft ActiveSync.

The VoIP part of BeWip is based on the open standard protocol SIP | Session Initiation Protocol. If necessary, BeWip uses STUN protocol to get through certains types of NAT routers (Full Cone, Restricted Cone et Port Restricted Cone) present in IP networks.

BeWip is pre-configured on different network of VoIP providers.

BeWip is a product of Aliantiz.
