[edit] Popups

You might want to check out Wikipedia:Navigation popups for doing the Lyla disambiguations. Mike Dillon 17:25, 22 December 2006 (UTC)

I guess I didn't point out what the popups will actually buy you here. When you mouse-over a link to a disambiguation or redirect page, the popups will give you a link that will rewrite the link and automatically provide an edit summary. For bypassing redirects like you're doing, it automatically saves the changes with one click. For disambiguations, it dumps you into the "Show changes" view with all of the replacements already made. Mike Dillon 17:39, 22 December 2006 (UTC)
P.S. You probably don't want to change redirects on people's talk pages, since it will make it look like they have new messages. Assuming that you're going to make Lyla a disambiguation page, it's not necessary to change links in the user talk namespace. Mike Dillon 17:41, 22 December 2006 (UTC)

Glad you like the popups. One thing I'd like to suggest is that you read the disambiguation guidelines at WP:SONG#Naming conventions (if you haven't already). The page you set up for King Of Fools is very close, but you put [[King Of Fools (EP)|''King Of Fools'' (Edguy)]] instead of [[King Of Fools (EP)|''King Of Fools'' (EP)]]. Using the latter form makes it clearer for disambiguators which link is "King Of Fools (EP)", since there is not actually a link to "King Of Fools (Edguy)" on that page.
