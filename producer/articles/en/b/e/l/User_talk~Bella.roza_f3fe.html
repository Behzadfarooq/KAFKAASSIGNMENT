[edit] License tagging for Image:Hanson-band.JPG

Thanks for uploading Image:Hanson-band.JPG. Wikipedia gets hundreds of images uploaded every day, and in order to verify that the images can be legally used on Wikipedia, the source and copyright status must be indicated. Images need to have an image tag applied to the image description page indicating the copyright status of the image. This uniform and easy-to-understand method of indicating the license status allows potential re-users of the images to know what they are allowed to do with the images.

For more information on using images, see the following pages:

  • Wikipedia:Image copyright tags
