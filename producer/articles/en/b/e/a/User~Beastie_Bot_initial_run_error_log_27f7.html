Part 1 of the initial run error log.

Please cross off entries you've checked over like so:

ERROR: no taxobox [added taxobox ~~~~]

By the way, number after checking is the red list id. You can find the red list entry at http://www.redlist.org/search/details.php?species=number.

Beastie Bot has not altered any the following pages, and the reasons why are given.

If you think Beastie Bot should have added matched the page, then please make sure the taxobox is valid (especially that it has a binomial or trinomial parameter), and that the bi/tri-nomial name matches that of the Red List (excluding the letters 'ssp.'), or that the Red List's bi/trinomial is listed as a synonym in the taxobox.

If you find Beastie Bot's behaviour lacking or suboptimal, please let me know, but I can't promise I'll do any major changes too soon.
