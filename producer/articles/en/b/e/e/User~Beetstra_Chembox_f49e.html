[edit] Notes on use

The template contains a consistent layout for a vast amont of details about a chemical. Don't worry about trying to fill in all the fields -- even if you can only get the basics, that can still be very useful.

To use, simply copy and paste this into the document, filling in whatever you can, or type {{chembox subst}} and press save.

NOTE: there is no need to subst this template, empty fields will not show up, and when formatting changes are applied to the template, it will also be visible in all documents transcluding the chembox new.

For the sub-box supplement to show up, the supplementary page "pagename (data page)" has to be available.
