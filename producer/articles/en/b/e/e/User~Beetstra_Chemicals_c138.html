[edit] Things that have been programmed

I am currently programming AWB to automatically clean up chemical compound related pages (though the script may work on other pages as well, as long as I don't add {{chemicalSources}} to it). It tries to do a lot of things in one run:

  1. General fixes (interwiki sort, stub to bottom, etc.) (internal AWB machine).
  2. Unicodify (internal AWB machine).
  3. RegEx typo fix User:Mboverload/RegExTypoFix (internal AWB machine).
  4. Tables within <table ...> ... </table> into a chembox-like table.
  5. Catching chemical identifiers, move them into a template (get all those numbers and identifiers all into their own template (and use only ONE for each), so when in the end Wikipedia:Chemical sources has turned into a Special:ChemicalSources, all we have to do is change the template to call the special page, and voila .. the system is up and running. Only one catch left .. get the special page up and running):
    1. CAS# -> {{CASREF|CAS=000-00-0}}
    2. IUPAC-name -> {{IUPAC|IUPAC=000-00-0}}
    3. SMILES -> {{SMILES|S=000-00-0}}
    4. Chemical formula -> {{ChemicalFormula|CF=000-00-0}}
  6. Catch {{nfpa}}, move into {{NFPA 704}} (also do an unsubst on this template)
  7. Remove 'empty' tags on chemical pages (these are already in Chemical Sources). I only remove them if the are not pointing to the active compound! e.g.
    1. {{ecb}}.
    2. {{nist}}.
    3. {{pubmed}}.
  8. Add the ChemicalSources template to chemical substances that don't have that yet.
  9. Revert substs of {{ChemicalSources}}.
  10. Change {{cleanup}} into {{cleanupdate}}.
  11. Remove the space from ' ]'.
  12. "url" becomes "[url url]".
  13. Move inline url's into refs.
    1. "[url text]" becomes "text<ref name=url>[url text]</ref>".
    2. [url] becomes <ref name=url>[url url]</ref>.
  14. Rebuilding {{Maintenance}} template into seperate stubs
  15. Format the <references/>-tag into a <div class="references-small"><references/></div>
  16. Attempt to automatically add a <references/> tag to an article containing <ref>whatever</ref>-tags.
  17. Remove some empty lines (e.g. before the {{ChemicalSources}}-template) and before headers.
  18. Attempt to input the {{ChemicalSources}} template automatically (difficult, should be in the last section, and the place for that is hard to 'find' using regex.
  19. Change [[Category:cat]] into [[Category:cat|pagename]], where in the pagename the prefix has been moved to behind the name of the compound. Caught prefixes include:
    1. 1-; 1,1-; 3,3'-;
    2. N-; N,N'-
    3. R-; S-; 1R,2S,3R- (latter not tested, but should work)
    4. n-
    5. o-
    6. m-
    7. p-
    8. ÃŸ-
    9. D-
    10. L-
    11. (-)-, (+)-
    12. Tert-
    13. Sec-
    14. Bis-
    15. Cis-
    16. Trans-
    17. Ortho-
    18. Meta-
    19. Para-
