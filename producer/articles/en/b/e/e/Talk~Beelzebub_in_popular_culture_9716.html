[edit] Not really a disambiguation page

This page isn;t so much a disambiguation page as it is a dumping ground for trivial fictioncruft, created as a WP:FORK file by an editor upset that his favorite videgame reference was removed. Wikipedia is not a list of trivia. This page should probably be redirected, like the other forks created by the same problem editor (like Beeelzebub in culture and so forth... but then if it's existence is a nice way to push all the constant WP:ENC-violating additions by kids about the videogames off into some other location so as not to drag the main article down, I suppose that's an alright compromise. DreamGuy 07:53, 17 December 2005 (UTC)
