[edit] Guidelines

  1. Keep World Wide Web links first.
  2. Add the note with the link to the Freenet article (not the Freenet site) before the links.
  3. Add the note stating that localhost is assumed as the base

Here is an example:

[edit] External links

[edit] World Wide Web links

  • http://www.whengendarmesleeps.org My personal page

[edit] Freenet links

Template:Freenet links

  • Anarchist Black Cross on Freenet Check out the guide of staying out of jail...
