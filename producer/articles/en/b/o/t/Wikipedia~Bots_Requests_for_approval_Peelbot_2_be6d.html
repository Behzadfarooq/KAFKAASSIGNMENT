[edit] Peelbot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Mike Peel

Automatic or Manually Assisted: Automatic tagging, manual selection of pages

Programming Language(s): AutoWikiBrowser with User:Kingboyk's plugin

Function Summary: Tagging articles with WikiProject banners

Edit period(s) (e.g. Continuous, daily, one time run): Overnight (approx 9pm-7am GMT, potentially starting/stopping a bit later or earlier depending on workload and my schedule), but not every night.

Edit rate requested: Average of about 3 edits per minute; see Special:Contributions/Peelbot for tagging speeds thus far.

Already has a bot flag (Y/N): Y

Function Details: I already have approval for tagging using Template:Physics (see Wikipedia:Bots/Requests for approval/Peelbot); this is a request to generalize that to any wikiproject banner. The main reason for this request is to tag talk pages with Template:WPAstronomy, but I would prefer a more general approval if possible.

I have had some complaints that the bot has been overzealous with tagging pages with Template:Physics; this was due to non-physics pages being within physics categories. I do not expect any future complaints, as I will be keeping a much closer eye on the pages within the categories that I am tagging. Mike Peel 17:13, 11 January 2007 (UTC)

[edit] Discussion

  • I'm assuming you're looking for approval for any wikiproject at their request, rather than to "go freelance", right? If so, this seems to be well-established by precedent. Alai 05:03, 12 January 2007 (UTC)
That's right. I've received a request to use the bot for WP Astronomy and WP Astronomical Objects (see my talk page). Once I have approval for this task, I will be putting a note on the user page of the bot saying that it is available for tagging other wikiprojects on request. Mike Peel 09:45, 12 January 2007 (UTC)
test run approved 50-100 edits Betacommand (talk • contribs • Bot) 18:47, 14 January 2007 (UTC)
Please see edits of 16 January 2007 at Special:Contributions/Peelbot. Specific examples include:
  • [1] Creating a new talk page, and placing the template on it
  • [2] Adding template to a page with content
  • [3] Tagging a page which already has other templates as well as discussion on it
  • [4] Tagging a category
The edits were run at the top speed of the bot, which is about 3 edits per minute (I have a fairly fast computer, but it's a PowerPC-based Mac so I'm having to emulate a really slow computer to run windows to get AWB to work). I set the bot going at this time of day so that I can keep an eye on its edits; it will normally be running overnight (9pm-7am GMT). Mike Peel 20:45, 16 January 2007 (UTC)

Approved. The bot shall run with a flag. —Mets501 (talk) 18:14, 21 January 2007 (UTC)
