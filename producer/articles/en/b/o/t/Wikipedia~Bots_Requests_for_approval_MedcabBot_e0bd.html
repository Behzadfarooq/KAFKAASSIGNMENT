[edit] MedcabBot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Xyrael

Automatic or Manually Assisted: Automatic

Programming Language(s): I didn't write the code, but I believe this is all Python - [1]

Function Summary:This bot has actually existed for some time, but apparently it wasn't flagged by the original owner, which surprised me. The owner is now switching to user:Xyrael, and all the bot does is update the list of cases in the Mediation Cabal cases list and list the status of each case.

Edit period(s) (e.g. Continuous, daily, one time run): Probably running around 2 times a day, perhaps a bit faster if necessary.

Edit rate requested: Not set in stone, but it won't be faster than 1 edit per 10 minutes as it was in the past.

Already has a bot flag (Y/N): N

Function Details: I couldn't go into the technical details myself. I'm not the bot creator, but I'm only submitting this as I'm trying to organize the handover of the bot to the new owner and discovered that it doesn't have a flag after all this time operating. The source code is at this link. Cowman109Talk 21:11, 6 February 2007 (UTC)
