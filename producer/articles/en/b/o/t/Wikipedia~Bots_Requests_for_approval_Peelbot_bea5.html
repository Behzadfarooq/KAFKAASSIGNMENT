[edit] Peelbot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Mike Peel

Automatic or Manually Assisted: Automatic tagging, manual selection of pages

Programming Language(s): AutoWikiBrowser with User:Kingboyk's plugin

Function Summary: Tagging the talk pages of physics-related articles with {{physics|class=|importance=}}, also {{WPBiography|class=|priority=}} in people-related categories (e.g. Category:American physicists).

Edit period(s) (e.g. Continuous, daily, one time run): Overnight (approx 9pm-7am GMT, potentially starting/stopping later depending on workload and my schedule), but not every night.

Edit rate requested: 6 edits per minute max

Already has a bot flag (Y/N): N

Function Details: See function summary above. Note that I've already made a large number of such edits with my normal user account - see Special:Contributions/Mike Peel. I'm sort of experimenting - I plan to use the bot for other things in the future for as yet unknown activites, which I'll seek approval for as/when needed.

Sure, post a couple diffs, this seems like a routine approval -- Tawker 08:07, 14 November 2006 (UTC)

I'm awaiting AWB authorization on the new account; I'll post diffs using that account once that's been approved. A couple of sample diffs from the exact setup that peelbot will use, run under my account: [1] adding {{physics}} and {{WPBiography}}, [2] adding just {{WPBiography}} to an article that's already tagged as physics, [3] adding just {{physics}} where {{WPBiography}} is already present. An older edit: [4], where I'm just tagging with {{physics}}. Mike Peel 09:28, 14 November 2006 (UTC)
Diffs from Peelbot:
  • [5] [6] [7] adding {{physics}} to pages. Also set to add WPBiography where appropriate, or change the variables to the latest ones if necessary, hence the rearranging of that template. That would not happen in non-WPBiography categories.
  • [8] [9] [10] adding both Template:Physics and Template:WPBiography. Note that I would just add WPBiography when tagging with Physics, or checking through physics categories. I will not be tagging non-physics categories.

Note that I'd make edits during the day instead of the night, as I can't (yet) sleep-edit - only my computer can do that. I'd only be making about 1 edit per minute in that case, as per the rate of editing on my Mike Peel (talk • contribs) account. I guess that this would be similar to a trial run; should I wait until that's been approved before making edits?
