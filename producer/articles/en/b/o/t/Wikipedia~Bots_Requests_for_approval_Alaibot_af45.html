[edit] Alaibot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Alai

Automatic or Manually Assisted: manually run, edits automatically

Programming Language(s): Python (pywikipedia framework)

Function Summary: Populate "category missing" maintenance categories from database dumps.

Edit period(s) (e.g. Continuous, daily, one time run): Intermittent, manually run on an as-needed/as-identified basis.

Edit rate requested: 6 edits per minute

Already has a bot flag (Y/N): yes

Function Details: Populating of "missing category" maintenance categories such as Cat:Uncategorised albums, from lists generated from offline database dumps. (See for example, User:Alai/uncat-albums, album-stubs lacking any (other) album categories.) To use the pywikipedia framework to add the category, while double-checking for categories added more recently than the db dump to avoid "false positives". Sample/test edit: [1]. Currently this just uses replace.py to do this: it might be slightly more elegant to do a code-merge with category.py, which I'll look at doing in due course. I suggest using the above list (or if preferred, part thereof) as a "trial run", given that a request specifically for that at WP:ALBUM is what inspired this task. Subsequent runs would be done on the basis of either existing similar maintenance categories, such as Cat:Year of birth missing, or creating additional same in co-ordination with wikiprojects expressing an interest in emptying same. Alai 07:31, 7 November 2006 (UTC)
