[edit] STBot

The following discussion is an archived debate. Please do not modify it. Subsequent comments should be made in a new section.

Operator: ST47

Automatic or Manually Assisted: Jobs will be manually entered and the page list will be checked, the bot would then proceed on it's own while I prepare the next job.

Programming Language(s): uses WP:AWB but I'm working on the pywikipedia framework to see if there is anything else I can do.

Function Summary: Will change categories due to WP:CFD, mostly in substed userbox and other large categories.

Edit period(s) (e.g. Continuous, daily, one time run): 3-4 jobs 2-3 times daily

Edit rate requested: I really have no idea, whatever you all think is ok

Already has a bot flag (Y/N): N

Function Details: Change or remove categories from pages in the main, template, user namespaces according to WP:CFD/W. I would change any templates and move the category text, the bot would then go through the remainder of the category and make the changes.
