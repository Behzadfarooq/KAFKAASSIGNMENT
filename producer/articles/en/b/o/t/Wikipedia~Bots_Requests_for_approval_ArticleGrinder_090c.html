[edit] ArticleGrinder

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Francis Tyers ·

Automatic or Manually Assisted: Automatic

Programming Language(s): Python with pywikipedia.

Function Summary: Mass create articles on Iranian counties.

Edit period(s) (e.g. Continuous, daily, one time run): One time run.

Edit rate requested: 8 edits per 1 minute

Already has a bot flag (Y/N): N

Function Details:

We have been experimenting with mass article creation on the Tajik Wikipedia. Nearly all our articles that we have mass created have articles here. We have created 320 articles on Iranian counties (below province subdivision) and would like to spread the wealth. The ArticleGrinder will mass create these articles from a template and a list of data in a csv file.

[edit] Discussion

A few questions:

  1. Can you provide a link to a tG: dif created by this bot?
  2. Where is this data comming from, and will it be referenced?

Thanks, — xaosflux Talk 00:32, 26 September 2006 (UTC)

Articles on tg: and sh: may be found here: tg:Шаҳристони Сароён and sh:Okrug Nihbandon. The data is coming from here, a list prepared by an Iranian user. It is not referenced. As you can see from :tg: and :sh:, some bugs in the script have been ironed out. - FrancisTyers · 09:37, 26 September 2006 (UTC)
The data currently looks like:
tgname   ,faname ,tgprov,      ,enname           ,shname   ,shprov

Исломшаҳр,اسلام‌شهر,устони Tиҳрон,Islamshahr County,Islomšahr,Tihron
etc. (if there is an odd unicode bug when you look at the display, check the source. - FrancisTyers · 09:39, 26 September 2006 (UTC)
Just to outline, some of the problems we had involved unicode issues (with BiDi text) — this has been fixed. Also silly issues with having variables in Latin in the template when the template was in Latin. Also fixed. - FrancisTyers · 09:50, 26 September 2006 (UTC)

About the reference, the data (Persian names) are from "اطلس گیتاشناسی استان‌های ایران" which is published in Tehran, 2004. Jahangard 20:32, 26 September 2006 (UTC)

Note: In Latin: "Atlas Gitashenasi Ostanhai Iran", English: "Gitashenasi Province Atlas of Iran" [1] - Francis Tyers · 11:22, 28 September 2006 (UTC)
