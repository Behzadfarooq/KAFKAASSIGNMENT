[edit] Mathbot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Oleg Alexandrov

Automatic or Manually Assisted: Automatical.

Programming Language(s): Perl.

Function Summary:

Most edits are for the WP:1.0 project, it updates the pages listed at Wikipedia:Version 1.0 Editorial Team/Index. The bot also adds an edit summary counter for administrator candidates, and updates the list of mathematics articles. The bot has been doing those tasks for a very long time.

Edit period(s) (e.g. Continuous, daily, one time run): It edits daily. Most edits are at the Wikipedia:Version 1.0 Editorial Team/Index which run from 4AM UTC for say six hours.

Edit rate requested: No more than one edit per two seconds for say six hours every day for the WP1.0 project.

Already has a bot flag (Y/N): No.

Function details:

It reads all subcategories of Category:Wikipedia 1.0 assessments and outputs the data as subpages of Wikipedia:Version 1.0 Editorial Team (see the index).

Also, it reads all math categories and outputs the results in the list of mathematics articles.

Some other miscellaneous functions are described at User:Mathbot. Oleg Alexandrov (talk) 15:42, 21 October 2006 (UTC)
