[edit] Bot-Schafter

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Mandavi

Automatic or Manually Assisted: Automatic

Programming Language(s): python

Function Summary: The bot will add the small () stars for featured articles of national wikipedias at their interwikilink.

Edit period(s) (e.g. Continuous, daily, one time run): once every one or two month

Edit rate requested: 2 edits per minute

Function Details: The bot will add the stars in order to give users with knowledge of foreign languages the chance to enrich the English articles with already approved content. Additionally I hope to get users more interested in foreign languages. The script will go through the equivalents of the Category:Wikipedia featured articles of other languages and adding a {{Link FA:xx}} to the according articles in the english wikipedia right beside the interWikiLink - if doesn't contain a {{Link FA:xx}}, {{link FA:xx}} or {{Link_FA:xx}} already. I've improved the original script to include more lists and to be more foolproof. The script is already running stable on the Swedish and Czech wikipedia.

[edit] Discussion

Bot trial run approved for the duration of one week. Seems interesting. Sources and procedure are clearly given and simple enough.Voice-of-All 23:42, 28 October 2006 (UTC)

Approved. This bot has been running unapproved for quite some time, but treating it as a trial, things have been great and this seems like a useful bot. I'm approving this bot for usage at up to 6 edits per minute and it shall run with the bot flag. -- RM 21:49, 28 November 2006 (UTC)
