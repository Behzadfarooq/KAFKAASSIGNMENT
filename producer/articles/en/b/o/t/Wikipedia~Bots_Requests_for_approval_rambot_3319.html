[edit] User:Rambot

Owner User:Ram-Man
Function Pending external permission, updating racial links to match accepted practice. Also potentially adding article local references instead of Wikipedia:Geographic references. Also the host of TODO tasks listed on User:rambot and in the talk page archives. There is a lot of work that is outstanding. And just for kicks, notification that in 2010ish I will be updating all the statistics with updated census information (although this was assumed in the past). I may even add census bureau estimates if I get around to it.
Language english
Program custom bot software
Mode started manually, run automatically, monitored frequently. IP Address posted on bot page for easy blocking by administrators.
Other Accounts en:User:RM (without any flags whatsoever)
commons:User:Ram-Man (ditto)
NOTES This is not strictly an approval request, since the nature of tasks being performed are already approved. But because they were approved years ago and it have been so long since I've run the bot, it's important to notify the group that I plan to start running them again, so it can be blocked in the event of problems.

For those who are part of bot policy that don't remember the rambot, see the rambot FAQ. The bot will run for days, so don't be alarmed. Update: Since the bot will be running for days at a higher than normal rate (due to 37,000 edits required), it will run on the weekend (Friday and Saturday) as requested by bot policy and avoid Monday at all costs). I'll probably add a technical solution where the bot can be controlled from my user page. — Ram-Man (comment) (talk) 12:28, 30 August 2006 (UTC)

Update: There is a discussion at Village pump (technical) regarding the fact that the rambot seems to be partially blocked. Perhaps this is because the rambot was never officially approved under (or grandfathered into) current policy since it effectively predates it. Perhaps someone in the approval group could do me a favor and add me to the Wikipedia:Bots/Approval log and this might solve my problem. Of course if all approval does is add a bot flag, that won't help. But I should be on the list anyway. — Ram-Man (comment) (talk) 13:57, 30 August 2006 (UTC)
