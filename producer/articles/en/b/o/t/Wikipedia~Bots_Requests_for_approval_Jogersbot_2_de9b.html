[edit] Jogersbot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Jogers

Automatic or Manually Assisted: automatic

Programming Language(s): AWB

Function Summary: tagging album articles' talk pages with {{Album}} template

Edit period(s) (e.g. Continuous, daily, one time run): continuous

Edit rate requested: 1 edits per minute

Already has a bot flag (Y/N): Yes

Function Details: The bot would go through articles containing {{Infobox Album}} or categorized as albums and tag their talk pages with {{Album}} template using Kingbotk Plugin.

[edit] Discussion

  • That's tagging the talk page of course. Please check that the plugin is up-to-date with support for {{Album}}; it's been a while since any logic changes have been made. If that's affirmative AFAIC you're good to go. --kingboyk 20:08, 4 March 2007 (UTC) PS After building the list I would then have AWB filter it against Category:WikiProject Albums articles so that you don't waste time and resources loading talk pages which have already been tagged. --kingboyk 20:20, 4 March 2007 (UTC)
    • I have tagged about 100 pages manually so far without any problems. I just had to enable the option to "skip the talk page if the existing template instance is bad" because it doesn't work well without it. I explained the problem here. Thanks for the suggestion. Jogers (talk) 20:26, 4 March 2007 (UTC)
      • Whether or not you agree with my determination of what's a bad tag and what isn't (I consider the example you gave bad), I'd strongly recommend tagging with Skip Bad on. The worst that can happen is that you get a logfile with a few bad tags written to it that you have to attend to yourself later. If you run with that option off, the tagging becomes only a semi-automatic operation because it will seek your input on questionable tags (iirc). --kingboyk 20:43, 4 March 2007 (UTC)
        • Yes, I realize that this option should be turned on in automatic mode. Jogers (talk) 21:05, 4 March 2007 (UTC)
    • And of course I meant talk pages. I have updated the function summary. Jogers (talk) 20:32, 4 March 2007 (UTC)

Actually it makes more sense to start with articles in album stubs categories than to go through pages containing an infobox so they can be automatically assessed. I have updated function details. Jogers (talk) 21:18, 4 March 2007 (UTC)

Approved for trial. OK, sounds like you know what you're doing and you have approval for AWB already :) Make 50-100 edits and make sure there's no problems, then report back. --kingboyk 21:22, 4 March 2007 (UTC)

OK, thanks. Jogers (talk) 21:25, 4 March 2007 (UTC)
So far so good. Jogers (talk) 23:48, 4 March 2007 (UTC)
Looks OK to me but in the interests of impartiality I'll let another member give the final nod. --kingboyk 23:57, 4 March 2007 (UTC)

Approved. — xaosflux Talk 02:29, 5 March 2007 (UTC)

Would it be a problem if I increased the edit rate to 1 edit every 10 seconds to clear the backlog? It's going to take me weeks at the current pace. Jogers (talk) 16:43, 5 March 2007 (UTC)
No problem. —METS501 (talk) 20:27, 5 March 2007 (UTC)
