[edit] Halibott

Operator: //Halibutt

Automatic or Manually Assisted: Fully manual (using AWB)

Programming Language(s): AWB

Function Summary: Tedious tasks like changing several hundred links in accordance with WP:RM

Edit period(s) (e.g. Continuous, daily, one time run):From time to time

Edit rate requested: I'm not sure if anytime I have some free time fits here... So far I've been able to make roughly 100 edits in one go.

Already has a bot flag (Y/N):Nope.

Function Details: See description above.

[edit] Discussion

Since it's basically an AWB account and all changes are applied manually, it's not really a proper bot and, as suggested at the talk page, it doesn't really fit the definition. However, I listed it here to avoid clogging the watchlists, as several hundred edits in a row might indeed be a problem for someone who has them all in his or hers watchlist. //Halibutt 13:27, 17 September 2006 (UTC)
