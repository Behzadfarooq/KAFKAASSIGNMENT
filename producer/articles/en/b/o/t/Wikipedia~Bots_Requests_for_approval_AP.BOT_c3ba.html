[edit] AP.BOT

This is a simple bot to clean up the huge userbox backlog Here. It will use AutoWikiBrowser and will only run when I give it the instructions.

For example it will replace {{User diy}} with {{User:Holek/Userboxes/Do It Yourself}}

It pretty much does the same job that Botdotcom bot does.

It is operated only by me, Andeh.

I could do this task on my current account but feel it would be better opening a separate account to do it.

When the backlog is cleaned up, I will request an administrator to block the account indefinitely. If I decide to give it another function, I'll request it here first before requesting the account to be unblocked.--Andeh 11:03, 9 August 2006 (UTC)

Ok, give us a few diffs and we'll approve it from there -- Tawker 05:24, 10 August 2006 (UTC)
Could you add the account to the AWB allowed list? Then I'll do a few to show you I know what I'm doing.--Andeh 15:30, 10 August 2006 (UTC)
Done -- Tawker 15:57, 11 August 2006 (UTC)
One week trial period approved, please limit edits to no more than 2/3 per min and limit the trial run to 250 replacements. — xaosflux Talk 02:11, 12 August 2006 (UTC)
Started, see AP.BOT (talk • contribs • logs).--Andeh 14:06, 14 August 2006 (UTC)
249 edits to user space;

talk • contribs • page moves • block user • block log • count--Andeh 15:06, 17 August 2006 (UTC)

  • Bot approved for continuous operations, that backlog is getting small now though, this should be able to be run without a flag; use an AWB delay of 13-20 seconds. — xaosflux Talk 15:30, 19 August 2006 (UTC)

[edit] AP.BOT, second function

Request to substitute vandal warning templates, such as {{test}} with {{subst:test}} and {{test2}} with {{subst:test2}}. And will be made to only edit in User talk space. And possibly in the future, other templates mentioned here.

My userbox replacing has gone near-perfect and the bot has only made 1 erroneous edit[1] which I fixed immediately [2] and made the adjustments for the bot to continue working correctly.--Andeh 08:24, 24 August 2006 (UTC)

Indeed, substing these is a good idea, but how does the bot get the list of pages that need substing? Does it check categories for the tags? How does it tell subst ones from non-subst ones?Voice-of-All 15:08, 27 August 2006 (UTC)
It will be using the "What transcludes here" function, I'll also filter it to only the user_talk space.--Andeh 15:53, 27 August 2006 (UTC)
Trial run approved, there are are a few other bots running these, but most don't run on any schedule. Report in in a week with some results. — xaosflux Talk 03:22, 28 August 2006 (UTC)
OK, see AP.BOT (talk • contribs • count).--Andeh 11:29, 28 August 2006 (UTC)
Sorry, but I'm getting lost in a sea of difs :) Can you be a little more specific? — xaosflux Talk 03:07, 29 August 2006 (UTC)
I've been doing anything listed at Wikipedia:Template_substitution, the only other task I have been doing is fixing the user box redirects. Everything seems fine so far.--Andeh 14:35, 29 August 2006 (UTC)
  • This looks fine, APPROVED, ping me if you think you will need a flag. — xaosflux Talk 01:22, 1 September 2006 (UTC)

[edit] AP.BOT, third function

Updating categories which have been renamed due to discussions at Wikipedia:Categories for discussion. So just replacing old categories to new ones, so the old ones can be deleted.--Andeh 18:36, 10 September 2006 (UTC)

Please fill these in: — xaosflux Talk 22:11, 10 September 2006 (UTC)

Automatic or Manually Assisted: manually given tasks

Programming Language(s): (AWB?) <--yep, same as usual

Edit period(s) (e.g. Continuous, daily, one time run): anytime, whenever I tell it to

Edit rate requested: X edits per TIME, well I don't really mind, but the backlog is pretty big so at least 5 per min, maybe higher at off-peak times.
