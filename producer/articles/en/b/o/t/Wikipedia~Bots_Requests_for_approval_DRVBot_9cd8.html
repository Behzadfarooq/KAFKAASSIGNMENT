[edit] DRVBot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Martinp23

Automatic or Manually Assisted: Auto

Programming Language(s): C#

Function Summary: Log management of WP:DRV

Edit period(s) (e.g. Continuous, daily, one time run): Daily

Edit rate requested: 5 edits per min

Already has a bot flag (Y/N): N

Function Details:

  1. Creates a daily log page
  2. Monthly, creates the monthly archive pages
  3. Will parse the monthly log page (in future) to trim its size down from full page transclusions to just links to the daily pages, and results.

Note: The date format for the pages written by the bot is ISO: 2007-03-04 (for example), which trialsanderrors tells me is catching on as a better standard.
