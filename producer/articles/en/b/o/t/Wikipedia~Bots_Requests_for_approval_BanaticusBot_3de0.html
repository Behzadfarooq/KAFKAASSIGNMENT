[edit] BanaticusBot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Banaticus

Automatic or Manually Assisted: I'll be manually assisting it the whole way. The automated part is mostly to step through userpages.

Programming Language(s): It uses, uhm, whatever language AutoWikiBrowser uses. That's also the name of the program.

Function Summary: Moving religious userboxes from regular namespace to user namespace

Edit period(s) (e.g. Continuous, daily, one time run): It will most likely be running for a month or two.

Edit rate requested:

Already has a bot flag (Y/N): N

Function Details: The bots purpose is to move userboxes from templated: User space to regular user namespace, namely my own, starting with the religious userboxes. It would replace the regular userbox with a "Hey, the userbox has changed, its new location is..." userbox. The main purpose of the bot would actually be to change userpages, to reflect the actual change, so that the "template: user userbox" page could be deleted once nothing links to it.

Userpage has been created. Yes, this proposed bot is based of the WP:GUS initiative. I could work on that page. I'd planned on starting with the generic religion userboxes, but since it appears that those pages are on a timer, perhaps it would be better to start with them. I don't see any point in moving an entire page when I can simply copy/paste the userbox information to userspace (most likely my own), changing the User: template to reflect the move then going through and changing each userpage page that points to the template (what the bot would be of great value for) then marking the template for deletion should be enough. Banaticus 05:20, 19 October 2006 (UTC)
The move, soft redirect processs isnt really needed unelss there are a lot of transclusions, if its only a few, it can be move/update/tag for delete. Why a move is preferable to c&p is that it preserves the GFDL history. — xaosflux Talk 05:40, 20 October 2006 (UTC)
Just so the BAG knows, if the bot is based on UBM (formally GUS) initiative, there hasn't been any talk about it on WT:UBM. In case that was what you're wondering. I also believe most religious boxes have already been migrated, check here. Anything new is a repost. -Royalguard11(Talk·Desk) 00:35, 23 October 2006 (UTC)
xaosflux, copy/paste is a better idea, when the individual userpages are edited as I proposed to do. As of right now, two religious userboxes on my personal userpage are now red "uncreated" links. Were they moved? I have no idea. Who deleted them? I have no idea. Where can I find out more information? As of right now I don't know. In the last week or so while I had no internet access, apparently someone(s) have come through and just willy-nilly deleted them instead of migrating them to userspace. Even from WP:Userboxes, they're just all gone -- no mention on the talk page of what happened to them. Royalguard, i presume that you had a hand in moving those.
