[edit] MartinBotII

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Martinp23

Automatic or Manually Assisted: Automatic

Programming Language(s): C#

Function Summary: As requested at Wikipedia_talk:WikiProject_Council#Hello_and_what_can_I_do (near he bottom of the topic), will put wikiproject peer reviews (originally localised) onto the centralised WP:PR.

Edit period(s) (e.g. Continuous, daily, one time run): Current working idea - twice daily (every 12 hours)

Edit rate requested: 6 edits per minute

Already has a bot flag (Y/N): Y

Function Details: WikiProjects reqiring the bot will put {{WikiProject peer review}} on thier localised peer review page. The bot then checks the template to find where it is transcluded, and gets the list of peer reviews transcluded on that page from the wikitext. Each of the pages listed is then parsed - if it contains {{WikiProject peer review a}}, the bot ignores it. If it doesn't, the template is added to to the page (and the page is saved) and the page is added to a list (in the program). The pages in this list are compiled with others from the other wikiproject peer review pages being checked and added to the main WP:PR page (by transclusion). To prevent requests being re-added after they are archived from WP:PR, the {{WikiProject peer review a}} template is added. This is empty, so has no effect on the page rendering, but its presence prevents the bot from adding it back to the PR page. The project is opt-in only, and I will act immeidately upon any comments I get from WP:PR about the bot. Thanks for considering this -- Martinp23 23:31, 30 October 2006 (UTC)
