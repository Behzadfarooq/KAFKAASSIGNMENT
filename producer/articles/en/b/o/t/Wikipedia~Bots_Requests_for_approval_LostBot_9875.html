[edit] LostBot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Lostintherush

Automatic or Manually Assisted: Manually assisted - I will set it up to start the run and monitor.

Programming Language(s): Will use AWB with plugin

Function Summary:To tag talk pages with {{WP India}} banner for assessment purposes. This will make it easier for Wikipedia 1.0 reviewers to locate unassessed articles.

Edit period(s) (e.g. Continuous, daily, one time run): Daily

Edit rate requested: AWB on 5 second throttle; up to 10 edits per minute

Already has a bot flag (Y/N): N

Function Details: I am planning to use the bot to tag all India-related article talk pages with {{WP India}} project banner. I will be doing this in conjunction with User:Ganeshk. See his request for an identical approval.

What do you mean "in conjuction with"? Additionaly, were are you populating the starting list from?Voice-of-All 23:39, 12 October 2006 (UTC)

I meant that both Ganeshk and myself will be doing the same task with our respective bots. The starting list will be populated from any articles that fall under Category:India or its sub categories. -- Lost(talk) 02:19, 13 October 2006 (UTC)

:I dont think we need two bots for this task I know from experiance that this can be done rather quickly with one bot and can be kept up to date very easly. I would move to reject this bot, no offence ment but one bot can handle this task. Betacommand (talk • contribs • Bot) 03:04, 13 October 2006 (UTC)

Both of us are not likely to be online all the time and the number of articles is quite large. (I am looking for a tool that gives me the exact number of articles in a category and all its subcategories. If you know of one, please guide me to it). So we would like to share the load amongst ourselves. That is the purpose of this request. -- Lost(talk) 03:20, 13 October 2006 (UTC)

Approved for trial. I know that the other bot was approved in totality, but I want to see this bot go through a trial period. Let's see 40-100 edits done by this bot using this method, just to make sure everything checks out. Post the results here when completed. Since this will be coordinated with the Ganeshbot, I see no reason why the two can't exist working together, and it is not clear from bot policy that two bots with the same task are not allowed anyway. If someone else disagrees with me, we can always not approve the request, but a trial shouldn't hurt any. -- RM 04:42, 13 October 2006 (UTC)

Thank you. I will get back to you in a few days with the results -- Lost(talk) 04:45, 13 October 2006 (UTC)
I withdraw my prevoius statement and I offer my bot to help this is a supermassive task over 16,00029323 pages. Betacommand (talk • contribs • Bot) 14:23, 13 October 2006 (UTC)

Approved for trial. This is a tad unorthodox, but I approve Betacommand for trial for this same task if he so pleases and the three can work out the details. -- RM 15:23, 13 October 2006 (UTC)
