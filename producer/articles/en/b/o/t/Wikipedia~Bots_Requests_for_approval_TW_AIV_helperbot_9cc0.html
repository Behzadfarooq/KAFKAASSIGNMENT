[edit] TW AIV helperbot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: TeckWiz


Automatic or Manually Assisted: Automatic

Programming Language(s): Perl

Function Summary:This is a backup version of HBC AIV helperbot, running the same source code. However, though it's a backup, it runs full time.

Edit period(s) (e.g. Continuous, daily, one time run): The bot will run all day, though it will only edit when an AIV report must be taken away. The most it will usually edit is twice a minute (if all bots are running).

Edit rate requested: 2 edits per minute (except if the other bots go down. Then the rate will probably be 5 per minute)

Already has a bot flag (Y/N):No

Function Details: The bots will all try to take a job, and the first one to get there takes it, with the other ones stopping and waiting for the next job. It removes blocked users from WP:AIV, combines reports, and adds and removes the admin backlog template when a specific amount of reports, specified on the AIV page, is reached. This version is in UTC -5, so as to make it so an operator is almost always available.

[edit] Discussion

As I mentioned on my talk page, I don't think we need a fourth instance of this bot. But I have been wrong before. HighInBC (Need help? Ask me) 15:18, 11 February 2007 (UTC)

Denied. We already have 3 instances of this bot. We don't need a fourth. —METS501 (talk) 15:20, 11 February 2007 (UTC)
