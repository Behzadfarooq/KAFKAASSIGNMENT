[edit] DomBot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Chidom

Automatic or Manually Assisted: Manually Assisted

Programming Language(s): will run on AutoWikiBrowser

Function Summary: Category cleanup on pages for renamed/deleted categories per WP:CFD / WP:CFDS

Edit period(s) (e.g. Continuous, daily, one time run): Continuously/daily/when I have time

Edit rate requested: 11 edits per minute (11 edits per minute is the fastest I've worked; if that's too high, the highest maximum below that)

Already has a bot flag (Y/N): N

Function Details: Summary pretty much covers it, I think.

[edit] Discussion

I've been using AWB to change category names on pages for renamed categories or delete category names from pages as appropriate. Unbeknownst to me, I was editing too fast and caused problems. (See my Talk page.)

Additionally, since I didn't create the DomBot account from the link above, it has been blocked by Pschemp; there is discussion on the bot's Talk page. (I created the account per the comment from After Midnight: "You might want to start by creating a new account for your bot activity."; I didn't realize until I was blocked that I should have done it from here.)

Note In case anyone got the wrong impression, I wasn't trying to place any blame on After Midnight; I should have read this page more closely. I appreciate all the help that's been given—thanks!—Chidom talk  03:47, 8 October 2006 (UTC)

I'm just looking for approval to make category deletion/renaming edits to pages without clogging up the system. Thanks.—Chidom talk  02:14, 8 October 2006 (UTC)

Speedy Approved; the bot shall run with a flag. Wait until it is flagged before increasing the edit rate. Be sure to always check the CfD history page to make sure that the results were not trolled around with. Other than that, there is not much to say.Voice-of-All 02:27, 8 October 2006 (UTC)
Thanks. It seems that Pschemp has taken a break for now; would it be possible to get DomBot unblocked? (I know to wait for a flag.) Thanks.—Chidom talk  03:47, 8 October 2006 (UTC)
Bot flag has been set. Redux 07:07, 8 October 2006 (UTC)
