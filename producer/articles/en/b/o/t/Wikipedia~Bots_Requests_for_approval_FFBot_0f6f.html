[edit] FFBot, new function

Request to expand on the current task of updating {{cleanup}} tags to help out with updating {{wikify}} tags too, to {{wikify-date|...}} as Pearle does currently. Just to help reduce the load off Pearle, and in case the other bot goes offline/on holiday. Thanks, — FireFox (talk) 16:14, 26 August 2006

Since another bot already does this, the only question is how would they load share, and how similar is the code for these functions?Voice-of-All 15:04, 27 August 2006 (UTC)
Well if my judgements are correct, Pearle doesn't usually edit late in the evening (UTC) which is when my bot usually runs. I don't think sharing the load would be a problem; there are two or three bots that currently update the {{cleanup}} tags, and they work perfectly together. The code between the cleanup and wikify functions is very very similar – the only real differences between them are the words "cleanup" and "wikify". — FireFox (talk) 15:21, 27 August 2006
So the code is a copy (same langauge and text) of Pearle's, just modified?Voice-of-All 16:34, 27 August 2006 (UTC)
Oh, the code isn't similar to Pearle's. I probably mistook the question. FFBot is an AWB bot. — FireFox (talk) 16:51, 27 August 2006
