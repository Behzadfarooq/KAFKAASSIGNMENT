[edit] NolBot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Nol888

Automatic or Manually Assisted: Automatic while supervised

Programming Language(s): C# - AWB

Function Summary: Replaces depreciated templates and/or infoboxes to new types, while keeping information.

Edit period(s) (e.g. Continuous, daily, one time run): Daily (21-23 UTC)

Edit rate requested: 6 edits per min

Already has a bot flag (Y/N): N

Function Details: NolBot is a bot that will use AWB to replace depreciated templates or infoboxes with the new one. Using AWB's find and replace feature, the bot will preserve the data and rename the parameter if required. If required, I may change the basic find and replace to regular expressions.
