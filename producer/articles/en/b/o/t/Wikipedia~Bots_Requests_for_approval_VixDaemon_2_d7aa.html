[edit] VixDaemon

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: KyraVixen

Automatic or Manually Assisted: Manual page gathering, automatic run

Programming Language(s): AutoWikiBrowser

Function Summary: Change {{SharedIP}} to {{SharedIPEDU}} on the 137.155 pages listed here.

Edit period(s) (e.g. Continuous, daily, one time run): One time run, 134 pages

Edit rate requested: 5 edits per minute (about a 10 second delay in AWB)

Already has a bot flag (Y/N): Y

Function Details: When I was gathering shared IP addresses from the public terminals on my campus, I was not aware that the {{SharedIPEDU}} template existed, so I just tagged them with {{SharedIP}}. Now that I know that we have a template specifically for shared IPs belonging to educational institutions, I'd like to change them to better reflect the IPs' owner, and not clutter my own history with this change.

I have already formed the list of pages to modify in my sandbox, which is where AWB will import the pages to modify. Since I tagged them all the same way, the search string would be {{SharedIP|[[Christopher Newport University]]}}, and it would be replaced with {{SharedIPEDU|[[Christopher Newport University]]}}. No other modifications will be made to the page (ie, no RETF, no general fixes, etc).

[edit] Discussion

Speedily Approved. —Mets501 (talk) 01:51, 8 February 2007 (UTC)
