[edit] SatyrBot

Operator: SatyrTN

Programming Language(s): Pywikipedia

Function Summary: To be used to help correct some common errors and generally help organize articles relating to the LGBT WikiProject. For instance:

  1. Review pages linked to the {{LGBT-stub}} for placement in WikiProject LGBT studies/Assessment
  2. Same for Category:LGBT and sub-categories.


Edit period(s) (e.g. Continuous, daily, one time run): One-time runs on an as-needed basis.

Edit rate requested: 4 edits per minute

Already has a bot flag (Y/N): N

Function Details: To be used to help correct some common errors and generally help organize articles relating to the LGBT WikiProject. For instance:

  1. Review pages linked to the {{LGBT-stub}} for placement in WikiProject LGBT studies/Assessment
  2. Same for Category:LGBT and sub-categories.

Since WP:LGBT is just gearing up, we're in the middle of some fairly routine procedures like the Assessment project mentioned above. A bot will help us flesh out (so to speak) articles and get to work on improving those that are needed.

[edit] Discussion

Since this is all edits are manually confirmed, you don't need explicitly need permission. However, I'm going to assume that you're seeking the bot flag, which is a reasonable request given the edit rate and number of edits. Could you provide me with a few diffs showing the exact kinds of changes that the bot will be making? Once you provide those, you should be approved speedily, unless there is some other concern in the meantime. -- RM 19:41, 15 December 2006 (UTC)

Speedily Approved. I've looked over the edits that you provided and I'm approving this bot to run manually with the bot flag. Since it is being run manually, we technically can't enforce a rate, because this falls outside the scope of our "jurisdiction". Nevertheless, as a courtesy please keep the rate at 6 writes / minute or less on average if possible. Just for your information, we do have a number of other bots doing this same thing, many doing it automatically. As such, if you wanted to run your bot automatically in the future, it would not be difficult to reactivate this request for the expansion of this task. -- RM 16:17, 17 December 2006 (UTC)
