[edit] STTWbot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: STTW

Automatic or Manually Assisted: Automatic

Programming Language(s): AWB

Function Summary: Posting newsletter to France and Germany WikiProject members and welcoming new members to the respective WikiProjects.

Edit period(s) (e.g. Continuous, daily, one time run): Twice a month

Edit rate requested: 6 edits per minute

Already has a bot flag (Y/N): Y

Function Details: STTWbot has been tagging France and Germany WikiProject related articles since some time now. Now as the projects have grown, the bot would be used in distributing project newsletters, for example users in Cat:WikiProject France members. Also in further expanding the project by welcoming/informing users from Cat:User fr-N and Cat:User de-N about the projects.

[edit] Discussion

I have no problem with the newsletter distribution; you can run a trial of that if you'd like. I do have a problem with spamming the talk pages of all users in Cat:User fr-N and Cat:User de-N though; yes, it would mean more exposure for the project, but very annoying. Also, bots to welcome new users have been denied in the past and as of now, will not be approved. —METS501 (talk) 21:58, 7 March 2007 (UTC)

OK ! I have changed the bot request as per suggestions. STTW (talk) 09:29, 8 March 2007 (UTC)

Approved for trial. Next time a newsletter needs distributing, try it out with the bot, and post back showing diffs. —METS501 (talk) 18:12, 10 March 2007 (UTC)
