[edit] GimmeBot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: User:Gimmetrow

Automatic or Manually Assisted: Automatic

Programming Language(s): Python

Function Summary: Removing, adding, or renaming templates

Edit period(s): as needed

Edit rate requested: 6 edits per minute

Already has a bot flag (Y/N): Y

Function Details: The bot was originally designed to work with multiple FA, GA and PR-related templates, which required procedures to handle templates in general. The code library can handle various template-related tasks, such as removing templates, adding templates, adding new parameters to existing templates, or renaming templates. The bot currently removes unapproved uses of the "featured article" star from articles, and was able to remove instances of a deleted template. The code could be used to remove templates after TfD, or to add wikiproject templates. I would probably do any additions of templates or parameters with supervised approval of each edit to avoid adding multiple templates. Removing templates is (generally) straightforward and could probably run automatically.
