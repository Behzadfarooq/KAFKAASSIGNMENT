[edit] TaeBot

tasks • contribs • count • logs • page moves • block user • block log • flag log • flag bot

Operator: Taestell

Automatic or Manually Assisted: Manually assisted.

Programming Language(s): Using only AWB at this time.

Function Summary: Performing various automatable tasks. Current tasks in mind: updating infoboxes, simple recategorization, etc.

Edit period(s) (e.g. Continuous, daily, one time run): Individual one-time runs for various individual tasks.

Edit rate requested: Expecting an average of 2-4 edits/min for most tasks; expect about 4 edits/minute maximum.

Already has a bot flag (Y/N): No

Function Details:

  • Simple infobox updating for WP Albums (such as Extra album cover -> Extra album cover 2) [manual/without autosave].
  • Recategorization per community consensus [with autosave].
  • Apply general fixes using AWB:
    • Fixes common mistakes in "see also" and "external links" sections, removes excess white space.
    • Sorts interwiki links alphabetically (individually selectable in menu), and puts them at the bottom of the page with stubs.
    • Unicodifies interwiki links.
    • Removes duplicate interwikis and categories.
    • Puts categories after article body, followed by interwiki links and stub templates. Recognises some comments as cat and interwiki headers.
    • Adds bullet points to external links after the ==External links==.
    • Replaces italic and bold html markup with wiki markup.
    • Repairs bad links.
    • Simplifies links like [[Dog|Dog]] to [[Dog]].
    • Simplifies links like [[Dog|Dogs]] to [[Dog]]s.
    • Adds bold text to the first occurrence of the title of the article (if there is no other bold text).
    • De-links self referencing wiki-links.
  • Fix typos with RegExTypoFix [manual/without autosave].

[edit] Discussion

Please define every task as each task needs approved. Betacommand (talk • contribs • Bot) 06:04, 27 October 2006 (UTC)

If you just want to use WP:AWB to update some articles, but not in BULK you don't need a bot account (although you are welcome to have one). Having a bot account has many benefits, including increased edit rate, but bot approvals can not be open-ended. — xaosflux Talk 13:32, 27 October 2006 (UTC)
I was making edits using AWB under my username, but found that the edit rate was getting rather high. Additionally, having the bot will allow me to save some changes automatically. I have updated the function details above. —taestell 14:34, 27 October 2006 (UTC)
As per my bot request above, those edits without the typo fixing, would be classable as 'insignificant', and you CANNOT do the typo fixing using the bot automatic mode Reedy Boy 09:26, 29 October 2006 (UTC)
Yes, I only plan on turning on typo fixing when I am doing a manual review of all changes. For automatic tasks, typo fixing and "general edits" will be turned off. I should also say that I will not run this bot just to fix typos and general problems. This bot will not be yet another "typo fixing"/"general cleanup" bot--the edits will be focused on more significant issues. —taestell 17:21, 29 October 2006 (UTC)
Where does it get the list of pages to edit? Cat:cleanup?Voice-of-All 16:54, 19 November 2006 (UTC)
The list of pages to be edited will be generated from "what links here" (transclusion) for the templates that will be updated. One conversion will be updating Template:Extra album cover to Template:Extra album cover 2 by fairly simple search-and-replace functions, generated from the list of what pages link to Template:Extra album cover. —taestell 20:42, 20 November 2006 (UTC)

Approved for trial. All of the manual tasks really don't need approval except for a faster edit rate. The automatic tasks however do. In any case, I'd like to see a trial run of this bot for all tasks before granting the bot flag. Please do a trial bot run of up to 100 edits at no faster than 1 per minute and post the results here within 1 week. -- RM 13:35, 28 November 2006 (UTC)

I am currently unable to use AWB with TaeBot because "Taebot" (with a lower-case "b") was added to the approval list. I will not be able to make any edits with the bot until this is fixed. —taestell 17:06, 3 December 2006 (UTC)
So far, TaeBot has performed 60 edits, doing the following tasks to maintain album infoboxes: converting "extra chronology" and "extra album cover" templates into "extra chronology 2" and "extra album cover 2" (respectively); removing "extra album cover" templates when the image itself has been deleted from Wikipedia; and manually disambiguating "type=album" when more specific type is known (i.e., "type=studio album"). AWB has also identified and performed several "general fixes" (all manually-approved by me). RegexTypoFix was also enabled but no typos were identified in these first 60 edits. The only other task for TaeBot that I have not demonstrated yet would be converting album ratings into the new standard form ((3/5) into ). —taestell 03:18, 10 December 2006 (UTC)
Are you planning on testing the remaining feature(s) or are you just seeking approval for the stuff so far? -- RM 14:58, 18 December 2006 (UTC)
Hmm, well I think this demonstrates what the bot will be used for. Is there anything specific that someone wants me to illustrate? —taestell 04:33, 28 December 2006 (UTC)
Are there any other comments about the bot? —taestell 18:21, 26 January 2007 (UTC)
You mentioned above that you have not demonstrated converting album ratings into the new standard format. So my question was this: Are you seeking approval for that task or just the ones that you have demonstrated? We need to see examples of each task before we can approve it. -- RM 19:16, 26 January 2007 (UTC)

Approved. This bot shall run with a flag. (By the way, RM, see [1] and [2] for examples of the album rating converting). —Mets501 (talk) 19:22, 26 January 2007 (UTC)
