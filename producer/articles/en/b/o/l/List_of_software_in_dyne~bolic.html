[edit] Video

[edit] Producing

  • Cinelerra
  • LiVES
  • FreeJ
  • Kino
  • MEncoder
  • XawTV
  • Transcode, for converting video format

[edit] Playing

  • MPlayer
  • FFmpeg

[edit] Streaming

  • Mp4Live — streaming MPEG4
  • Palantir — streaming MJPEG
  • HasciiCam — converting webcam images to ASCII images

[edit] Networking

  • Netsukuku - mesh network-based network layer protocol

[edit] Clustering

  • openMosix - cluster-management system

[edit] Word Processor

  • AbiWord
  • Ted
  • Scribus — Desktop publisher and output to PDF and PostScript
  • Bluefish — used to edit HTML script
  • Nedit — editing source codes
  • Antiword — a commandline to convert .doc into plain text

[edit] Software module in dyne:bolic version 2

  • Default: general media production and boardcasting tools. There are total five modules available: audio, network, text, network, and video
  • Devel: various development tools
  • OpenOffice.org 2.0
