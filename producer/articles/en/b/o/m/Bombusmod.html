Bombusmod is a modified version of the Bombus instant messenging client for mobile phones. As does the official version, it works with the Jabber/XMPP protocol. Bombusmod by nature being a Jabber client allows the user to connect to a variety of messenging programs including ICQ, Yahoo, MSN, AIM and Google Talk through the use of "transports".

[edit] Features

  • Client-specific roster icons
  • Customisable roster grouping
  • Wide array of cross-client compatible emoticons
  • Composing events (dependent on contacts client)
  • Connect to AIM, ICQ, MSN, Yahoo and Gtalk
  • Comprehensive colour scheme customiser built in
  • Alert profile can be selected
  • Set your status (e.g. DND, Invisible)
  • Auto status away
  • Keylock shortcut (with clock)
  • Minimise feature
  • Search for, navigate and moderate jabber conferences
  • Supports muc protocol
  • Copy/paste feature
  • Manage multiple accounts
  • Save messages on your "archive" or as a "template"
  • Block specific contacts
  • Manage your Vcard and view other users Vcards
  • Traffic data counter
  • Interface can be set to English, Russian or Polish (default English)
  • Supports RSS and atom news feeds
  • Constructor website allows user to customise application before downloading
