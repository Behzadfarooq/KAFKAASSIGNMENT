Bombus (software)

From Wikipedia, the free encyclopedia

Bombus

Bombus roster window screenshot
Developer: Eugene Stahov (evgs)
Latest release: 0.4.0.680 / September 22, 2006
OS: J2ME/MIDP
Use: Instant messaging client
License: New BSD license
Website: bombus-im.org

Bombus is a free software instant messaging client for the Jabber/XMPP protocol. It is written in Java, and runs on J2ME/MIDP capable cellular phones.

[edit] Features

  • Roster groups
  • Roster bookmarks
  • Multi-user conferences
  • Ignore list and Privacy lists editor
  • Registration on servers and services
  • List of alternative accounts, fast switching between them
  • Service Discovery
  • Jabber Search
  • Ad-hoc commands (i.e. performing tasks on remote clients/services)
  • vCard editor
  • Logging messages and status changes to a text file
  • SSL connections
  • Auto key lock
