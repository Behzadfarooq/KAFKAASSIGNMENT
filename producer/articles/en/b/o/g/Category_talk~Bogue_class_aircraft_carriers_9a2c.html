[edit] Proposed moves to RN names

Ok, I've decided that rather than have move discussions on every single ship's page, it'd be much better to have a single discussion here.

Here's the story with the Bogue class aircraft carriers. The majority were transferred to the Royal Navy under Lend-Lease either immediately upon commissioning or within a few weeks. When these ships were returned to the US, they were all scrapped or sold as merchant ships. None of the Lend-Lease ships ever served as aircraft carriers in the US Navy.

I believe Bogue class carriers which never saw USN service should be listed under their RN names. Some are already listed this way, but most are listed under their American names. Below is a list of moves that need to occur:

  • USS Prince William (CVE-19) → HMS Striker (D12)
  • USS Vermillion (CVE-52) → HMS Smiter (D55)
  • USS St. Simon (CVE-51) → HMS Arbiter (D31)
  • USS St. Joseph (CVE-50) → HMS Ruler (D72)
  • USS St. Andrews (CVE-49) → HMS Queen (D19)
  • USS Perdido (CVE-47) → HMS Trouncer (D85)
  • USS Niantic (CVE-46) → HMS Ranee (D03)
  • USS Prince (CVE-45) → HMS Rajah (D10)
  • USS Keweenaw (CVE-44) → HMS Patroller (D07)
  • USS Jamaica (CVE-43) → HMS Shah (D21)
  • USS Edisto (CVE-41) → HMS Nabob (D77)
  • USS Delgada (CVE-40) → HMS Speaker (D90)
  • USS Carnegie (CVE-38) → HMS Empress (D42)
  • USS Bastian (CVE-37) → HMS Trumpeter (D09)
  • USS Bolinas (CVE-36) → HMS Begum (D38)
  • USS Baffins (CVE-35) → HMS Ameer (D01)
  • USS Pybus (CVE-34) → HMS Emperor (D98)
  • USS Glacier (CVE-33) → HMS Atheling (D51)
  • USS Winjah (CVE-54) → HMS Reaper (D82)
  • USS Willapa (CVE-53) → HMS Puncher (D79)
  • USS Sunset (CVE-48) → HMS Thane (D48)
  • USS St. George (CVE-17) → HMS Pursuer (D73)
  • USS Hamlin (CVE-15) → HMS Stalker (D91)
  • USS Estero (CVE-42) → HMS Premier (D23)
  • USS Croatan (CVE-14) → HMS Fencer (D64)
  • USS Cordova (CVE-39) → HMS Khedive (D62)
  • USS Chatham (CVE-32) → HMS Slinger (D26)
  • USS Breton (CVE-10) → HMS Chaser (D32)
  • USS Block Island (CVE-8) → HMS Hunter (D80)
  • USS Barnes (CVE-7) → HMS Attacker (D02)

Unfortunately, all of these moves are obstructed, so this has been listed on Wikipedia:Requested moves. It seems uncontroversial to me, so hopefully it will happen quickly and smoothly. Please weigh in on this issue below.
