Images in this category are claimed to be fair use book cover images under United States copyright law. Public domain book covers do not need to be in this category, and should simply be tagged with the relevant public domain template.

To place an image in this category, add the tag {{book cover}} to the bottom of the image's description page. If you are not sure which category an image belongs to, consult the image copyright tag page. If this category is very large, please consider adding a table of contents ({{CategoryTOC}}) or place your image in a new or existing subcategory.
NB 
{{Book cover}} can be customised with an optional parameter. If you add {{book cover | Title = title of book }} to the image, it will appear below under the appropriate initial letter.
