Please do not add inappropriate external links to Wikipedia, as you did to Wikipedia:Book_sources. Wikipedia is not a mere directory of links, nor should it be used for advertising or promotion. Inappropriate links include (but are not limited to) links to personal web sites, links to web sites with which you are affiliated, and links that attract visitors to a web site or promote a product. See the external links guideline and spam policy for further explanations. Since Wikipedia uses nofollow tags, external links do not alter search engine rankings. If you feel the link should be added to the article, then please discuss it on the article's talk page before reinserting it. See the welcome page to learn more about Wikipedia.
