[edit] website-blog

Without parameters, links to Wikipedia articles website and blog

and a blog.
{{User:Booyabazooka/Userboxes/website-blog}}

Optional parameters website and blog specify the URL of each

This user has a website
and a blog.
{{User:Booyabazooka/Userboxes/website-blog
| website = http://en.wikipedia.org/
| blog = http://en.wikipedia.org/}}
