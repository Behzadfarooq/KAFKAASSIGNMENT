Bookmarklet

A bookmarklet is a small JavaScript program that can be stored as a URL within a bookmark in most popular web browsers, or within hyperlinks on a web page. Because Internet Explorer uses the term favorites instead of bookmarks, bookmarklets are also less commonly called favelets by users.

Bookmarklets can be saved and used like normal web page bookmarks. Therefore, they are simple "one-click" tools that can add substantial functionality to the browser. For example, they can:

  • Modify the way a web page is displayed within the browser (e.g., change the font size, background color, etc.).
  • Extract data from a web page (e.g., hyperlinks, images, text, etc.).
  • Jump directly to a search engine, with the search term(s) input either from a new dialog box, or from a selection already made on a web page.
  • Submit the page to a validation service.

[edit] Concept

The browser uses URI's for bookmarks and the href attribute of an <A> tag. The first part of the URI, such as http:, file:, or ftp:, specifies the kind of services and the form for the rest of the string. However, the browsers implement a prefix javascript: that to the parser is just like any other URI. Internally, the browser sees that the protocol is javascript and instead of using the rest of the string to fetch the page to load, it executes the string as a program and uses the resulting string as the new page.

However, the executing script has access to the current page, which it may inspect and change. And, if the return is an undefined type rather than a string, no new page is loaded and the only effect was to run the code against the current page.

[edit] Examples

The following set of codes, (linebreaks added for readability) when used as the destination of a bookmark, will search en.wikipedia.org for a word you currently have highlighted on a web page, or else, if nothing is highlighted, it will display a dialog box form field in which to enter a search term. This bookmarklet works in Firefox and Konqueror (Minitools):

You may need to copy & paste each line separately, concatenating them together into one long line.

javascript:(function(){
 q=document.getSelection();
 if(!q) q=prompt('Wikipedia:');
 if(q) location.href='http://en.wikipedia.org/w/wiki.phtml?search=' + escape(q);
 })()

This version works in Apple's Safari browser:

javascript:void(
 q=prompt('Wikipedia:',getSelection()));
 if(q) void(location.href='http://en.wikipedia.org/w/wiki.phtml?search=' + escape(q))

This version works in Internet Explorer:

javascript:(function(){
 q= document.selection.createRange().text;
 if(!q) q=prompt('Wikipedia:','');
 if(q) location.href='http://en.wikipedia.org/w/wiki.phtml?search=' + escape(q);})()

To install the previous bookmarklet in Internet Explorer, follow these steps:

  • Make sure the Links toolbar is visible (ensure View/Toolbars/Links is checked).
  • In the Favorites menu, select Add to Favorites...
  • Type "Wikipedia" in the Name field.
  • In the Create In box, select the Links folder. Click OK.
  • Copy the text of the bookmarklet, above.
  • Right-click on the new "Wikipedia" link in the Links bar, and select Properties.
  • Paste the bookmarklet text into the URL field. You may need to copy/paste each line separatly, concatenating them together onto one long line. Click OK.

The next example illustrates a striking change to the layout of this page, and works by adding an embedded style sheet which sets the right margin to 45%.

javascript:(function(){
 var newSS, styles='body { margin-left:0%; margin-right:45%;}';
 if(document.createStyleSheet) {
   document.createStyleSheet(%22javascript:'%22+styles+%22'%22); 
   } else {
   newSS=document.createElement('link'); 
   newSS.rel='stylesheet';  
   newSS.href='data:text/css,' + escape(styles);
   document.getElementsByTagName(%22head%22)[0].appendChild(newSS); 
   } 
 }
)()

The next example will take all of the images on the current page and blow them around the webpage:

javascript:
 R=0; 
 x1=.1;  y1=.05;  x2=.25;  y2=.24;  x3=1.6; 
 y3=.24; x4=300; y4=200; x5=300; y5=200;
 DI=document.images; 
 DIL=DI.length; 
 function A(){
   for(i=0; i-DIL; i++){
      DIS=DI[ i ].style;
      DIS.position='absolute'; 
      DIS.left=Math.sin(R*x1+i*x2+x3)*x4+x5;
      DIS.top=Math.cos(R*y1+i*y2+y3)*y4+y5
      }
   R++
   }
 setInterval('A()',5);
 void(0);

[edit] History

Steve Kangas of www.bookmarklets.com coined the term "bookmarklet," which he started to create based on an idea suggested in the Netscape JavaScript Guide. The term favelet was used early on by Tantek Çelik on 6 September 2001 (personal email). Brendan Eich, who developed JavaScript at Netscape, gave this account of his invention of the javascript: URL (which makes bookmarklets possible):

They were a deliberate feature in this sense: I invented the javascript: URL along with JavaScript in 1995, and intended that javascript: URLs could be used as any other kind of URL, including being bookmark-able. In particular, I made it possible to generate a new document by loading, e.g. javascript:'hello, world', but also (key for bookmarklets) to run arbitrary script against the DOM of the current document, e.g. javascript:alert(document.links[0].href). The difference is that the latter kind of URL uses an expression that evaluates to the undefined type in JS. I added the void operator to JS before Netscape 2 shipped to make it easy to discard any non-undefined value in a javascript: URL.

– Brendan Eich, Source unknown

[edit] See also

  • The Slayer Office Suite offers a variety of bookmarklets specifically for web development.
  • The Bookmarklet Builder helps create and edit your own bookmarklets.
  • Sample Bookmarklets and JavaScript Editor
  • gibney.org Bookmarklets
  • Jesse's Bookmarklets Site offers a wide selection of high quality bookmarklets.
  • CyberNet News has a nice list of bookmarklets and specifies the compatible browsers.
  • Lorelle offers a list of bookmarklets useful for web design and development, especially useful for bloggers.
  • Greasemonkey enables functionality similar to that offered by bookmarklets —with one substantial addition—"bookmarklets" can be automatically loaded when a given website is loaded. In Greasemonkey terminology, these scripts that execute when a matching URL is loaded are called "User Scripts". Mark Pilgrim's DiveIntoGreaseMonkey.org is a free book about Greasemonkey.
  • iMacros enables functionality similar to that offered by bookmarklets in both, Firefox and Internet Explorer. It adds the ability to visually record and replay the bookmarklets (macros) and to automate a wider range of tasks like single sign-on, form filling, web data extraction, web performance testing or printing.
  • Blummy is a kind of drop down menu that can be used to manage one's bookmarklets. Displays as a drop down menu in the current page.
