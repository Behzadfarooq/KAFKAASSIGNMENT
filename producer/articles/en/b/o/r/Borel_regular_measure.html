Borel regular measure

In mathematics, an outer measure μ on is called Borel regular if the following two conditions hold:

  1. every Borel set is μ-measurable
  2. for every set (which need not be μ-measurable) there exists a Borel set such that and

An outer measure satisfying only the first of these two requirements is called a Borel measure, while an outer measure satisfying only the second requirement is called a regular measure.

The Lebesgue outer measure on is an example of a Borel regular measure.
