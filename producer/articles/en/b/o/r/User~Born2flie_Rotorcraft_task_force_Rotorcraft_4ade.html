[edit] Project banner

The {{Rotorcraft}} project banner template should be added (not subst:ed) to the talk page of every article within the scope of the task force. While the template does not require any additional parameters, it has some optional ones that add extra features to the banner. The full syntax is as follows:

{{AircraftProject
|class=  
|importance=
}}

General parameters:

  • class – a rating of the article's quality; see the assessment department for more details.
  • importance – a rating of the article's importance; see the assessment department for more details.

To avoid needlessly cluttering up talk pages, it is usually appropriate to remove any unused parameters from the template.
