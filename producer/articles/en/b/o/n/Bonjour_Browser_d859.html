#BonjourBrowser

Bonjour Browser is a Creative Commons licensed Mac OS X application that displays all services declared using Bonjour. The program was originally called "Rendezvous Browser", but changed it's name in version 1.5.4 after Apple changed the protocol's name to Bonjour. For certain protocols, double clicking a list item will launch the associated helper. 1.5.6 is the first universal binary version.

Future versions will allow users to completely define a service, instead of relying on the author to do so.

Bonjour Browser was recommended for service discovery in MacAddict #123.
