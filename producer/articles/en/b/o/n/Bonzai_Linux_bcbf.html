Bonzai Linux is a Debian-based Linux distribution that fits on a 180 MB MiniCD (8 cm single) CD-R or CD-RW. It includes the current stable version of KDE and has been modified for easier installation. It was formerly known as miniwoody which refers to Debian's 3.0 release (codenamed Woody) on which it is based.

The most recent release is version 3.2, released on January 14, 2004.[1]
