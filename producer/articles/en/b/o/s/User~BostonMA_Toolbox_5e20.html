[edit] How to handle sockpuppets

  1. to the sock user page, add:
    {{subst:socksuspect|PUPPETMASTER}}
    [[Category:Suspected Wikipedia sockpuppets of PUPPETMASTER|{{PAGENAME}}]]
    [[Category:Suspected Wikipedia sockpuppets|{{PAGENAME}}]]
  2. go to evidence page linked by just created template and paste:
    {{subst:Socksuspect2| 1=PUPPETMASTER|2=PUPPET}}
    into page and save.
  3. go to the page Category:Suspected Wikipedia sockpuppets of PUPPETMASTER and add:
    [[Wikipedia:Suspected sock puppets/PUPPETMASTER]]
  4. add the line
    {{Wikipedia:Suspected sock puppets/PUPPETMASTER}}
    here
  5. Notify the suspect
    {{subst:Socksuspectnotice|1=SOCKPUPPETEER}} ~~~~
  6. Add sockpuppetteer tag
    {{sockpuppeteer}}

[edit] User Page Deletions

"Just add to the page: {{db-userreq}}. A sysop will then delete it after checking that the page does not contain evidence of policy violations that may need to be kept. If there has been no disruptive behavior meriting the retention of that personal information, then the sysop can delete the page straight away in order to eliminate general public distribution of the history containing the information.
