[edit] Subst

Hi, when you post a welcome message to a newcomer, could you use {{subst:welcome}}, {{subst:welcome2}}, etc., instead of just {{welcome}}, {{welcome2}}, etc. It means that the text of that particular template will be added to the talk page of the user instead of the user just seeing a sort of mirror image of it. Two advantages are that if the actual template is vandalized, the message you have sent will not be altered in any way, and also less load is placed on the server. For more information, please see WP:SUBST. Cheers. Mkdwtalk 09:20, 22 November 2006 (UTC)
