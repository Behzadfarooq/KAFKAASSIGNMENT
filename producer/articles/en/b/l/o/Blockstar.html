Blockstar

Blockstar Website Builder
Developer: Blockstar
Latest release: 2.9 / Jul 06, 2006
OS: Cross-platform
Use: HTML editor
Website: www.blockstar.com

Blockstar is a WYSIWYG HTML editor and web hosting service allowing users to create blogs, photo galleries, and web pages. Blockstar was created by PIC Web Services, Inc., a Delaware corporation founded in 2002 by Chris Furlong and Teodor Zlatanov. The company received venture funding from WeddingChannel.com in 2004.

Blockstar uses a new method of web page construction that eliminates the need for templates by storing both the information to be presented and its ordering in a database as nested hierarchical ordering instructions. This method allows Blockstar to combine a database driven content management system with a WYSIWYG web page builder. PIC Web Services has applied for a US patent titled "Computer systems and methods for platform independent presentation design" to cover the method of building web pages used in Blockstar.

Blockstar is available for Linux, Mac OS X and Microsoft Windows. The software can be downloaded directly from Blockstar.com or from independent software distribution websites like Downloads.com, Softpedia.com, Freewarefiles.com, and ZDNet.com.

Blockstar also maintains a web page containing a timeline of early blogs. This page has become an authority on early blogs, and has been referenced by sources such as the Duke Law & Technology Review and Google Answers.
