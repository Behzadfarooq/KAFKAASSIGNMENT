[edit] Unspecified source for Image:CHICK fIT.png

Thanks for uploading Image:CHICK fIT.png. I notice the file's description page currently doesn't specify who created the content, so the copyright status is unclear. If you have not created this file yourself, then there needs to be a justification explaining why we have the right to use it on Wikipedia (see copyright tagging below). If you did not create the file yourself, then you need to specify where it was found, i.e., in most cases link to the website where it was taken from, and the terms of use for content from that page.

If the file also doesn't have a copyright tag, then one should be added. If you created/took the picture, audio, or video then the {{GFDL-self}} tag can be used to release it under the GFDL. If you believe the media meets the criteria at Wikipedia:Fair use, use a tag such as {{fairusein|article name}} or one of the other tags listed at Wikipedia:Image copyright tags#Fair_use.   1. Please do not insert unverified speculation into Wikipedia, because Wikipedia is not a crystal ball. This includes the creation of articles on albums that haven't yet been officially announced, such as TBA Ashlee Simpson, TBA Mariah Carey etc., and uploading fake album covers to put in these articles. For albums that haven't yet been announced, I'd recommend editing the article on the singer and adding the information about any forthcoming albums there. Remember to reliable sources to accompany the information.
  2. Please do not copy and paste text material from copyrighted websites. If the information is relevant to include, rewrite it in your own words, but do not simply copy and paste because that is plagiarism and will be removed because it could get Wikipedia into serious trouble.
  3. Please do not copy or cut content from one article to another without mentioning the original article in the edit summary. Thanks. Extraordinary Machine 17:47, 21 February 2007 (UTC)
