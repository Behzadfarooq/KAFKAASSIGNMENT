Hello Wikipedians,

As I mentioned on my userpage we've put together a RSS (file format) Feed so that anyone can get any Wikipedia topic via RSS. The feed was produced by me, parsed by Christian Roy and is hosted on my servers at BlinkBits. The feed is cached to prevent runtime queries of Wikipedia.

A few sites have asked me about indexing and or pulling these feeds so over the few months BlinkBits could have tens of millions of requests. I am doing my best to keep as many of these away from the Wikipedia servers so only about 600,000 should come through to wikipedia over this period of time and my aim is to have most of these come during low traffic periods of time.

The latest version validates but it still needs some formatting work on some wikipedia pages that use tables, images, etc... We'd love to improve upon this even more and publish it under an open-source license so let either of us know if you are interested in helping put the finishing touches on the feed.

The feed works for every topic in the following languages:

  • English - en
  • Dutch - nl
  • French - fr
  • German - de
  • Spanish - es
  • Italian - it
  • Swedish - sv

To use the feed just replace language and the topic_name with the exact wikipedia topic to the end of url.

  • Here is the url format: http://www.blinkbits.com/en_wikifeeds_rss/Topic_name
  • Here is an example: http://www.blinkbits.com/en_wikifeeds_rss/Derek_Jeter

Also, you can use this to track your own user profile and changes by using this file format.

  • http://www.blinkbits.com/en_wikifeeds_rss/User:Blinklmc

Replace Blinklmc with your name...

All feeds also includes attribution back to wikipedia and the GNU comment so please do not use the feed without providing wikipedia proper attribution etc...

If you are a wikipedia sysops please let me know if you'd like to add the RSS feed to each of the topic pages or use it in some other fashion.

-- User:Blinklmc
