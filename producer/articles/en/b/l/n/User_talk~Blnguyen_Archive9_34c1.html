[edit] Block of User:172.144.0.222

FYI: I removed your 48 hour block to impose a longer one. In my opinion, there is no reason for us to tolerate someone who is clearly a returning vandal who is using intentionally profane edit summaries. Wikibofh(talk) 03:33, 26 June 2006 (UTC)

I have a major problem with blocks like this. There is no actual evidence that this is a returning vandal. And by blocking an IP that is probably dynamic or shared for a month, you are creating a large possibility for collateral damage. --Woohookitty(meow) 08:38, 26 June 2006 (UTC)
You can change it if you want then - I'm a dunce with IP technics - so do as you please with the technical details.Blnguyen | rant-line 08:41, 26 June 2006 (UTC)
Well with IPs, you have 3 types. You have static IPs. Those are folks who are on the same IP address every time they logon. Those are easy to spot because they generally edit the same articles over and over again. People using cable modems or DSLs are usually static.
Then you have shared IPs. These are often at schools. Several students share the exact same IP address. So blocking the IP blocks every user using the IP from editing. 90% of the time, the talk page of the IP user says it's from such and such a school. This almost always means shared IP.
Finally, you have dynamic IPs. With them, their IP # changes every time they go online...and with AOL and other ISPs, it'll even change while they are online. These you can spot by either looking at the talk page (where it says it's from an ISP or whatever). Or the other clue is an edit history with edits to totally different articles. The problem with blocking dynamic IPs for longer than say an hour or 3 hours is that once that vandal logs off, his IP is assigned to a different person. So if you block the IP for longer than a few hours, you are probably stopping legit editors.
If you have any questions, let me know. I'm not an expert either. Just knowing the difference between the 3 types will help you out alot as a new admin. I mean, blocking an IP that might be dynamic or shared for 1 week could literally be blocking 40-50 users (or more) from editing. We can always block longer later. --Woohookitty(meow) 08:54, 26 June 2006 (UTC)
