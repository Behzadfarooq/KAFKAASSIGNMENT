This template compares the current day of the month to a set value, and if the value matches, the target cell in a table (usually a calendar) is highlighted in blue  . A zero (0) parameter value can be used as a placeholder to prevent the template from operating until the appropriate month/date. This template is currently used in Calendars of 2006 and Calendars of 2007.
