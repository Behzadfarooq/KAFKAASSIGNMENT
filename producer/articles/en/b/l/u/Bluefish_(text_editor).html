Bluefish

Bluefish 1.0 with default toolbars & XHTML
Developer: Bluefish Dev Team
Latest release: 1.0.7 / October 30, 2006
OS: Linux, Mac OS X, OpenVMS, Unix (POSIX)
Use: Text editor
License: GPL
Website: bluefish.openoffice.nl

Bluefish is a free software/open source text editor for POSIX-conforming operating systems such as Linux and Mac OS X. Its use is suitable for many programming and markup languages, with particular focus on their use for web development.

[edit] Syntax support

  • XML
  • HTML
  • XHTML
  • CSS
  • PHP
  • Python
  • C
  • Java
  • JavaScript
  • JSP
  • SQL
  • Perl
  • ColdFusion
  • Pascal
  • R
  • GNU Octave/MATLAB
