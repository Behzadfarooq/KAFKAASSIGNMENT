[edit] Images in userspace

Nice images. FYI, fair use images aren't allowed in userspace like your userpage. They have to be used educationally for it to be fair use, and userpages don't count. I used to have like 50 on mine, till I learned (I did the first 12 Black Lagoon images). You can put a colon in front of the image name within link brackets and the name will show up instead of the image like Image:Black Lagoon 01 Captain.jpg. - Peregrinefisher 19:12, 4 January 2007 (UTC)

What I was planning to do originally, but couldn't figure out how to. Thanks for the heads up. --BrokenSphere 19:43, 4 January 2007 (UTC)

Thanks for uploading Image:MB16_Principal.jpg. In addition to the boilerplate fair use template, you must also write out on the image description page a specific explanation or rationale for why using this image in each article is consistent with fair use.

Please go to the image description page and edit it to include a fair use rationale.
