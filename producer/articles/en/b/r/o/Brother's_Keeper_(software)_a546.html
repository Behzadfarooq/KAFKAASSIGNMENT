Brother's Keeper (software)

From Wikipedia, the free encyclopedia

Brother's Keeper
Developer: John Steed
Latest release: 6.2.42 / February 18, 2007
OS: Windows
Use: Genealogy software
License: Shareware
Website: www.bkwin.net

Brother's Keeper is a popular genealogy program for Windows that is distributed as shareware. Version 6.2 works with Windows 95, 98, ME, NT, 2000, XP and Vista. The DOS version is no longer updated, but the latest version 5.2B for DOS is still available.


[edit] International versions

The software is written in English but version 6.2 has been translated and is fully available in the following languages:

  • Czech
  • Danish - Dansk
  • French
  • German
  • Icelandic
  • Norwegian - Norsk

Additionally the software is available, without help files, for the following languages:

  • Afrikaans
  • Catalan
  • Dutch
  • Finnish
  • Polish
  • Portuguese
  • Russian
  • Slovenian
  • Swedish
