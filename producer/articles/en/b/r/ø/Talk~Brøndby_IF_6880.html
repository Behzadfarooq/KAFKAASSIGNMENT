[edit] GA

I have promoted this to good article status. It met most of the criteria, but failed on one which I have fixed myself - most of the images were appropriately tagged, but the logo was being used as fair use without a rationale. In fair use cases, it is not enough to include a blanket fair use tag to an image page, as each individual use needs to be justified. Since Image:Brøndby IF.png is being used here to illustrate the team it is the logo of, this should easily be sufficient rationale, so I added that to the image description page. (The bottom line is that although it is fair use here, it would not be fair use if used utterly gratuitously elsewhere - the context is important, which is why a note about the particular context needs to be appended to the image page). Well done to the editors on an excellent article; with a bit of work there is no reason for this not to become a featured article :-) TheGrappler 22:28, 25 March 2006 (UTC)

I think this is an excellent article that doesn't need all that much work to become a featured article. I've been looking at three other featured articles about football clubs (Arsenal F.C., IFK Göteborg and Manchester City F.C.) and compared to those it needs a section about the colours, crest and supporters. I don't know if this is a flaw in this article, but it seems strange that the section about Brøndby Stadium is larger than the main article about Brøndby Stadium. Anyway, with this done I think it should be submitted for a peer review, so it could become a featured article soon. --Maitch 22:01, 11 April 2006 (UTC)
