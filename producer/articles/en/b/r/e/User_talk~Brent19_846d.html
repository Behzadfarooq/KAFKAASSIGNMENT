[edit] License tagging for Image:Jeffparkes.jpg

Thanks for uploading Image:Jeffparkes.jpg. Wikipedia gets hundreds of images uploaded every day, and in order to verify that the images can be legally used on Wikipedia, the source and copyright status must be indicated. Images need to have an image tag applied to the image description page indicating the copyright status of the image. This uniform and easy-to-understand method of indicating the license status allows potential re-users of the images to know what they are allowed to do with the images.

For more information on using images, see the following pages:

  • Wikipedia:Image use policy
  • Wikipedia:Image copyright tags

This is an automated notice by OrphanBot. If you need help on selecting a tag to use, or in adding the tag to the image description, feel free to post a message at Wikipedia:Media copyright questions. 19:07, 14 May 2006 (UTC)

Thanks for uploading Image:Jeff-parkes.jpg. The image has been identified as not specifying the copyright status of the image, which is required by Wikipedia's policy on images.
