[edit] Re: Citing References

I posted a short tutorial over at Iain's talk page on how to start eking out references at Gaelic Traditionalism and thought I'd post it for you over here as well.

At the end of a sentence where you want to cite a source, place a footnote like this: <ref>O Hogain(1991) pp.165-168</ref> which should render like this:[1]

The reference will automagically appear down in the Notes section like this:

  1. ^ O Hogain (1991) pp.165-168

Then put the whole book reference down in the Reference section using this format:

 {{cite book
 | last =O Hogain
 | first =Daithi 
 | authorlink = 
 | coauthors = 
 | year =1991
 | title =Myth, Legend and Romance - An Encyclopaedia of the Irish Folk Tradition
 | publisher =Prentice Hall Press 
 | location = 
 | id =ISBN 0132759594
 }}

Properly formatted, it will appear like this:

  • O Hogain, Daithi (1991). Myth, Legend and Romance - An Encyclopaedia of the Irish Folk Tradition. Prentice Hall Press. ISBN 0132759594.

This is the easiest way which I know of to cite sources, but there are several different ways it can be done. You can learn more about it than you ever wanted to here and here. Hope this helps! WeniWidiWiki 06:34, 3 March 2006 (UTC)
