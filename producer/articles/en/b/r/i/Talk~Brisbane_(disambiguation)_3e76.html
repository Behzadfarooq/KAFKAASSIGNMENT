Can I suggest we get back to basics here! A Disambiguation page is there to clarify between different places/people/things with the same name, not to be a directory for all articles with Brisbane in the title. Thus all entries relating to the Brisbane, Queensland area should either be included in or linked to the main Brisbane article, or frankly they will be found by direct search (for example, searching 'Brisbane River'). See Wikipedia:Manual of Style (disambiguation pages). So I've comletely refreshed the page, removing spurious links but including names (and ships!) not previously mentioned.
