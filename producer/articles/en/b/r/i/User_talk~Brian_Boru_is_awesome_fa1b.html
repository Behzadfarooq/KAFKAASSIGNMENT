[edit] Article namespace

The article namespace refers to the actual articles in the encyclopedia. Other things to do with the encyclopedia that are not articles are in other namespaces. For example your user page User:Brian Boru is awesome is in the user namespace, and my talk page User talk:Reyk is in the user talk namespace. The different namespaces help to organize the various different things connected with the encyclopedia that are not actually articles.

You can find more information about namespaces here.

Hope this helps, Reyk YO! 10:31, 5 December 2006 (UTC)
