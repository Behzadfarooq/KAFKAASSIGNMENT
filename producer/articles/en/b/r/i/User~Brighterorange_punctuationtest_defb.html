This (user sub)page is test input for the Autopunctuation script. It contains many errors that the script is supposed to find. Please feel free to test the script on this page, but don't save it, and don't save any fixes manually!

[edit] Spelling (puSPELL)

We test for a few common spelling errors. If you write "seperately" or "seperate" then we'll fix that. How embarassing!

[edit] Commas (puCOMMA)

If you have commas normally, then nothing happens.

If you have commas with no spaces,then we fix.

If you have too many spaces , then we also fix.

Also this strange combination ,is fixed too.

[edit] Link syntax (puLINKSPACE)

The only thing we do with links is remove trailing spaces . Those should never be there. We don't touch regular links.

Also the syntax [[Category:Sandbox| ]] is a common idiom used to use the space character as a sort key in the category, which makes it show up first in the list. We filter this out.

[edit] Born (puBORN)

Some other encyclopedias use the abbreviation "b." for "born", which means that we sometimes see (b. 1979) in biographical articles. The correct style is (born 1979). But we shouldn't touch, say, "A.b.c.'s" or "slab."

[edit] Decades

Sometimes people write 1980's instead of 1980s. They shouldn't do that. But we shouldn't touch something like 20's (or should we?) or 1980'sum'2001.

[edit] Parentheses

A common error is to forget space (of whatever sort)after a closing parenthesis. Sometimes people put stray spaces (who knows why ) before closing parentheses too. But there needn't be space unless the next thing is a word (so we should detect this).

[edit] XHTML

Empty tags in XHTML should have a slash before the closing >. Probably the most common tag in Wikipedia articles is <br/>. So we should turn linebreaks without
the
slash
into proper XHTML.
