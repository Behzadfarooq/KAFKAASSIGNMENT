This article is about the desktop reactivity software. For the UK electoral ward, see Shiregreen and Brightside. For the parliamentary constituency, see Sheffield Brightside. For the 1989 album by musical band Killing Time, see Brightside. For the display technology company, see BrightSide Technologies Inc..

Brightside adds actions which are activated when the cursor is moved to the corners or edges of a GNOME desktop. It was created to add functionality which the designer perceived as lacking from Metacity and XScreensaver. As of June 2005, Brightside is not an official component of the GNOME desktop environment.

At present, the only action available at screen edges is edge flipping (moving to another virtual screen by hitting the edge of the desktop). This was omitted from Metacity since it interferes with (for example) the panels situated at the screen edges, breaking Fitts' law.

At corners, the available actions are:

  • Fade out volume
  • Prevent screensaver starting
  • Start screensaver and lock screen
  • Enter DPMS standby mode
  • Enter DPMS suspend mode
  • Enter DPMS off mode
  • Dim screen backlight
  • Custom action
