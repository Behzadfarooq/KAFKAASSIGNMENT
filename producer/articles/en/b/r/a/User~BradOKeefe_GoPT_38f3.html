Go PowerTools
Developer: BPS Software Inc
Latest release: 2006.0.7 / June 12, 2006
OS: Microsoft Windows XP
Use: IRC client
License: Shareware
Website: www.gopowertools.com

Go PowerTools is an Internet Relay Chat client for Windows, created in June 2006 and developed by BPS Software.

The latest version is 2006.0.6 and was released on June 12th 2006.
