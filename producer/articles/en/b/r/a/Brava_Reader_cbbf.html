Brava Reader

From Wikipedia, the free encyclopedia

Brava! Reader[1] is a software program from Informative Graphics [2] that opens, views, prints and measures PDF Portable Document Format, TIFF and CSF files as a Microsoft Windows desktop application or within a web browser.

It occasionally has expiration dates, where the user must then download the update to continue using the software, commonly known as a "time bomb". The update is not made available until the expiration date occurs.
