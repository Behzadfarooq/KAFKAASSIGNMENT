[edit] Branch (computer)

There should be a page on cvs/svn branches. --Jcmaco 10:02, 13 June 2006 (UTC)

It does seem this information is missing or not easily found, so please write it if you can. I would suggest though that a new section be added to the Revision control article first as appropriate, unless you have enough information on branches to warrant a separate article. If you do create an article though, you may want to pick a more specialized parenthetical qualifier (other than computer), as the term branch also has other meanings in computing. Then feel free to add a link to the section (or article) on the disambiguation page, Branch (disambiguation). -- Dmeranda 05:36, 20 June 2006 (UTC)

There already exists a separate disambiguation page though, Branch (disambiguation), which would be a more appropriate place to list these; following Wikipedia:Manual of Style (disambiguation pages). -- Dmeranda 05:28, 20 June 2006 (UTC)

I went ahead and did the merge. All content that was on this page (except for in the sense of a tree branch) is now on the Branch (disambiguation) page. -- Dmeranda 17:11, 21 June 2006 (UTC)
