Release date(s) January 04, 1998
Genre(s) Sports
Mode(s) Single player, multiplayer
Platform(s) Nintendo 64
Input Standard controller

Shin Nihon Pro Wrestling Tōkon Honō: Brave Spirits (Japanese title: 新日本プロレスリング 闘魂炎導~BRAVE SPIRITS~) is a wrestling game for the Nintendo 64. It was released only in Japan in 1998. It was created by the same developers that created the WWF Smackdown! series on the PlayStation.
