[edit] The 'Allo 'Allo! Project

In my spare time, I am currently updating the 'Allo 'Allo! entries in Wikipedia. When I first started, the main 'Allo 'Allo! page was looking rather cluttered. Since then, I have tidied up and expanded the main page, and started created an episode guide for the series. My aim is to have a plot summary and details for each episode, and an expanded page for each character from the show.

Whilst this is a bit of a hobby-horse for myself, please feel free to add your own content to these pages, or to correct any errors or omissions!
