[edit] Collapsible sections?

Hi Shane, we had a talk with Nehrams about the unresolved problem of long/short film plots. Some people want to be able to find an extended plot for a film, but others argue that one shouldn't have to go through too much. So one idea was that we have a Plot summary in the article and we create separate pages for the Extended plot. But we were told such standalone plots aren't allowed. So I had another idea, but I don't know if it's possible. Do you see any way of hiding a section and offering some button which if pressed makes it display?
