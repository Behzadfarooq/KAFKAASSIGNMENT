[edit] Image copyright problem with Image:Audio.jpg

Thank you for uploading Image:Audio.jpg. However, it currently is missing information on its copyright status. Wikipedia takes copyright very seriously. It may be deleted soon, unless we can determine the license and the source of the image. If you know this information, then you can add a copyright tag to the image description page.

If you have any questions, please feel free to ask them at the media copyright questions page. Thanks again for your cooperation. Angus McLellan (Talk) 14:43, 20 February 2007 (UTC)

P.S. If you took the photo yourself, you have a choice of licenses that you can use. These include:
  • {{GFDL-self}} - if you want to release the picture under the GFDL.
  • {{Cc-by-2.5}} or {{Cc-by-sa-2.5}} - release under Creative Commons licenses.
  • {{PD-self}} - if you want to release your picture into the public domain.
If you want to use GFDL or CC, those can be combined, but PD images can't have any other license.
Hope this helps. Please let me know if you have questions. Angus McLellan (Talk) 14:43, 20 February 2007 (UTC)
