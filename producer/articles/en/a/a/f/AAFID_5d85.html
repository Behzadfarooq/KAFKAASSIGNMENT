AAFID

AAFID stands for Autonomous Agents for Intrusion Detection, a distributed intrusion detection system. In this architecture, nodes of the ids are arranged in a hierarchical structure in a tree. The first generation of Distributed Intrusion Detection Systems had hierarchical architecture.

The general types of distributed intrusion detection systems are as follows:

  • hierarchical
  • network architecture
  • hybrid architecture
  • mobile agent architecture

Agents in AAFID were not mobile. AAFID was one of the earliest hierarchical distributed intrusion detection systems.
