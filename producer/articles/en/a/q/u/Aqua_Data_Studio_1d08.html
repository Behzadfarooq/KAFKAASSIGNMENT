Aqua Data Studio

From Wikipedia, the free encyclopedia

Aqua Data Studio is a database query and administration tool written in Java which was created by AquaFold. It allows developers to create, edit, and execute SQL scripts, as well as browse and modify database structures visually. It also provides an integrated environment that can connect to major relational databases such as SQL Server, MySQL, PostgreSQL, Oracle, DB2, Sybase and Informix, plus the generic ODBC and JDBC connections with downloads available for Windows, Linux and OSX. This allows database administrators and developers to tackle multiple tasks simultaneously from a single GUI frontend application.

[edit] Query Analyzer/Window

Aqua Data Studio's sophisticated Query Analyzer allows users to work on database scripts with specific RDBMS syntax highlighting and auto-completion to develop and test SQL scripts. Other features to speed up development include SQL automation, auto-describe and sortable multi-grid results. The query analyzer also provides client side variable binding, allowing a user to execute stored procedures or functions with local variables bound to receive out parameters. The query window also provides two modes of operation, using a SplitPane View or a MultiTab View for the editor and the query results.

[edit] Schema Browser & Visual Editing

Its graphical browsing capabilities allow you to understand the structure and dependencies of the database schema with a single mouse click. The schema browser also allows you to visually edit any schema object with a graphical design form to CREATE, ALTER or DROP the object. Visual editing supports Tables, Indexes, Procedures, Data Types and other schema objects. The visual editors also provide an SQL preview of all the commands to be executed to commit the operation.

[edit] Schema Extraction & DDL Scripting

Its graphical browsing allows you to extract the structure definition of all schema objects and to script the SQL DDL and DML (eg CREATE, ALTER, DROP, SELECT, INSERT, UPDATE and DELETE) syntax for database schema objects (including Tables, Views, Triggers, Stored Procedures and Functions).

[edit] Table Data Editor

A powerful Table Data Editor allows you to modify your results graphically and save them. By writing a single table SELECT statement in the Query Analyzer and clicking on the 'Execute Edit' button, a separate window is opened with the query results for you to edit and save. You may also browse to a table in the schema browser and select the Edit Table from the popup menu to edit the first 1000 records in a table.

[edit] Visual Explain Plan

Aqua Data Studio's Query Analyzer allows users to generate Visual Explain Plans with or without executing the SQL statements. The explain plan may be displayed in both a tree-table view and in diagram view. Visual Explain currently supports Oracle, DB2 and Microsoft SQL Server, providing a common and easy to use interface to all three databases. Visual Explain may be enabled to execute everytime statements are executed, or they may be executed individually with the "Execute Explain" command in the Query Window.

[edit] Visual Editing for Storage

Its Visual Editing capabilities allow you to CREATE, ALTER, RESIZE and DROP storage devices for a given database. The schema browser also allows you to navigate the storage structure, visually alter any of the components of the storage objects and then commit the changes. The visual editing also provides you with an SQL Preview of the SQL which will be used to commit the changes. Storage objects supported include Databases, Tablespaces, Datafiles, Rollback Segments, Redo Logs, Containers and Backup/Dump devices.

[edit] Visual Editing for Security

Its Visual Editing capabilities allow you to CREATE, GRANT, REVOKE and DROP security objects and permissions for a given database. The schema browser also allows you to navigate the storage structure, and to visually alter any of the components of the security objects and then commit the changes. The visual editing also provides you with an SQL Preview of the SQL which will be used to commit the changes. Security objects supported include Users, Roles, Profiles, Logins, Server Roles and Object/Column Permissions.

[edit] DBA Object Extraction & Scripting

Its graphical browsing capabilities allow you to browse through all database administration objects and extract the definitions to script the SQL DDL syntax for creating, altering and dropping the storage and security objects (including Databases, Tablespaces, Datafiles, Containers, Users, Groups/Roles and Profiles).

[edit] Import & Export Tool

Its Import Tool allows you to import any delimited text file into an existing table or into a new table. The import wizard provides you with an easy to use interface that guides you through the import process and provides you with a preview of the data that is to be imported before you commit the data. The Export Tool allows you to export the contents of any table or the results of any SELECT statement. Exported data may be saved in a delimited text file or a file with INSERT statements. Aqua Data Studio also has a context sensitive "Tools" menu in the schema browser popup menu, so a user may browse through the schema to the Table to be exported and then run the Tool. The Tool will use the currently selected Server Connection and the currently selected database/schema/table.
