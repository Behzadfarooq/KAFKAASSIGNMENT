AdvanceMAME

From Wikipedia, the free encyclopedia

AdvanceMAME is a derivative of MAME, which is an arcade game emulator. It differs from MAME in that it runs on GNU/Linux and Mac OS X, as well as DOS and Microsoft Windows. AdvanceMAME is also designed to be able to work with arcade monitors, TVs, and computer monitors. The software is licensed under the GPL, with an exception that allows it to link with the MAME components; MAME is licensed under the MAME License, which is incompatible with the GPL license.
