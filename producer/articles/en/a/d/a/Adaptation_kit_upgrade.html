[edit] AKU Versions for Windows Mobile 5

[edit] AKU 1.0

  • landscape screen mode support [1]
  • start devices without a SIM card[1]
  • checking for incoming mail at defined intervals[1]
  • full-fledged SIM contacts support[1]

[edit] AKU 1.1

  • Improves support for many Bluetooth profiles, such as stereo headset (A2DP) [1][2][3]
  • Wi-Fi connectivity now displays more status information. Wi-Fi performance improved.[1]
  • official hard drives support [1]

[edit] AKU 2.0

  • Messaging and Security Feature Pack (MSFP), including DirectPush email functionality[1]
  • Internet Explorer Mobile adds accelerated scrolling, Fast Back feature, support for compiled WML, WMLScript over HTTP, and enhanced DOM standard.[1]

[edit] AKU 2.1

  • dynamic switching between screen modes (landscape and portrait).[1]

[edit] AKU 2.2

  • ActiveSync improvements.[1]
  • boot speed improvements.[1]

[edit] AKU 2.3

  • bug fixes.[1]

[edit] AKU 2.5

  • bug fixes.
  • SmartDial from the Today screen[1]

[edit] AKU 2.6

  • bug fixes.
  • zooming (up to 2x) for video clips played back in Windows Media Player.[1]

[edit] AKU 3.0

  • language provisioning (enabling local manufacturers to add as many languages to a device as its onboard memory can store)
  • WiFi connection wizard (new Internet Sharing utility replaces Modem Link).
  • WPA2 support for WiFi
  • Quick start wizard
  • support for 2nd display (for instance on phones with two displays)
  • better SMS handling
  • Bluetooth profile "Personal Area Network" (PAN) is added, and "Dial-Up Networking" (DUN) profile is removed.[1]
  • vCard contacts can be transferred via SMS.
  • SIM card contacts are sorted by Name in general contacts list.
  • IE now supports AJAX [4]
