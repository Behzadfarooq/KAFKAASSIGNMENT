Adaptive quadrature

From Wikipedia, the free encyclopedia

You have new messages (last change).

Adaptive quadrature is a process in which the integral of a function f(x) is approximated using static quadrature rules on adaptively refined subintervals of the integration domain.

[edit] General scheme

  • Adaptive quadrature requires a single or set of basic rules with error estimators
  • If tolerance not met, either increase order of rule or subdivide interval

[edit] Basic quadrature rules

  • Newton-Cotes formulas
  • Gaussian quadrature
  • Gauss-Kronrod rules
  • Clenshaw-Curtis quadrature

[edit] Error estimation

  • Richardson extrapolation (see also Romberg's method)
  • Null rules
  • Epsilon algorithm

[edit] See also

  • Adaptive Simpson's method for an example of adaptive quadrature
