[edit] Usage notes

[edit] Data sources

  • Local authorities are listed here: England, Scotland, Wales (PDF Map)
  • National Rail Usage stats: Spreadsheet 2004/5 (also gives managing TOC and UK region)
  • London Underground Usage stats: [1]
  • Glasgow Subway Usage stats: SPT Statistics and Trends 2005
  • National Rail station code: [2]

[edit] Syntax

{{UK stations|                         
name        = Crewe |                   
manager     = [[Virgin Trains]] |       
locale      = [[Crewe]] |               
borough     = [[Crewe and Nantwich]] |  
platforms   = 12|                       
}}

OPTIONAL FIELDS

image       = [[Image:Crewe station platform12.jpg|300px]] |  
caption     = Crewe                     
owner       = [[Network Rail]] |         
code        = CRE |                      
exits0405   = 0.764 |            
years       = 1861 |                      
events      = Opened |

ADDITIONAL EVENTS CAN BE ADDED

years1    = [[5 May]] [[1896]]
events1   = [[Lanarkshire and Dunbartonshire Railway|L&AR]] to Clydebank opened
years2    = [[10 August]] [[1896]]
events2   = [[Glasgow Central Railway|GCR]] services commenced through to Glassgow Central
years3    = [[3 August]] [[1959]]
events3   = Station closed to passengers
years4    = [[10 August]] [[1964]]
events4   = Line closed to all traffic
years5    = [[5 November]] [[1979]]
events5   = Re-opened as '''Finnieston'''
years6    = [[1986]]
events6   = Renamed as '''Exhibition Centre'''
years7    =                
events7   =                
years8    =                
events8   =                
years9    =                
events9   =

If you enter a station code, the template will automatically link to National Rail Enquiries Live Departure Board and include links to lists of UK railway stations.

OPTIONAL FIELDS FOR STATIONS WITHIN A PASSENGER TRANSPORT EXECUTIVE AREA

pte         = [[West Midlands Passenger Transport Executive|West Midlands]] |   
zone        = 1 |

OPTIONAL FIELD FOR STATIONS WITH WELSH/GAELIC/ETC NAMES

other_name  = Caerdydd Canolog |

OPTIONAL FIELDS FOR STATIONS WITH SUBWAY STATION

|subway    = gs
|subplat   = 2 
|subexits  = 1.01

Where subway can be set to 'gs' for Glasgow Subway stations or 'lu' for London Underground stations.
