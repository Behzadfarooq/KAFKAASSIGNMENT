This is a draft replacement for the Infobox SPT station incorporating data from Category:SPT railway stations and Template:Glasgow stations. It was inspired by the template, West Yorkshire railway stations. My suggestion is that instead of having a specific infobox for an SPT station, we use the standard one with this template at the bottom of each SPT station (except Glasgow stations which can use the more specialised template) to include any SPT specific info. It might be appropriate to create a specific station infobox for the Glasgow Subway stations.
This template would also add the station to the SPT stations category if it isn't already.
