[edit] evolution article

I made a small change. I know of no one that uses the term "universal genetic code" because we have no proof that the genetic code, as we know it on Earth, is universal. There are many alternative plausible possibilities to the current genetic code, and Watson and Crick tried a host of others before finally stumbling on the correct 4-base, 3-code word, double-helix that pertains to life on Earth. Still, those other numerous genetic code possibilities have not been forsaken, because they may pertain to life elsewhere. Watson and Crick's trials-and-errors that led up to the correct one is a fascinating subject.

You wrote: "Various processes can bring genes from species back together long after those species diverged." I don't know what you are trying to say here? If I am uncertain, then? Are you eluding to HGT? If so, it's not stated in the correct context. Long after two species diverge, then they become unique species that can no longer interbreed: bring back their genes together. This is one definition of what constitutes a species. Whatever it is, you're disembodying the genes from the organism. Better use the phrase: "different genetic composition," or "species with a different genome sequence." But again, I don't know what you want to explain here.Valich 07:20, 20 March 2007 (UTC)

You are using an extremely out-dated term. Children in Junior High School, and even Elementary School today learn about genetics, and I have even have heard of them teaching PCR (Polymerase Chain Reaction) techniques in High School. Give the up-and-coming younger generation some credit. The word "Universal Genetic Code" is an old-fashioned name for the standard genetic code. It is no longer ever used. It was once thought that there existed just one, universal genetic code used by all living organisms. However, beginning in 1979, numerous non standard genetic codes have been discovered. Thus there is no universal genetic code and the phrase needs be avoided. I will chasnge this to genetic code and right an article about the universal code that states what I have written here. Don't use this term! It is clear that you must not be familiar with the pronciple behind biological, chemical evolution, else you would not be using this term. Whoever write the evolution article should know how it works. Please.
"Back in the early 1970s, evolutionary biologists did think that a given piece of DNA specified the same protein subunit in every living thing, and that the genetic code was thus universal," explains biologist Jonathan Wells, another Discovery Institute Senior Fellow. "This was unlikely to have happened by chance, so it was interpreted as evidence that every organism had inherited its genetic code from a single common ancestor. In 1979, however, exceptions to the code were found in mitochondria, the tiny energy factories inside cells. Biologists subsequently found exceptions in bacteria and in the nuclei of algae and single-celled animals. It is now clear that the genetic code is not the same in all living things, and that it does not provide 'powerful evidence' that all living things 'evolved on a single tree of life,'" concludes Dr. Wells, who holds a Ph.D. in molecular and cell biology from the University of California at Berkeley. He is author of the new book Icons of Evolution. A current listing of exceptions to the genetic code can be found at http://www.ncbi.nlm.nih.gov/htbin-post/Taxonomy/wprintgc?mode=c. (above quotation source: "PBS Charged with "False Claim" on "Universal Genetic Code" [[4]]
