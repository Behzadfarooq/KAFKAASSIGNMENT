Administrator intervention against vandalism
This page is intended to get administrator attention for obvious and persistent vandals.

For other types of vandalism or problematic usernames try the links at the top of this page. The response to a report will be usually be included in the edit summary when it's removed, not on your talk page. For a guide on how and when to report see Wikipedia:Guide to administrator intervention against vandalism.

Editors: Before listing a vandal here make sure that:

  1. The vandal is active now, has received a proper set of warnings, and has vandalized after a recent last warning, except in unusual circumstances.
  2. Your report is concise as to what the vandal is doing (including an edit summary with the IP address/username).
  3. If you are reporting an IP address with a long history of vandalism, please help us check if it is shared and tag its talk page as such if this hasn't already been done.
  4. New usernames that are obvious vandals or blatantly violate the username policy may be listed, other username reports should be placed at WP:RFCN.
  5. Suspected sock puppets should be reported at Wikipedia:suspected sock puppets.

Administrators

  1. The HBC AIV helperbots will remove blocked users from the list.
  2. When performing {{usernameblock}}'s, please remember to disable the autoblocker.
  3. If users make an invalid report, consider letting them know using template {{uw-aiv}}.
  4. IMPORTANT: Before blocking any IP address, run a WHOIS and Reverse DNS lookup on the address. Be aware of any implications your block reason has for other people on Shared IPs and enter "{{anonblock|comments}}" or "{{schoolblock|comments}}" directly into the Reason: field when required.
