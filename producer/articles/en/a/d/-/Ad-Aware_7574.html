Ad-Aware

Ad-Aware SE Personal 1.06 Screenshot
Developer: Lavasoft AB Sweden
Latest release: SE 1.06 / May 27, 2005
Preview release: 2007 Beta 4 / March 23, 2007
OS: Microsoft Windows
Use: Spyware removal software
License: Proprietary
Website: Lavasoft.com

Ad-Aware is an anti-spyware program from Lavasoft that detects and removes software on a user's computer that is determined to be spyware. It also detects dialers, trojans, malware, data-mining, aggressive advertising, parasites, browser hijackers, and tracking components.

There is a freeware version called Ad-Aware Personal, as well as two retail products—Ad-Aware Plus and Ad-Aware Professional. The free download includes Ad-Aware SE Personal Edition, skins, help manuals, RegHance and the latest definition file. However, real-time protection is not included in the personal edition. The freeware version of Ad-Aware is compatible with Microsoft Windows 98/Me/NT/2000/XP/2003. Since January 6, 2006, Ad-Aware SE Personal is a part of Google Pack. [1]

Since May 27, 2005 when version 1.06 appeared on Download.com, the program has been downloaded over 230 million times, as of February 5, 2007. [2]

Norman ASA, together with Lavasoft, offers the Ad-Aware Plus and Ad-Aware Professional versions. The main difference between the freeware and the Plus and Professional versions is the addition of real-time scanning. Real-time scanning means that users are protected when surfing and do not have to start the scanning of spyware manually, as in the freeware version.

It should be noted that the Ad-Aware is not extremely aggressive in detecting and removing certain adware. The recent test results show how a bundled adware program, "WhenUSave" is ignored by Ad-Aware Personal SE, but removed by other scanners. This is perhaps because WhenUSave is part of the popular free eDonkey P2P client. Removal would disable the free, advertising-supported eDonkey parent program. This was not always the case, as test results from 2004 show detection and removal of the WhenUSave application by Ad-Aware Personal.

[edit] System requirements

Processor: Pentium 166 MHz (or higher)

RAM: Operating system + 24 MB

Hard Disk: 2.7 MB free space (minimal configuration)

Operating Systems: Windows 98, Windows 98 SE, Windows Me, Windows NT 4 Workstation, Windows NT 4 Server, Windows 2000 Professional, Windows 2000 Server, Windows Server 2003, Windows XP (Home/Professional), Windows XP 64-bit Edition, Windows Terminal Services [3]

Other: Internet Explorer version 5.5 or higher
