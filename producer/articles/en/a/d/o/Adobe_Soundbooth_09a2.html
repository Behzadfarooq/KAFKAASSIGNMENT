Adobe Soundbooth
Developer: Adobe Systems
Initial release: 2007-03-27
Latest release: CS3 / March 27, 2007
OS: Mac OS X and Windows XP
Platform: x86 only
Use: audio editor
Website: http://labs.adobe.com/

Adobe Soundbooth is a digital audio editor by Adobe Systems for Mac OS X and Windows XP. Adobe has described it as being "in the spirit of Sound Edit 16 and Cool Edit 2000". Due to Intel-specific code, Adobe has stated that the Macintosh version will only be available for machines using Intel processors. The software is currently available as a public beta which is functional but has many rough edges, including placeholder text and GUI mishaps.
