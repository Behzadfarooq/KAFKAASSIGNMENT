In mathematics, a logarithm is the inverse operation of an exponent. This article contains derivations of logarithmic formulas. For a more general overview, see logarithm. See logarithmic identities for formulas without proof.

The logarithm of a number x in base b is the number n such that x = bn. This is usually written as where and . Usually b > 0 and x > 0 so that each real number has only one, unique logarithm in any given base. The two most common bases for logarithms are 10 and e. Both of these have their own standard function names: log(x) = log10(x) and ln(x) = loge(x). The exponential definition must be recalled often in order to work with logarithms.
