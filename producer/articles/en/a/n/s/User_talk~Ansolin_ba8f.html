[edit] Answers to questions

Hiya Ansolin-- happy to help out. As to why Wikipedia:Sandbox keep resetting?-- It'll do that, so don't use the Wikipedia sandbox for anything you want to save. If you want to work on something, you should create a new page in your own user space. So, to create a page in your user space called "Sandbox", just go to User:Ansolin/Status of religious freedom in Canada. It'll tell you the page doesn't exist and ask if you want to start it. Click on that, and you'll get a blank page you can edit. And it doesn't have to be called Sandbox. You can go to User:Ansolin/anythingatall and it'll do the same thing.

Don't hesitate to ask if you have more queries. :) --Alecmconroy 16:57, 12 July 2006 (UTC)
