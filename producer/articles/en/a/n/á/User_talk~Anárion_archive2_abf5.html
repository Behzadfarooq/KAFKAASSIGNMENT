[edit] Gujarati Script

Saw your entry at Wikipedia:Requested pictures#Culture. Probably the font you have does have all the important glyphs (I can't find a reason why someone should distribute a font with only some glyphs) but whatever software you were using didn't have support for "complex text layout" (i.e. combining characters to form ligatures properly). I don't know any Gujarati but I can help if you could give some Gujarati text to be displayed. You could convey to me the text that you want by writing it in a different script, e.g. Devanagari, or you could use a suitable transliteration scheme and give the text in Roman script, or point me to a website that has that text (if that text is public domain). [edit] Signature

Hello Anárion. Would you mind telling how to make a signature template like {{User:Anárion/sig}}? Thank you. — Instantnood 10:55 Feb 24 2005 (UTC)

Sure.
  1. Create a subpage like User:Instantnood/sig. (You can click the link here to create it).
  2. Type your signature: to replicate your current signature, just add — [[User:Instantnood|Insta]][[User_talk:Instantnood|ntnood]] there (Make sure you have a link to your user and/or talkpage.)
  3. Go to your preferences, and the signature field. Delete whatever is there now, and add (only) {{User:Instantnood/sig}}
  4. Check the 'raw signature' checkbox to make sure the automatic links are not created (this would mess up your signature).
  5. Test your signature at Wikipedia:Sandbox or a similar place. ~~~ should now just place {{User:Instantnood/sig}} there, which will automatically update to whatever is in User:Instantnood/sig. Jordi·✆ 15:13, 24 Feb 2005 (UTC)

Thank you very much. Could I set my format of time stamp by this way too? — Instantnood 19:59 Feb 24 2005 (UTC)
