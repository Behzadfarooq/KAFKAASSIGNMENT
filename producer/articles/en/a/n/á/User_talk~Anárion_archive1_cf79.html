[edit] Crist is unreadable in Safari

Your poem shows up as a solid black box in Safari on Mac OS. I think it's from one or more of these CSS properties:

background: ActiveCaption;
color:CaptionText;
border:WindowFrame;

I guess these are some proprietary Microsoft property, but they're not valid CSS. If you really like whatever they do, you might be able to hide them from web browsers with MS conditional comments.

—Michael Z. 19:52, 2004 Aug 28 (UTC)

Actually, these colours are standard CSS 2, and are perfectly valid. See: System colors in the CSS 2 spec.
From the spec: "In addition to being able to assign pre-defined color values to text, backgrounds, etc., CSS2 allows authors to specify colors in a manner that integrates them into the user's graphic environment. Style rules that take into account user preferences thus offer the following advantages:
  • They produce pages that fit the user's defined look and feel.
  • They produce pages that may be more accessible as the current user settings may be related to a disability.
The set of values defined for system colors is intended to be exhaustive. For systems that do not have a corresponding value, the specified value should be mapped to the nearest system attribute, or to a default color."
I suspect either Safari does not support CSS2 yet (as I do not have a Mac I cannot test it myself), or incorrectly assigns the same colour to both ActiveCaption and CaptionText — which ought to provide sufficiently contrasting colours. If there is any bug report form for Safari, can you make a bug report? [[User:Anárion|Image:Anarion.png]] 20:59, 28 Aug 2004 (UTC)
Indeed, this is a Safari bug. See also http://webcoder.info/reference/dereliction.html ("System Colors: Safari on Macintosh is broken (black text on a black background, white text on a white background)") -- Chucker 21:15, 2004 Aug 28 (UTC)
Sorry for that. I've gotten used to Safari having pretty darned good standards support, and my knee-jerk reaction was to blame MS. Cheers. —Michael Z. 18:06, 2004 Aug 31 (UTC)
