[edit] Usage

Please note: This template should be transcluded ({{AncientEgyptBanner}}) and not substituted (subst) because it employs conditional code. Transclusion also allows easy updating of all the Project's talk pages without having to edit thousands of pages.

{{AncientEgyptBanner
|living =
|class  = 
|importance =
|attention =
|peer-review =
|old-peer-review =
|needs-infobox=
}}
  • living: Answer yes if the subject is living; remove line if subject is dead.
  • class: Options are FA, A, GA, B, Start, Stub, Dab, Template, Cat. If blank, this will default as Unassessed. Descriptions of the options can be found at Wikipedia:WikiProject Ancient Egypt/Assessment#Quality scale.
  • importance: Top, High, Mid, Low. If blank, this will default as Unknown importance. Descriptions of the options can be found at Wikipedia:WikiProject Ancient Egypt/Assessment#Importance scale. Note: only Top will have a visible effect on the template - the other options will just assign the correct category.
  • attention: Answer yes if the page needs immediate attention, otherwise remove this line.
  • peer-review: Answer yes if the page is currently nominated for Peer review by Wikipedia:WikiProject Ancient Egypt, otherwise remove this line.
  • old-peer-review: Answer yes if the article has received a peer review from WikiProject Biography per the option above, otherwise remove this line.
  • needs-infobox: Answer yes if the article lacks an infobox, otherwise remove this line. Nothing changes to the appearance to the template if yes is used, however it will be added to the category of Biography articles needing one.

[edit] Example 1: basic

{{AncientEgyptBanner}}

[edit] Example 2: full

{{AncientEgyptBanner
|living = yes
|class  = A
|importance = Mid
|attention = yes
|peer-review = yes
|old-peer-review = yes
|needs-infobox=yes
}}

[edit] Example 3: Non-article

{{AncientEgyptBanner
|class  = NA
}}
