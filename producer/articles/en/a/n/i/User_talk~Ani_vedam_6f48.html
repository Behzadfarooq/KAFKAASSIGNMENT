About
The WikiProject SpongeBob SquarePants newsletter is a monthly newsletter posted on the first of the month. AMK152 writes and publish it. If you are interested, please see the WikiProject's talkpage.
Interviews
Sign up if you would like to be interviewed for the newsletter. Sign ups and details are here.
Guidelines
The Project is creating a set of Guidelines for the organization of SpongeBob SquarePants articles. Please see this discussion for more information.
Concerns and questions
Please leave a message with AMK152 if you have questions concerning the project. You may also post at the WikiProject's talkpage and someone will answer.
Featured articles
We are trying to make several articles good or featured articles. We would like your help reaching featured article status. We are currently working on SpongeBob SquarePants, Krusty Krab, and Bikini Bottom. Please help us improve these articles, based on the peer reviews and your knowledge of Wikipedia featured articles.
Project objectives
  • Episode Pages:
  • Infoboxes need to be complete on each episode article. (See Template:SpongeBob episode)
  • A screenshot of each title card for each episode needs to be complete on each episode article to improve it.
  • Please try to help incorperate all trivia into the rest of the articles and non-notable quotes to Wikiquote
  • Games: The SpongeBob game articles need to be expanded, especially the stubs. See here.
  • SpongeBob Books - On the List of SpongeBob SquarePants Media, books needed to be listed and a description of the book.
For more information, please click here.
