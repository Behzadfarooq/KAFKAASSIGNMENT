[edit] Notes & References

As was said, please use the same footnote format as exists in the D치l Riata article when adding references. The format is: Sentence which need a reference.<ref>Author, "Title".</ref>. If adding references which refer to works not already on the list of references, please add them. Non-references such as "Colin Renfrew, Archaeology and Language allude[s] to this" when, in fact, Renfrew has nothing to say about D치l Riata per se, only about the meta-question of language spread, are not at all helpful and look like name-dropping. Given the size of JP Mallory's oeuvre, specific works, and ideally page numbers, need to be cited. Lastly, if you want to write an essay on O'Rahilly's model, there's an article on it at O'Rahilly's historical model, and also one on Prehistoric Ireland, either of which would be a better place to do so.
