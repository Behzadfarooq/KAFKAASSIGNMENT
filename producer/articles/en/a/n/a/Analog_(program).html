Analog (program)

From Wikipedia, the free encyclopedia

 Free software Portal

Analog is a free web log analysis software program that runs under Windows, Mac OS, Linux, and most Unix-like operating systems. It was first released on April 29, 1995, by Stephen Turner as generic freeware; the license was changed to the GNU General Public License in November 2004. The program is comparable to Webalizer, though it does not use as many images, preferring to stick with simple bar charts and lists to communicate similar information.

Turner is now the chief technical officer of ClickTracks, which offers more advanced web analytics software.
