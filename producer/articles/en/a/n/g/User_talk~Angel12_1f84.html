I don't think anyone else has made it clear what the problem is, so I'll try. There's no issue with whether the links are valuable or not. It is simply that you are the operator of the site to which you are linking. The consensus opinion is that if a website has content valuable to Wikipedia, that someone other than the site owner will eventually think so and add a link to it on Wikipedia. If the site owner adds the link instead, though, we can only assume that it is because they have a vested interest in having their site linked by Wikipedia. Thus, only links that are added by neutral third parties are allowed to survive.  — Saxifrage | ☎ 17:15, Apr 21, 2005 (UTC)
