[edit] Stub Templates

No, I won't be using stub templates anytime soon. I am using the Auto Wiki Browser (AWB) to correct common misspellings. It automatically marks the articles that fit certain criteria as stubs. If you wish it to use stub templates, perhaps you can convince the author to add that functionality. If a certain misspelling is bothering you though, please feel free to mention it on my talk page and I will try to correct it using AWB as soon as I find the time. Lkinkade 01:21, 20 February 2006 (UTC)
