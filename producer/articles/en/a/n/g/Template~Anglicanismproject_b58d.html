 TemplateAnglicanismproject is part of WikiProject Anglicanism, an attempt to better organize information in articles related to Anglicanism and the Anglican Communion. If you would like to participate, you can edit the article attached to this page, or visit the project page, where you can join the project and/or contribute to the discussion.

[edit] Usage

Parameters
Parameter Potential values Usage
class NA • stub • start • B • A • GA • FA Use to classify article by its quality
importance low • mid • high • top How important the article is to the Anglicanism WikiProject
attention yes For articles requiring immediate attention
orphan yes For articles without a WikiProject. Shouldn't need to use
collaboration-candidate yes If article is a canditate for the Anglicanism WikiProject collaboration of the month
past-collaboration yes If an article has been the Anglicanism WikiProject collaboration of the month
peer-review yes If a request for peer review has been made
old-peer-review yes If an article has already been peer reviewed
