I made this tab layout mostly because I was bored. It was quite a challenge to make, because apparently MediaWiki implements a whole bunch of "fixes" which made the layout freak out in Internet Explorer (when it didn't on a normal XHTML 1.0 Strict compliant page). If you have a better way of circumventing the fixes, please tell me, because right now the tabs are 1 pixel too far down on Firefox (Yes, I'm a perfectionist).

In any case, here are all the components used, in case you're interested in having a look:

  • Header template
  • Tab template
  • Footer template

Changes to the tab layout that I may or may not do:

  • Support adding tabs to the footer as well (so that more tabs can be viewed properly even on low-resolution screens).
  • Make the CSS less obfuscated while retaining a pixel perfect layout across browsers (Silly IE "fixes").
  • Feel free to suggest changes on my talk page.
