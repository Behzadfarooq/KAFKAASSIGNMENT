[edit] Units

I'm working on a infobox template to implement on prefix unit pages. Please help me out. Am I missing anything? Could anything be worded better? How about the layout? and the biggest question of all, should we recreate the prefix unit pages (which now currently redirect to the base unit)?

Decagram
Unit sign dag or dcg
Measure Mass
Base Unit Kilogram
Multiple of Base 10-2
System SI
Common usage The decagram is not commonly used.
Examples
A lethal dose of caffeine for an adult is 1 dag.
An adult mouse (Mus musculus) weighs 1.2–4 dag.
One ounce (avoirdupois) is 2.835 dag.
Conversion
SI 10 grams = 1 dag = 10-2 kg
Imperial 1 dag ≈ 0.3527 ounce ≈ 0.022 pound
