[edit] Template:AndonicO's Happy Holidays template

I removed the floating box because it was floating around aimlessly on users' talk pages and making posting cumbersome. I also recommend you go and subst all those inclusions so that the template itself can be deleted. This has clearly no encyclopedic purpose. ~ trialsanderrors 20:57, 27 December 2006 (UTC)

I'll have to second that. --Tewy 21:10, 27 December 2006 (UTC)
Re: I second that you subst those talk pages, and it wouldn't hurt to delete the template. If you don't want to delete it, then you can make a subpage on your userspace and keep it there. --Tewy 21:16, 27 December 2006 (UTC)
Moving into user space won't fix it, unless it's done by hand (change all transclusions to the new address). ~ trialsanderrors 23:26, 27 December 2006 (UTC)
What would that do? Should I already know that? | AndonicO Talk | Sign Here 23:43, 27 December 2006 (UTC)
That's what I meant. My idea was to subst each template, so that nothing transcluded the template. Once that was done, the code for the template could be stored in the user space for future use. But yes, if the pages weren't subst'ed, then the links would have to be changed. --Tewy 01:26, 28 December 2006 (UTC)
Re: And in response to your last comment, yes, subst'ing a template copies the code as it was at the time of transcluding directly into the page. (See this.) --Tewy 01:31, 28 December 2006 (UTC)
Substing a template is to include a "subst:" before the template name in the brackets: {{subst:AndonicO's Happy Holidays template}}. That way the wikicode gets posted into the page instead of a link to the template. ~ trialsanderrors 01:43, 28 December 2006 (UTC)

Template:AndonicO's Happy Holidays template has been nominated for deletion. You are invited to comment on the discussion at the template's entry on the Templates for Deletion page. Thank you. ~ trialsanderrors 06:49, 28 December 2006 (UTC)
