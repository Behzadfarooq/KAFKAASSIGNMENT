And-httpd is an open source web server authored by James Antill. It is intended to be simple, fast and comes with a US$2000 security guarantee.[1] And-httpd is distributed under the GNU Lesser General Public License (LGPL).

Features include: chroot support; single-process polling design; conditional rewrites; I/O transfer limits; server-side includes support via Scons-based tools; and virtual hosting.
