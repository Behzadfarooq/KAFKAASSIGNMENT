[edit] Medal summary

Use this style to summarize the winners in a single table:

  1. On a "Sport at the year Olympics" page, the first column would list the events (wikilinked to detail pages)
  2. On the overview "Sport at the Olympics" page, this table would be used to summarize all the medal winners for a specific event, across all Games. The first column would list the Games (wikilinked) to the "Sport at the year Olympics" page.
Games Gold Silver Bronze
19xx City User:Andrwsc (CAN) User:Andrwsc (USA) User:Andrwsc (GBR)
Event name User:Andrwsc (CAN) User:Andrwsc (USA) User:Andrwsc (GBR)
