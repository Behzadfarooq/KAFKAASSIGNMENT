[edit] WARNING:

DO NOT USE ON A SUBPAGE THAT HAS MORE THAN ONE PARENT!!! THIS WILL CAUSE BROKEN LINKS!

[edit] How to use:

Use {{User:Andrew_Hampe/Templates/Menu2}} to get:

Andrew Hampe/Templates's User Pages
   
 Home
   
 Talk
   
 Awards
   
 Sandbox
   
 Vandal Box
   
 Subpages

To set the name on the header of the template, use {{User:Andrew_Hampe/Templates/Menu2|Name}}

Example:

(Name)'s User Pages
   
 Home
   
 Talk
   
 Awards
   
 Sandbox
   
 Vandal Box
   
 Subpages

To set page names of the awards, sandbox, and vandal box pages, along with the name of sandbox (so you can call it workshop or something else, use: {{User:Andrew_Hampe/Templates/Menu2|Name|AwardsPage|SandboxPage|SandboxName|VandalBoxPage}}


Example: {{User:Andrew_Hampe/Templates/Menu2|Name|AwardsPage|SandboxPage|SandboxName|VandalBoxPage}}

Creates:

(Name)'s User Pages
   
 Home
   
 Talk
   
 Awards
   
 SandboxName
   
 Vandal Box
   
 Subpages
