[edit] Live Preview

Hi. I'm afraid I've lost the latest "unpacked" source code in an accident with my HD (I was careless and kept no backups). There is an unpacked version somewhere in the Spanish Wikipedia, but it's quite outdated. The packed version, however, is just the actual source with line breaks, indentation and comments stripped out. Variables and function names are unchanged, so taking it back to nice shape shouldn't be too hard (only a bit annoying). It still won't be as good without all the commentary.

About the licence, I thought I had addressed this on the Live Preview page, but it seems I hadn't. It has no licence at all, but you can do with it whatever you feel like doing, I don't care.

Anyway, I don't think I'll ever get back to maintaining it for several good reasons. One reason is that someone else has started a very similar project but in a larger scale (something I was planning to do myself at about the time I heard of this). In case you don't know this already, please check out this page. His parser is a lot more powerful than mine (only a bit slower, maybe) and will be released as GPL once released (stated at the bottom of that page). Perhaps that will interest you more than my Live Preview.

Hope this helps. Please message me again if you need anything else. Pilaf 19:08, 8 Apr 2005 (UTC)
