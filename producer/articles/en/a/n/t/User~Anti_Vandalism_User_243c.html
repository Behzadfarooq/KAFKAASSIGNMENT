This user account is not a bot but is a second account of Tellyaddict. This account is used only for fighting vandalism; it will NOT participate in any discussion/deletion debates or other open discussions

It is not a sock puppet, but a manual account which is not semi or fully automated.

[edit] What this account is

This account is used only for fighting vandalism on Wikipedia, as mentioned above it is not a bot and will participate only in the following areas:

  • Reverting and warnings vandals
  • Marking pages for speedy deletion - should this be necessary
  • Reporting Violations of the username policy to Administrator intervention.
    • Other areas of Wikipedia for fighting vandalism, User talk pages will also be edited for warnings of IP addresses/User accounts.

[edit] What this account is not

As mentioned above, this account is not a bot and is not automated. As well as this, the account does not have Administrator tools and never will have, it cannot block users, if you are here after you have been unblocked to ask me why, please investigate further as this account will not have blocked you. The owner of this account (Tellyaddict) is too not an administrator so he will not have blocked you either.

[edit] Contact

To contact this account then simply leave a message on the Talk Page and sign your name as you would with any other account on Wikipedia - this one is no different. If you urgently need to contact me the you would get a quicker response at Tellyaddict's talk page. It does not have email enabled so the only way to contact me is above.

  • Administrators, in the unlikely event of this account vandalising Wikipedia or being hacked, please instantly block it and leave a message on Tellyaddict's talk page to resolve the problem.
Thank you!
