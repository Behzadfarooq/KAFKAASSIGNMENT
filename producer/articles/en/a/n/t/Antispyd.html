[edit] Overview

Antispyd is a HTTP/HTTPS proxy server designed to be fast, efficient, modular and secure. It's written in pure C and uses a simple and customizable configuration file. It's POSIX compliant, has been developed under GNU/Linux system and is tested with Valgrind. Another main aspect of Antispyd is its ability to produce customized logs for better Web usage monitoring.

[edit] Technical Details

The server is designed with a modular software architecture, the following filters can be used and configured independently :

  • Url Filtering, urlblacklist.com url's and domain's lists are supported
  • Mime-type Filtering
  • Advanced HTTP flow's size Control
  • Web-Identity Masking
  • Shellcode and Zero-day threat Filtering
  • Cookie Management
  • Signature based Filtering
  • Inappropriate content Blocking
  • Pop-up Deleting
  • Fault-tolerant Proxy Cache Inter-connexion Capability

[edit] Usage

So, Antispyd can be used to satisfy some current personal and corporate security needs like :

  • Web Usage Policy establishment
  • Web Usage Monitoring
  • Identity Theft Avoiding
  • Instant Messaging Control
  • Peer-to-Peer Filtering
  • Worms Proliferation Blocking
  • Malware Protection

[edit] Platforms Supported

Antispyd supports Linux, NetBSD and FreeBSD. A port for OpenBSD is planning. There is no port for Microsoft Windows at the moment.
