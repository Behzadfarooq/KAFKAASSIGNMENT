Many users split their user page up into multiple pages and store the extra pages as subpages of their user page (in this format: User:pagename/subpagename). Either their user page has grown too big to be a single page, or they like to keep their subjects more sectionalized. [edit] Menus

Creating subpages has a navigation problem, however, for typing the entire page name into the search box every time you want to go to a subpage is cumbersome, so one solution is to link the family of pages together with a common menu. The menu is stored in a subpage of its own, and is transcluded onto each page just like a template. This affords the opportunity to add the talk page and special pages to the mix (like displaying the user's contributions and other display commands from Wikipedia's "special" menu). Some menus look just like nav bars, others have icons, some are integrated into the page's border, and others are verticle lists (usually boxed). One example of an integrated user menu is mine at the top of this page.
