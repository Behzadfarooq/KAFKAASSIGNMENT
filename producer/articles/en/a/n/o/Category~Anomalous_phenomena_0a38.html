Category:Anomalous phenomena

From Wikipedia, the free encyclopedia

The main article for this category is Anomalous phenomena.

An anomaly is an event which currently does not have an explanation. In the process of cognition some anomalies literally are not perceived if one does not have a category or class for the event. This category simply lists those items which are yet to be explained or understood in one's current ontology.
