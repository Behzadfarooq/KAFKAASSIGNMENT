Anonym.OS is a LiveCD operating system based on OpenBSD 3.8 with strong encryption and anonymization tools. The goal of the project is to provide a secure anonymous web browsing access to everyday users. The operating system is OpenBSD 3.8, although many packages have been added to facilitate its goal. It uses Fluxbox for its window manager.
