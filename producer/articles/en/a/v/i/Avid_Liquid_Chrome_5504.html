Chrome HD

Avid Liquid Chrome uses the same base software as Avid Liquid and Avid Liquid Pro. Liquid Chrome has SDI based hardware. Liquid & Liquid Pro are aimed at prosumer to professional users, while Liquid Chrome is designed for industrial use, such as in broadcast networks, or more commonly feature film editing seats.

[edit] Features

Some of the key features of Avid Liquid Chrome:

  • AutoSave
  • On the timeline DVD authoring
  • Native HD and HDV editing
  • GPU real-time effects
  • Multicam
  • Surround Sound editor
  • Advanced effect editors
  • Background rendering
  • Advanced colour correction
  • Proxy & XDCAM compatibility
  • Wide array of I/O
  • Full SDI Integration and Support
  • Avid open timeline
  • More codecs supported
  • Acelerating TARGA interface card
  • Future InTime support
  • Low drop-frame errors
  • More real-time editors
  • Extensive render farm support

[edit] I/O Options

Avid Liquid Chrome HD Plus Break-Out-Box
Avid Liquid Chrome HD Deluxe Break-Out-Box
Avid Liquid Chrome HD Elite Break-Out-Box

Avid Liquid Chrome HD combine's the Liquid software with SDI hardware. The Avid Liquid Chrome HD family offers a range of solutions that can handle almost any I/O format or codec and then author to a wide range of output formats, including standard- or high-definition tape, DVD, or Web. The family includes three scalable systems designed to meet a range of professional and comercial needs. These systems include:

[edit] Avid Liquid Chrome HD Plus

Standard-definition SDI I/O with 4-channel AES/EBU digital audioa. A package priced at $4,995 USMSRP for independent videographers who are looking for an integrated application.

[edit] Avid Liquid Chrome HD Deluxe

Standard- and High-definition SDI with 4-channel AES/EBU digital audio. It is a comprehensive solution priced at $7,995 USMSRP that offers all of the functionality of Liquid Chrome HD Plus and adds HD SDI I/O;

[edit] Avid Liquid Chrome HD Elite

A full range of analog and digital standard- and high-definition connectivity in a professional rack-mounted breakout box. It is the most advanced system in the line priced at $11,995 USMSRP, including comprehensive SD and HD I/O in a rack-mountable broadcast chassis.

[edit] System Requirements

Avid Liquid Chrome HD is only certified to run on the HP XW8200 and HP XW8400 workstations.

CERTIFIED CONFIGURATION:

RAM: 2048 MB

CPU: 2x Intel Xeon CPU 3.4 GHz, L2 Cache 1MB

VGA: Connect3D X1900XTX 512MB VGA/TV/DVI-I, driver Catalyst 6.2 or later

AUDIO: SoundMAX Integrated Digital Audio driver 5.12.1.3620 (27.05.2003)

SYSTEM DRIVE: WD740 GD 74 GB S-ATA

VIDEO DRIVES: 2 x FUJITSU MAT3147NP SCSI 320(Windows stripe set)

OS: Microsoft Windows XP Professional 5.1.2600 SP2, Build 2600, DirectX 9.0c

SLOT ASSIGNMENT:
PCI Slot 1 empty (32bit)
PCI-E Connect3D X1900XTX 512MB (x16)
PCI Slot 2 empty (32bit)
PCI-E empty (x8)
PCI Slot 3 Chrome HD codec (PCI X133)
PCI Slot 4 empty (PCI X100)
PCI Slot 5 empty (PCI X100)
