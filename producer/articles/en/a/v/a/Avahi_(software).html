Avahi is a free Zeroconf implementation, including a system for multicast DNS/DNS-SD service discovery. It allows programs to publish and discover services and hosts running on a local network with no specific configuration. For example you can plug into a network and instantly find printers to print to, files to look at and people to talk to. It is licensed under the GNU Lesser General Public License (LGPL).

Architectural overview of the Avahi software framework.

Avahi is an implementation of the Apple Zeroconf specification, and implements mDNS, DNS-SD and RFC 3927/IPv4LL. Other implementations include Apple's free/open source Bonjour framework.

Avahi provides a set of language bindings (Python, Mono etcetera) and ships with most Linux and *BSD distributions. Because of its modularized architecture, Avahi is already integrated in major desktop components like GNOME's Virtual File System or KDE's input/output architecture.

The Avahi project was originally started due to the controversial license of Apple's Bonjour Zeroconf implementation. Since then, Bonjour has been relicensed under the uncontroversial Apache License. However, Avahi had already become the de-facto standard implementation of mDNS/DNS-SD on free software desktops operating systems such as Linux. Many consider Avahi the superior implementation of a Zeroconf stack.[citation needed]

Avahi is nowadays included in almost all Linux distributions.

[edit] Trivia

Avahi is a species of primates (separate article) and Trent Lloyd, one of the developers, found it, liked it and it stuck. The logo obviously reflects this.[1]

Avahi has been developed by Lennart Poettering and Trent Lloyd. It is the result of a merger of Poetterings's original mDNS/DNS-SD implementation called "FlexMDNS", and Lloyd's original code called "Avahi" that happened in 2005. While most of today's code originates from the former project the name of the latter was used for the joint project. Development on "FlexMDNS" started in late 2004, work on the original "Avahi" began in early 2004.

Avahi was originally developed under the freedesktop.org umbrella, but has now become a separate project. Avahi, however, makes use of freedesktop.org's D-Bus IPC layer.
