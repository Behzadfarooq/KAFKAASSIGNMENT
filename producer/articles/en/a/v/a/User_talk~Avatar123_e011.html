[edit] Further advice on help

So is it the editing Wikipedia what you are having trouble with? If you want to edit a article then simply click the edit this page button at the top of the screen and then make your changes ot the text and then write an edit summary (Briefly describe the chnages you have made) and the save the page although you may wish to preview the page first to check that everything is in order and you changes should be visible immediately. on User talk Pages if you wish to leave a message to the user then click the + icon next to the edit this page and leave a comment that way on talk pages

For help signing your name To sign your name on talk Page then you can write your text and then type four tildes which can be done by holding down the 'shift' k ey and then hitting the tilde key directly above the shift key. Alternatively you can click the four tildes directly below the editing box and this will insert them for you and when the page is saved it will appear with the username and times of the comment.

For citing references and sources on articles When you type information into an article on Wikipedia, chances are it will need to be cited so that viewers know that it is a reliable source and that it's not just made up. so to cite your sources in an article you would do this. (note: After the web address of the website has been inserted, leave a space and then type what the website is describing.)

<ref>[add the address of the website where you got your information off (please don't copy and paste it though) then here write what the webpage is describing]</ref>

(note: After the web address of the website has been inserted, leave a space and then type what the website is describing.) Once all of the necessary article references have been added create a section to the article at the bottom saying

==References== Then in there add this <div class="references-small"></div> <references/>

Instead of remembering this though a link can be found to important buttons below the edit summary box and it saves you having to type them out. This is about as simply as I could put it but if you don't understand contact me and I'll try to explain it simpler or you can check outHere. It took me ages to learn this so don't give up, you'll be able to do it perfectly in 2 weeks! Please tell me when you are ready to learn more on my Talk Page


Their is still a little to learn and tell me on my talk page when your ready to learn more and I'll tell you. Tellyaddict Talk 12:38 14 January 2007 (UTC)

[edit] Contributing

Well you asked me about the different ways in which you can contribute to Wikipedia, well there are many ways, If you have an idea for an article which sticks with the riles of the Notabbility Guideline then you can Start a new Page. If this is not for you then you can check out The Backlogs where all of the articles which need attention are listed there into there own categories, from NPOV to Wikifying articles there are many ways to help out there. Other ways include hitting the [[Wikipedia:Random|Random article button on the left toolbar and possibly make any changes to that which could be done, even if it is just one Wikilink. You may also want to participate in:

  • Articles for deletion
  • Templates for deletion
  • Images and Media for deletion
  • User Categories for discussion
  • Categories for Discussion
  • Redirects for discussion

This is where wikipedians dscuss on whether an article should be deleted or kept, however n these users are reminded to remain civil. You may also want to patrol the NewPages and cleanup them up if necessary or mark them for deletion should it meet the criteria for speedy deletion - you can do this by using the Speedy deletion templates at the bottom of This Page (Note: Some attack pages or inappropriate pages created not to help wikipedia can sometimes contain bad language and inaproppriate text which some people could find offensive). Or if you see a article which has a reference link to an external site on the bottom and you discover it has been copy and pasted from there then should tag the article with {{copyvio|URL here of the website copied from here}}. and then blank the page (say in your edit summary thats its a copyvio just so other people know you have not just blanked it for the sake of it) to reduce the chance of this being seen and any further action being taken against wikipedia. You may also wish to do some vandalism fighting by patrolling the Recent changes and after you are experienced enough in vandal fighting you can appyly for programmes such as VandalProof or Vandal Fighter and moderators of this tool will choose whether to accept or decline your request and provide a reason. Anyway I think I've wrote too much, hope this helps! TellyaddictEditor review! 12:12, 21 February 2007 (UTC)
