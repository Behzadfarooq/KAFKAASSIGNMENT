, a proprietary application design specification created for mobile computing users to run applications from a specially design usb flash drive. Flex-It is created for Microsoft Windows operating systems and no installation process is required for all Flex-it compliant applications.

Flex-it compliant applications are executed from the Flex-It device and data is written to the flash storage zone without leaving any trace in host computer. Flex-It device comes with Flex-It Launcher pre-loaded in the CD-ROM zone. It is a menu navigation program that makes organizing and accessing of files, folders and applications in flash storage and CD-ROM zone simple and efficient.

[edit] Flex-It platform

[edit] Flex-It Launcher

Flex-It Launcher

Flex-It Launcher is a menu navigation program that pre-loaded in the CD-ROM zone of Flex-It device. Upon insertion of Flex-It device, Flex-it Launcher is auto played. This program is built to work with Microsoft Windows 2000 Service Pack 4 and above.

[edit] Flex-It applications

Flex-It fully compliant application runs only on Flex-It device. Software provider requires Flex-it SDK (system development Kit) to program lower-level access to Flex-It device.

[edit] Flex-it SDK

It is the API (Application Program Interface) that enable software provider to program lower-level access to parameters such as the Unique key, fixed hidden memory and etc of Flex-It device.

[edit] Hardware Technology

Flex-It device is a special design usb drive that has three partitions namely CD-ROM, normal flash storage and hidden memory zone. The hidden memory zone is invisible by operating system. CD-ROM zone is created to provide autoplay functionality similar to a normal CD-ROM drive.
