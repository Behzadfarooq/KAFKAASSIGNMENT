[edit] Hi, I added the parameter you wanted on your listal template

Hi there. I noticed you asking about adding a parameter to your listal template so that others could use it, so I was bold and updated a few pages in your userspace to do so. I hope that's O.K. with you. For future reference, when you want to add a parameter to a template, you use {{{1}}} for the first parameter, {{{2}}} for the second, and so on. Thus, your template would use the format {{template|value 1|value 2}}. If you want a parameter to have a default value, you use the syntax {{{1|default value}}}. If you want to give a parameter a specific name that allows it to disregard use in the order of the template parameters, then you can give it a name, like {{{username}}}. A template with a named parameter like that uses the syntax {{template| namedparameter=foo}}. That's just the basics - if you're really interested in templates, you can check out m:ParserFunctions. Hope my changes help, Nihiltres 02:17, 7 December 2006 (UTC)
