AegeanLinux is an i686 Linux distribution created by Tim Groeneveld. The upcoming version will use the Frugalware branch of the Pacman package manager originally created by Judd Vinet. In common with Arch Linux, AegeanLinux aims to be flexibile for advanced users; however, AegeanLinux ships with KDE configured by default rather than beginning with a command line interface. Additionally, the utility aegconf is provided to configure various system software, including X11, Pacman, and the firewall.

The first release of AegeanLinux, 0.48, appeared on 11 July 2006. The latest release is 0.5.02, released on 11 October 2006.
