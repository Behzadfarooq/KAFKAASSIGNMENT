The AMA Alert System is a communication board for members of the Association of Members' Advocates. It works by Advocates adding a template to their talk page which displays summaries when this page is updated. If you are not a member of the AMA, you don't need to worry about this.

When there is an Alert posted, the template displays. When there is no current alert, the template becomes invisible.
