[edit] Welcome to WikiProject Morocco

Welcome, Aelfthrytha, to the WikiProject Morocco! Please direct any questions about the project to its talk page. If you create new articles on Morocco-related topics, please list them at our announcement page and tag their talk page with our project template {{WikiProject Morocco}}.

A few features that you might find helpful:

  • The project's Navigation box points to most of the pages in the project that might be of use to you.
  • Most of the important discussions related to the project take place on the project's main talk page; you may find it useful to watchlist it.
  • We've developed a number of guidelines for names, titles, and other things to standardize our articles and make interlinking easier that you may find useful.

[edit] image categories

Hi, some time ago, you created a number of categories for images and maps, which seem rather redundant to the purpose of WP:Commons. Was there any discussion about those back then, or is it ok to replace them with links to the respective commons categories? --Latebird 13:26, 13 March 2007 (UTC)

  • I looked around and I can't find them - can you link me so I can comment appropriately? Aelfthrytha 12:58, 14 March 2007 (UTC)

User:Electionworld has created a few more subcategories for historical maps (but switched to supporting the Commons:Atlas since). Many of them are completely empty. Most of the images listed in the others should be moved to Commons (or have already been moved). You also created some local pages for Commons based images with the sole purpose of listing them in those categories. All of this looks like a lot of parallel work, promising to turn into a maintenance nightmare sooner or later. Interestingly, the same question has been repeatedly raised here, but was never properly addressed as far as I can tell. --Latebird 13:57, 14 March 2007 (UTC)
