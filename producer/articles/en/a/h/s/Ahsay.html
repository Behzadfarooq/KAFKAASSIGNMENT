Ahsay provides a suite of software to support automated back up of computer data online or offsite from systems running on the most popular platforms. These include:

  • Microsoft Windows
  • Linux
  • Mac OS X
  • Solaris

The software has specialized agents to back up databases such as:

  • Microsoft Exchange
  • Microsoft SQL Server
  • Oracle database
  • MySQL
  • Lotus Notes / Domino

It has various features that make it fast and efficient for online remote backup across the Internet:

  • Incremental Backup - only modified files will be uploaded to the Backup Server
  • Differential Backup
  • In-file delta technology - only the changed blocks within a file will be uploaded to the Backup Server, making backup of large files efficient; a propritary technology developed by Ahsay)
  • Volume Shadow Copy - even files that are opened can be backed up under Microsoft Windows
  • Seed Loading Utility - back up to a local hard disk for the initial full backup and physically transfer the hard disk to the backup server to save bandwidth over internet transfer

The Ahsay software suite consists of three major components:

  • OBS - This is the Offsite Backup Server where all backed up data is stored.
  • OBM - This is the Online Backup Manager that runs on client systems to initiate back up.
  • RPS - This is the replication server that is designed to replicate (in close to real-time) the contents of an OBS server to another server on a different site.

This software is used by many of the Internet based Remote backup services including some mentioned on the Managed backup providers list. Some examples are Backup to the Web, BackupsAnywhere and Storeitoffsite.
