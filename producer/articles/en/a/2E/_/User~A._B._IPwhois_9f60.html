  • Example:
    • {{subst:User:A. B./IPwhois|127.0.0.1}}

  • I adapted this from Template:IPvandal for use with IP accounts not involving vandalism -- mostly working with WikiProject Spam. Since so much inappropriate linking is done in good faith and therefore not necessarily spamming, I did not want use a template incorporating a perjorative name such as vandal or spam. I made several changes:
    • Deleted the "block user" function
    • Added a "traceroute" link
    • Gave the stuff in parentheses a small font
    • Added in "tor" from Template:Proxyip2
    • Added in Google search from Template:Proxyip2
    • Added "links" (i.e., what links to user's user page)
    • Added "count" - Interiot's tool from Template:User6
    • Logs:
      • "user logs" - log of actions take by user
      • "user page logs" - log of actions taken on user's account


  • This is for personal use; please ask before using.
    • If you forget to use "subst:", the last thing I want is some anonymous IP editor storming back to this user subpage wanting a piece of User:A. B. because the angry note you posted still said "{{User:A. B./IPwhois|127.0.0.1}}" when he opened an edit window. Yikes!
    • If you like this page, why not just plagiarize it to a subpage of your own?


  • Note to self: always use "subst"!
    • {{subst:User:A. B./IPwhois|127.0.0.1}}
