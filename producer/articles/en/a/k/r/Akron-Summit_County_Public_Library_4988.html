The Akron-Summit County Public Library system services all the suburbs of Akron, Ohio within Summit County. The main library is on South High Street in downtown Akron.

[edit] Branches

There are 18 branches in Summit County:

Main branch located in downtown Akron.
  1. The Ellet Branch in the Ellet area of East Akron
  2. Fairlawn-Bath
  3. The Firestone Park branch in South Akron
  4. The Goodyear branch in the Goodyear Heights area of East Akron
  5. Green
  6. The Kenmore branch in Southwest Akron
  7. The Main Library in Akron (pictured)
  8. The Maple Valley branch in Akron
  9. Mogadore
  10. Nordonia Hills
  11. The North Hill branch in North Akron
  12. The Northwest Akron branch in Northwest Akron
  13. Norton
  14. The Odom Boulevard branch in Southwest Akron
  15. The Portage Lakes branch in South Akron
  16. Richfield
  17. Tallmadge
  18. The West Hill branch in Northwest Akron
