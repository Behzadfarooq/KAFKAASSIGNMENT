[edit] How To Make Your Own Ribbon

The following instructions are for Paint Shop Pro.

  1. Find a suitable image of a ribbon.
  2. Import it into Paint Shop Pro.
  3. Image -> Blur -> Motion Blur -> 0 degrees at maximum length.
  4. For every colour band
    1. Select the brightest part of the band using the eyedropper.
    2. Use the paint bucket to drop a square of flat colour onto the band.
    3. Check for large lines of off colour.
  5. Image -> Blur -> Motion Blur -> 0 degrees at maximum length (again).
  6. Select a thin band arcoss the image and Ctrl-V to create a new image.
  7. Resize the image to 105 x 30.
  8. Import into Paint Shop Pro.
  9. Multiply your image with the band using 255 as a divisor.
