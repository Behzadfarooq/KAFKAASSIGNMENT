Template:Awayuser has been nominated for deletion. You are invited to comment on the discussion at the template's entry on the Templates for Deletion page. Thank you.

[edit] Optional parameter

I've changed the wording to just "vacation" - which is a default parameter that can be changed to something else (e.g. "holiday):

{{Awayuser|holiday}}
Which changes the message to "This user is currently on holiday from Wikipedia."
