[edit] Summary

Español: Apollo 17 - El último vuelo a la Luna

Edit of Apollo_17_The_Last_Moon_Shot.jpg [1] for Apollo 17 FPC. Ran through neatimage twice to remove artifacts then manually edited jpeg artifacts along edges and touched up dust and scratches. Tried to improve sky texture but I think I've left some posterization in the sky. :( Strange, because in photoshop, my copy doesn't have any posterization but even saving JPEG at highest possible quality creates posterization in the sky. If anyone can remove it, would be appreciated. ;) I have a PNG file (700kb :S) that is a lossless version of this file here if you would like to try re-saving it as a JPEG.
