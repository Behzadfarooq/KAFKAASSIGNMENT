Apple Icon Image
File extension: .icns
Type code: icns
Uniform Type Identifier: com.apple.icns
Developed by: Apple Computer
Type of format: icon file format

The Apple Icon Image format is the icon format used in Apple Computer's Mac OS X. It supports icons of 16×16, 32×32, 48×48, 128×128, 256×256 and 512×512 pixels, with both 1- and 8-bit alpha channels and multiple image states (example: open and closed folders). The fixed-size icons can be scaled by the operating system and displayed at any intermediate size.
