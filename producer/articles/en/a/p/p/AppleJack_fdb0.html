AppleJack

AppleJack's main menu
Developer: The Apotek
Latest release: 1.4.3 / 2006-11-13
OS: Mac OS X
Use: simplified interface for system repair
License: GPL
Website: http://applejack.sourceforge.net/

AppleJack is a command line user interface for Mac OS X that provides a simplified interface for single user mode system repairs.
