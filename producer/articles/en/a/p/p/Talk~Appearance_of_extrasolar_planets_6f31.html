[edit] Applying this system to exoplanet articles

I've noticed that this system is being applied to extrasolar planet articles. If you are going to do this, please make sure you have a reference where the particular Sudarsky class is assigned (note that Extrasolar Visions is a speculative site, and the speculations there should not be presented as fact). As far as I have found, the only place where the classes in this system are applied to specific planets is the paper outlining this system. Even then this should be presented as a theoretical calculation, not as a fact - these planets have not been directly observed, so saying what the dominant cloud type is is impossible. Chaos syndrome 23:23, 14 January 2006 (UTC)

[edit] Question

Does anyone know what the formula is to get an exoplanets temperature? And the formula to get an exoplanets albedo.
— Hurricane Devon ( Talk ) 00:04, 3 March 2006 (UTC)

In the case of temperature, it depends how good an approximation you want - the simple formulae often used do not take things like greenhouse effect or clouds into account - a typical predicted value for Earth is 250-260 Kelvin, which is rather chilly. As for the albedo, you need to specify which albedo you mean (there are several, e.g. geometric, Bond), and to get values for it you really need to observe the planet. It is possible to predict values by modelling the planet's atmosphere, but this requires knowledge of the atmospheric composition, which in general we do not have. Chaos syndrome 00:30, 5 March 2006 (UTC)
