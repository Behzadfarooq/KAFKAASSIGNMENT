Category:Application layer protocols

From Wikipedia, the free encyclopedia

For more information, see the article about application layer.


Note that the Application layer of the Internet reference model includes session layer protocols, the presentation layer protocols as well as application layer protocols in the OSI reference model. This category involves application layer protocols according to both models.
