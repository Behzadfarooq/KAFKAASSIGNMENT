Application Markup Language (AppML) is an XML language for describing Internet applications. AppML uses the Internet standards HTML, CSS, and XML to describe applications.

[edit] Short Description

  • AppML uses XML to describe Internet applications
  • AppML applications are self-describing
  • AppML is a declarative language
  • AppML is platform independent (NB: "AppML Services" require ASP.) True?
  • AppML uses AJAX technology
  • AppML is an open source initiative from W3Schools

[edit] How does it work?

  1. Describe the elements of your application with XML.
  2. Save the XML file on a web server.
  3. Link to an AppML web service to execute your application.
  4. If you want to modify your application, edit the XML file.

To execute an AppML description you will need an AppML web service. The purpose of the web service is to execute the application based on the AppML description. To access an AppML web service, you only have to link to it. The following link: appml.asp?appname=customers&displaytype=report links to "appml.asp" with a request to run an HTML report based on an AppML description named "customers.xml".

[edit] Example

  <appml>

    <title>CUSTOMERS</title>

    <database>
      <connection>northwind</connection>
      <keyfield type="text">customerid</keyfield>
      <maintable>customers</maintable>
      <sql>Select CompanyName, City, Country from Customers</sql>
    </database>

    <filters>
      <query>
        <field>CompanyName</field>
      </query>
    </filters>

    <htmlform />
    <htmllist />
    <htmlreport />

  </appml>

[edit] Original Design Goals

  • AppML must run over the Internet
  • AppML must use Internet standards only
  • AppML must be platform independent
  • AppML must be compatible with XML and XHTML
  • AppML must support a variety of application needs
  • AppML must be self-describing
  • AppML applications must be easy to develop
  • AppML applications must be easy to maintain
  • AppML applications must be easy to change
  • AppML applications must be future proof
