I tried the command in Terminal.app to create an AppleDouble file, but couldn't find the resource/Finder attributes until I read the Wikipedia article on AppleDouble. I see now that the lost half was in a file name that starts with a period. I can't remember the command to take any Mac file and convert it to AppleDouble format, but what I realize now is that what I really want is the Terminal command to convert any file to the AppleSingle format -- or maybe better a function in the Carbon API to convert any Mac file to the AppleSingle format.

Take a look at SplitForks and FixupResourceForks, as well as ResMerger command-line developer tools. Hope this helps. MFNickster 00:31, 6 March 2006 (UTC)
