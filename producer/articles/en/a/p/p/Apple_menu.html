Apple menu

The Apple menu has been a feature in Apple's Mac OS since its inception. It is the first item on the left hand side of the menu bar. The Apple menu's role has changed throughout the release history of Mac OS, as detailed in the following sections. The menu has always featured a version of the Apple logo.

[edit] System 6 and earlier

The System 6.0.8 and earlier Apple Menu featured a Control Panel manager, as well as Desk Accessories such as a Calculator, the Scrapbook and Alarm Clock. If MultiFinder (an early implementation of computer multitasking) was active, the Apple Menu also allowed the user to switch between multiple running applications. The Macintosh user could add third-party Desk Accessories via the System Utility Font/DA Mover. However there was a limitation to the number of Desk Accessories that could be displayed in the Apple Menu. Third-party shareware packages such as OtherMenu added a second Apple Menu (with two Apple logos) which allowed users to install Desk Accessories beyond Apple's limitations.

[edit] System 7.0-9.2.2

System 7.0 introduced the Apple Menu Items folder in the System Folder. This allowed users to place Alias(es) to their favorite software and documents in the menu. The Menu Manager forced these additions into alphabetical order, which prompted users to rename their aliases with leading spaces, numbers and other characters in order to get them into the order that suited them the best. There were several third-party utilities that provided a level of customization of the order of the items added to the Apple Menu without having to rename each item. The Apple Menu also featured a Shut Down command, implemented by a Desk Accessory. An alias to the Control Panels folder is also present, allowing users to directly select one via a submenu. Corresponding to this change, System 7.0 was also the first version to feature the rainbow striped logo, as opposed to flat black as it was in previous versions.

System 7.0 featured built-in multitasking, and so MultiFinder was removed. The feature allowing users to switch between multiple running applications as in System 6 was given its own menu (appearing as the icon of the active application), on the opposite (right) side of the menubar.

Starting in Mac OS 8.5, this new menu was given a unique tear-off capability, which, when the user dragged the cursor downwards off the bottom of the menu, the menu detached from the menu bar to become a free-floating window. In this case it ran the application called Application Switcher (programmed under contract for Apple by James Thompson, author of DragThing).

[edit] Mac OS X

Mac OS X features a completely redesigned Apple Menu. System management functions from the Special menu have been merged into it. The quick file access feature implemented in System 7 was removed, though a popular 3rd-party utility, Unsanity's FruitMenu, restores the Apple menu to its classic functionality. The menu is now dedicated to managing the Macintosh computer, with commands to get info on the system, update software, open System Preferences, set Dock preferences, set the location (network configuration), view recent items (applications, documents and servers), Force Quit applications, power management (sleep, restart, shut down) and log out.
