Apple Symbols

Apple Symbols is a font introduced in Mac OS X 10.3 "Panther." The format is Truetype. Its function is to provide various Unicode symbols.
It ships with Mac OS X as part of the default installation. The path is

/Library/Fonts/Apple Symbols.ttf

[edit] List of Glyphs

The font has glyphs in the following Unicode blocks:

  • General Punctuation
  • Currency Symbols
  • Letterlike Symbols
  • Arrows
  • Mathematical Operators
  • Optical Character Recognition
  • Box Drawing
  • Block Elements
  • Geometric Shapes
  • Miscellaneous Symbols
  • Deseret

[edit] History

Apple lists this font as Copyright 2003.

[edit] Mac OS 10.3

  • Introduced with the following:
    • General Punctuation
    • Currency Symbols
    • Letterlike Symbols
    • Arrows
    • Mathematical Operators
    • Optical Character Recognition
    • Box Drawing
    • Block Elements
    • Geometric Shapes
    • Miscellaneous Symbols
    • Deseret

[edit] Mac OS 10.4

Version 5.0d5e1:

  • Braille
  • Yijing Hexagram Symbols
  • Small Form Variants
  • Musical Symbols
  • Tai Xuan Jing Symbols
  • glyph variants
