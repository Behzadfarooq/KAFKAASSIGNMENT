Application software is the general designation of computer programs used for performing specific tasks using a computer (e.g. word processing). This contrasts with system software which is the set of programs used to start and run the computer system and applications; and programming tools, such as compilers and linkers, used to translate and combine computer program source code and libraries into executable programs (programs that will belong to one of the three said categories).


For more information, see the article about application software.
