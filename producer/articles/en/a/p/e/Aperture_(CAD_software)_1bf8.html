Aperture (CAD software)

From Wikipedia, the free encyclopedia

Aperture Visual Information Manager, a software program for Microsoft Windows, is developed and marketed by Aperture Technologies, Inc, Stamford, Connecticut. Aperture Visual Information Manager combines a high precision drawing system (CAD) with database connections, enabling users to create sophisticated information systems for such applications as office space planning and data center floor planning. Originally developed for the Apple Macintosh in 1990, it now runs on the Microsoft Windows operating systems. The current shipping version is 9.0. Aperture Technologies (www.aperture.com) markets Aperture VISTA, an enterprise software application to manage the physical infrastructure of large scale data centers. It uses Aperture Visual Information Manager as a component of its solution.
