Ape Escape Academy 2 also known as Ape Academy 2 the sequel to the PSP party game Ape Escape Academy.

This game like its predecessor is a collection of minigames which can either be played in single player or multiplayer mode, though this one borrows many of the elements of Ape Escape 3. The game has already been released in Japan and a USA and European release date is expected soon. The game will be released in Australia on October 19, before the USA release date which is October 2006
