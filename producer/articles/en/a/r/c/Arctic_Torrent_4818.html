Arctic Torrent

Screenshot of Arctic
Developer: Don Phrostbyte
Latest release: 1.2.3 / September 4, 2004
OS: Windows
Use: BitTorrent client
License: MIT License
Website: dev.int64.org/arctic

Arctic Torrent is an open source minimalist C++ BitTorrent client for Windows XP/2k/2k3.

The program is designed to use as little RAM and CPU power as possible.
