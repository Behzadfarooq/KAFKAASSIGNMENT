Hi Archenzo, I saw your changes on some footers, particularly Template:Biology-footer and Template:Genomics-footer. Making the table 100% particularly for the genomics footer makes the footer look very strange because there is only a small number of items, and having it stretched so much looks odd on most browsers. Since most page footers, such as country footers (such as Template:North America don't use width=100% (in fact most are indented by default), since this is a site-wide issue if you want to change things globally, you should probably discuss this on at Wikipedia talk:Page footers first.
