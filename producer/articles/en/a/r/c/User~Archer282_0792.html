[edit] About Me

First allow me to explain the meaning of Archer282... The sign of Sagittarius happens to be an Archer or a Centaur. As I'm sure you can guess from this information, my Zodiac sign is Sagittarius. As for the number, when coming up with a name and realising "Archer" would normally be taken. I decided if I must use a number that I wanted to use a number with some sort of meaning behind it. So eventually I decided that I would use the number sequence you would dial, if you were to dial my initials (A.T.C.) on a standard telephone.
