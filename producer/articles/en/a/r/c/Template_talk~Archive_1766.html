[edit] link

I'm currently using [[talk:{{NAMESPACE}}:{{PAGENAME}}|talk page]] which should work on all spaces perfectly but I'm interested in knowing if this will work everywhere and if this is the most efficent way to link to an archive page's talk page since this would require pulling up the link every time a user loaded the page. Jtkiefer T | @ | C ----- 09:00, July 30, 2005 (UTC)

This is a sister template to Template:Talkarchive. This specific template is designed for use on page archives, the major difference between this and the talk archive tag is that this jas a talk page link and it is phrased slightly differently. Jtkiefer T | @ | C ----- 04:57, August 1, 2005 (UTC)

[edit] Fixed Talk page.

Talk page was [[Talk:{{NAMESPACE}}:{{PAGENAME}}|talk page]] I changed it to [[{{NAMESPACE}}_Talk:{{PAGENAME}}|talk page]] And I tested it and on the template page it takes me here. --Actown 06:47, 5 November 2005 (UTC)

[edit] Category.Archived ?

I think this template as well as Template:Talkarchive should be placed into a category such as Archive, Archived, or Archives, thereby allowing a quick check using Recentchangeslinked of any clueless edits or vandalism done to any archived pages (barring template removal/blanking). Example: Uncyclopedia:Special:Recentchangeslinked/Category:Archives Splarka (rant) 05:07, 23 March 2006 (UTC)

Ah, there we go. Check out the newly created subcategories in Category:Wikipedia history. They're organized by {{ARTICLESPACE}} (for {{Archive}}) or {{TALKNAMESPACE}} (for {{Talkarchive}}). ~MDD4696 04:29, 20 May 2006 (UTC)
Very groovy. Splarka (rant) 07:06, 20 May 2006 (UTC)
