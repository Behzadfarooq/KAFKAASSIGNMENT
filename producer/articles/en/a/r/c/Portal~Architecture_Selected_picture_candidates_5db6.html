[edit] Procedure

[edit] Nominate

  • Place new candidates at the bottom of the current nominations list.
  • Use this borderless format for the image [[Image:candidate.jpg|none|center|280px|]]
  • Include a photo credit, after the image <small>Photo credit: [[User:A.N.Other|A.N.Other]]</small>
  • Optional: Add a short caption for the image from the relevant article, less than 100 words +/–

[edit] Discuss

Post comments or votes below each image, and sign the message ~~~~

  • Nominate / Comment / Support / Oppose
  • Place the {{Architecture SI}} tag into the Image:talk page
"This picture has been nominated for Selected picture, at the Architecture portal"

[edit] Archive

  • Remove "Archive" and "Nominations" from the Selected picture file
  • "Include" the Selected image file as a template to the Selected picture archive by enclosing the filename in curly brackets

[edit] Post

  • Change the {{architecture SI}} to {{architecture SI|2007|2}} for week 2 in 2007, for example
  • Automatic weekly rotation on Portal:Architecture when using the standard filename format
