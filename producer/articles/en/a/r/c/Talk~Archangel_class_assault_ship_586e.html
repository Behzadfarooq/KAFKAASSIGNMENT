[edit] Positron Cannons Removed?

Anyone notice that in SEED Desntiny the archangel's positron cannons seems to be missing?

A positron cannon being fired in Earth's atmosphere can cause gamma ray radiation to pollute the surrounding environment, so Murrue Ramius avoided using it in combat on Earth.
