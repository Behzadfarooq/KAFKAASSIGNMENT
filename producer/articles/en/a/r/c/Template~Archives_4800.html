[view] [edit] Template documentation

{{archives}} Adds a listbox containing links to archived discussions

Usage

{{archives
  |width = optional specific width
  |archivelist = optional other list to point to
  |index = relative path to index file
  |small=yes
  |auto=[long|yes]
}}
