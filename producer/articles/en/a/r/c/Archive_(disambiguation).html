Archive may refer to:

document or data storage
  • Archive, a collection of documents, or the location they are stored
  • Archive Corporation, a data storage company active in the 1980's.
  • Archive site, a website that stores information from the World Wide Web
  • File archiver, a computer program that combines a number of files together into one archive file
  • Archive bit, a file system attribute used for making incremental backups
  • Web archiving, the process of archiving the World Wide Web


In music
  • Archives (Asia album) a 1997 album by Asia
  • Archive (band), a trip-hop band
  • Archives (Darkest Hour album), a CD by Darkest Hour
  • Archives (Rush album), a 1978 album by Rush
