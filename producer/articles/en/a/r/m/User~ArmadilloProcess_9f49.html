[edit] British railways

(November 2006)
In some British Railway station articles, both the {{UKrailwaystations}} and {{UK stations}} templates have been used. This is redundant, the smaller template give an alpha directory to other stations, and this feature is already in the larger template. In these cases, remove the {{UKrailwaystations}}

See Template talk:Infobox UK station

[edit] Notes

Userboxes

1. Many userboxes do not have the following code
<noinclude> [[Category:xxxxxx]] </noinclude>
and this causes the user of the box to get his/hew userpage included in the template category - which is incorrect

2. Users who subst a userbox, will get the category in their userpage, and this ends up putting the user page in the template category they should note the instructions from each template category

Note to users who subst: user templates

After you save the user page, please edit it again to insure that no user template categories have been placed on the user page. User pages are not user templates, so they do not belong in user template categories.
