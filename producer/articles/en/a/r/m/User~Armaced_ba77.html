[edit] Contrawiktion

An editor disagrees with a sentence, but rather than rewriting it, adds a "however" clause. These can stack up pretty deep. Examples:

  • 1. Contrawiktion level: 3
      • (1)Thrintun had been mentioned in a number of the earlier Known Space stories, most notably in The Soft Weapon
      • (2)but this is the only one of Nivenâ€™s stories they appear in
      • (3)with the possible exception of The Handicapped.
    • Rewrite: The Thrintun had been mentioned in the earlier Known Space stories The Soft Weapon and The Handicapped.
  • 2. This is a weak example since it does not exactly contradict itself as much as deliver information in a disorganized manner. Contrawiktion level: 2
      • The PlayStation Portable features an IrDA port located on the top left of the device;
      • (1)however this is not currently used by any games or any system features,
      • (2)but is by several homebrew applications
    • Rewrite: The PlayStation Portable features an IrDA port located on the top left of the device. To date, the only games or applications to leverage this feature have been homebrew.
