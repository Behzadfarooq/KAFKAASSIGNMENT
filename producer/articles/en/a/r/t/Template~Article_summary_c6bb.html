The {{Article summary}} template is used to produce a summary text for any article. It does not have any additional parameters, but rather draws the body of the summary text from the ../Summary page which can be generated through the template.

[edit] Instructions

  1. Place the template in the "see also" section of the article page.
  2. Save the edit. An empty Translation summary title should now appear on the article page.
  3. Use the edit link inside the summary box to edit the summary text.
  4. Save your edit.
  5. Return to the original article. The summary box should now contain your edit.

Further discussion of this template can be found at WikiProject Organized Labour/Summaries.
