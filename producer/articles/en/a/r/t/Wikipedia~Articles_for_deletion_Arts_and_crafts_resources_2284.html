< Wikipedia:Articles for deletion

This page is an archive of the discussion about the proposed deletion of the article below. This page is no longer live. Further comments should be made on the article's talk page rather than here so that this page is preserved as an historic record.
The result of the debate was delete with an option to transwiki somewhere.

After reviewing the article and the associated images, I started to question whether this would fit in the mandate for Wikibooks. I also started to ask if the images are copyrighted. In order to allow a bit more time for investigation, I've moved the article to the primary author's user space User:Chrissy snow/Arts and crafts resources. I will work with the author to determine if this can/should be transwiki'd or ultimately deleted. Rossami (talk) 22:37, 24 Jan 2005 (UTC)

[edit] Arts and crafts resources

I'm not too happy nominating this as clearly a lot of work has gone into it, but this simply isn't what Wikipedia is for. Is there another Wikimedia project it could live on? David Johnson [T|C] 15:57, 8 Jan 2005 (UTC)

  • Strongly agree this content would be helpful somewhere, WP is not a web guide, so this is a tough one... Delete. Wyss 18:03, 8 Jan 2005 (UTC)
  • Delete. Original research, no idea if the links chosen are chosen because of any particular bias, etc. RickK 00:45, Jan 9, 2005 (UTC)
    • Concur. Edeans 09:14, 9 Jan 2005 (UTC)
  • Transwiki. A lot of work went into this, and it would be a shame to lose it. I'm thinking that WikiBooks might be a good place to put it, though I admit that I don't know enough about WikiBooks to make a good recommendation. At the very least, I'd like to recommend that the author place this on her user space (or maybe a personal web site) before it gets deleted. --Deathphoenix 16:34, 11 Jan 2005 (UTC)
  • A lot of effort, and worthy info, some of this might be condensed and added to Arts and crafts wikibooks might be a good place to point this contributor, do recommend we inform the creator before this is deleted Pedant

This page is now preserved as an archive of the debate and, like some other VfD subpages, is no longer 'live'. Subsequent comments on the issue, the deletion or on the decision-making process should be placed on the relevant 'live' pages. Please do not edit this page.
