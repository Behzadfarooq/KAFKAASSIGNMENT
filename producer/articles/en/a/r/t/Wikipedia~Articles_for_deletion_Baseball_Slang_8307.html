[edit] Baseball_Slang

This page needless duplicates the "Baseball slang" page. This is not to be confused with this page which is entitled "Baseball Slang". This page is not formated as well, nor contains as much information as the similarly named other page. The other page is properly linked with List_of_baseball_jargon. Only one page needs to exist, and it should be the other page. 70.106.208.134 06:42, 4 August 2005 (UTC)
