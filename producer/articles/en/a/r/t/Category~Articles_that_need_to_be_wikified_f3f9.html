    Transwiki log cleanup

    Articles that have been transwikied and need to be checked for possible merging or deletion.
    Main - Inactive - Mini

    These articles need to be wikified. This can be done in a few easy steps:

    1. Find an article to work on. Articles are now sorted by date. Many editors choose to work on the oldest articles first, so that none of them sit around for too long waiting for attention. It should be relatively easy to wikify an article even if you are not familiar with the subject.
    2. Check if the article is a copyright violation. There are a few red flags that indicate text was probably illegally copied into Wikipedia - this actually happens quite frequently. User:Kjkolb/Copyvio has a quick guide. Instructions for reporting violations are at Wikipedia:Copyright problems.
    3. Check if another article already exists on this subject. New users often do not do this before creating articles, which leads to content forks. Use the search button at the left of the screen to check for duplicate articles, and create a redirect if necessary.
    4. Wikify! Make links to other articles, format the lead (first sentence), and arrange section headers as described at Wikipedia:Guide to layout.
    5. Remove the {{wikify}} tag. That will remove it from the list.
    6. Articles that have the correct formatting, but are messy should NOT be tagged with the wikify tag. Use cleanup or copyedit instead.
    7. Enhance your edit summary. You can encourage others to help with the wikification backlog by using something like the following as your edit summary when you save your changes:
    Wikified as part of the [[WP:WIKIFY|wikification drive]].

    Repeat as often as you like! Wikification can go quite quickly once you get the hang of it.

    There is also a template that you can add to the talk page of the article you are currently working on to help keep track of what you've done and organize a wikification effort. {{User:The Thadman/Template/Wikification}}

    And a cut-down, modified version: {{User:Draicone/Wikify}}

    Since you are looking at the article anyway, it would also be helpful if you:

    • Nominate the article for deletion, if appropriate. Wikipedia:Criteria for speedy deletion is a good guide; add {{subst:prod|your reason for deletion here}} to the top of the article if you think deletion would be uncontroversial. See Wikipedia:Articles for deletion if you think discussion is warranted or you aren't sure.
    • Tag for other problems. If you happen to notice that an article has problems beyond simple formatting and you can't fix it yourself, please add the appropriate tag from Wikipedia:Cleanup resources to the article itself. Put short articles (less than about 3 paragraphs) in one of the stub categories or else add the generic {{stub}}.

    [edit] How to add articles to the list

    Add the tag {{wikify}} to the top of the article. A bot will automatically change this to {{wikify|April 2007}}. Any articles you see listed near the bottom of this page have been tagged and are waiting to be automatically sorted into a monthly subcategory.

    [edit] Bots

    • Pearle and DraiconeBot do the per-month categorization.
    • SmackBot canonicalises templates and dates them.
    • Bluebot adds and sorts wikification tags.
    • Gnome is preparing to add wikification tags.
