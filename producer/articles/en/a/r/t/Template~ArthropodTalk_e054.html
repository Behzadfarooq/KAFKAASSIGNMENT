[edit] About the template

This template adds article talk pages, category talk pages, template talk pages, etc (whatever it is placed on) to Category:WikiProject Arthropods articles. It is also used to track article classifications and presents readers with a current "to do" list.

Please be very careful when editing this template if you don't understand MediaWiki Parser Functions. This template is now rather complicated and could easily be broken. (Note that you may find m:Magic words and m:ParserFunctions very helpful in understanding what this does... consider consulting with User:Sarefo before making any changes, please)

[edit] How to use this template

  • The template always goes onto Talk pages, not article pages. The template must be transcluded and not substituted (using subst).
  • The template accepts two optional parameters.
    • The following fixed values denote the "article quality" as decided upon at WP:ARTH. Passing one of these values as the parameter places the Talk page into the applicable category which is scanned by Mathbot to create the Wikipedia Version 1.0 article lists.
      • If the attached article is a featured article, use {{ArthropodTalk|FA}}
      • If the attached article is a good article, use {{ArthropodTalk|GA}}
      • If the attached article is A-Class, use {{ArthropodTalk|A}}
      • If the attached article is B-Class, use {{ArthropodTalk|B}}
      • If the attached article is Start-Class, use {{ArthropodTalk|Start}}
      • If the attached article is Stub-Class, use {{ArthropodTalk|Stub}}
      • If the attached article is Merge-Class, use {{ArthropodTalk|Merge}}
      • If the attached article is MergeDel-Class, use {{ArthropodTalk|MergeDel}}
      • If the attached article is Merged-Class, use {{ArthropodTalk|Merged}}
      • If the attached article is AFD-Class, use {{ArthropodTalk|AFD}}
    • If any other value is supplied as the parameter (such as "template", "category", or even an incorrect lower case "start") the parameter is presumed to be a word describing what sort of page it is. Use this if the attached article is not in the main space (it's a category, project page, etc). For example:
    {{ArthropodTalk|category}} outputs "This category is part of ..."
    • Parameter 2 denotes the article's importance. The possible values are:
      • Top
      • High
      • Mid
      • Low
      • NA, for no importance rating (e.g. if the article was merged and should have never have existed in the first place - see for example - from the Beatles WikiProject - Talk:Baby, You're a Rich Man).
      • No value places the article in Category:Unassessed-importance Arthropods articles.

For main space articles which have not yet been assessed, simply use {{ArthropodTalk}}. This default places the article in Category:Unassessed Arthropods articles and Category:Unassessed-importance Arthropods articles, and doesn't change the wording of the template in any way.
