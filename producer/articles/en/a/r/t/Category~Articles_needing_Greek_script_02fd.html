This is a category of articles that need the Greek script versions of their names added to the lead paragraphs, or in an appropriate template. See Wikipedia:Naming conventions (Greek).

To add an article to this list, add {{Greek}} to the top of the article's talk page.

If you add the Greek script to an article in this category, simply remove the {{Greek}} from the talk page, and it will no longer show up in this list. Many of the articles in this category also need to be cleaned up, wikified, or expanded. Feel free to help these articles out while you add the Greek script to them, if you like.
