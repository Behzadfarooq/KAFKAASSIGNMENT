Transitive set

Also a warm welcome from me. In transitive set, you write that A is transitive if "whenever x∈A, and y∈x, then y∈x." There seems to be some mistake here; perhaps the last "y∈x" should be "y∈A"? Apart from that, it is a nice start, though it could probably be expanded (I don't know enough set theory to do it myself). By the way, you can write [[class (set theory|]] instead of [[class (set theory)|class]].
