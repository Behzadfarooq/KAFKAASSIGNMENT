[edit] No source tag

When you are using the no source tag on images, you need to include the parameters like {{no source|month=January|day=17|year=2006}}. Otherwise, it shows up in Category:Images with unknown source as of unknown date 2006. -- Ricky81682 (talk) 05:45, 18 January 2006 (UTC)
