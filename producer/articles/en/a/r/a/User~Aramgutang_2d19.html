[edit] Unicode conversion:

After getting frustrated with typing foreign (mainly Russian and Armenian) letters in the &#XXXX; format in my articles, I wrote a simple PHP script to convert any text you type in UTF-8 (Unicode) into the Wikipedia format.

 <html><head><meta http-equiv="Content-Type" charset="UTF-8"></head><body>
 <?php
 $text = $_POST['text'];
 for($i=0; $i<strlen($text);)
 {
        $c = decbin(ord($text{$i}));
        if(strlen($c)<8)
                echo htmlspecialchars($text{$i++});
        else
        {
                $m = strpos($c, '0');
                $i += $m;
                $f = substr($c, $m+1);
                for(--$m;$m>0;$m--)
                        $f .= substr(decbin(ord($text{$i-$m})), 2);
                echo '&#'.bindec($f).';';
        }
 }
 ?>
 <p><form action="unicode.php" method="post">
 <input type=text name="text"><input type=submit></form></body></html>

Anything you type in the box and click on "submit" will show up at the top will all non ASCII characters replaced with &#XXXX; equivalents. This should work on PHP 3 through 5, although I only tested it on 4, with MSIE as my browser. Make sure you change the "unicode.php" line if you name the file something else. Any improvements and bugfixes you come up with are welcome, just post them here.
