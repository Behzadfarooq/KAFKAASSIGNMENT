Area code 705

From Wikipedia, the free encyclopedia

Area code 705 (1-705) includes most of Northeastern Ontario and Central Ontario. The code was created sometime during 1957 when it split from portions of 613 and 519. Area code 807 was later split out of Area Code 705, sometime during early 1962.

The incumbent local exchange carriers in 705 are Bell Canada, Northern Telephone, Ontario Northland Telecommunications and some municipally-owned carriers.
