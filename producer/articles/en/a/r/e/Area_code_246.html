The area code (246) is the local telephone area code of Barbados. The (246) area code, or "(BIM)" perhaps short for "Bimshire" -- was created during a split from the original (809) area code which was done on or around the date July 1996.

When in Barbados, use the seven digits alone. When calling to Barbados from anywhere in the United States or Canada simply dial 1(246) + seven digit phone number.
