Area code 937 is a telephone area code serving much of the southwestern part of the U.S. state of Ohio. It covers Marysville, Springfield, Fairborn, Dayton, Hillsboro, public parts of Wright-Patterson Air Force Base, and areas north of Cincinnati. It split from area code 513 in 1996. One possible reason for area code 937 is that it spells "YES" on a touch-tone phone.
