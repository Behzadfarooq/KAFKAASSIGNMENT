Area code 780 is a telephone area code in the Canadian province of Alberta, encompassing the northern two-thirds of the province. The code was established in 1997; prior to this date the entire province was served by the 403 area code.

The incumbent local exchange carrier in 780 is Telus.

[edit] Relief Planning

As of October 2006, the projected exhaust date for Area Code 780 is November of 2011. For more information see the Canadian Numbering Administrator Website. A proposed solution is that a new area code, either 587 or 825, will be assigned by spring of 2007. The 780 area code, simultaneously with 403, will be overlaid with the new area code, which will cover the entire province. Ten-digit dialing (area code plus seven-digit subscriber number) will be made mandatory province-wide.
