Area code 813

From Wikipedia, the free encyclopedia

Area code 813 is the telephone numbering plan code for the city of Tampa, Florida and surrounding areas such as Zephyrhills and Oldsmar.

[edit] Description

Area Code 813 covers all of Hillsborough County, the eastern two-thirds of Pasco County, and the city of Oldsmar in Pinellas County.

[edit] History

A map of the North American Numbering Plan (NANP) in 1947 showed the region originally belonged to the Area Code 305, an area code that once covered the entire state of Florida and now serves only Miami-Dade County and portions of Monroe County, Florida. In 1953, 813 become Florida's third area code. The area code stretched from Pasco County to Collier County and include the cities of St. Petersburg, Sarasota, Fort Myers, and Lakeland.

[edit] Area Codes that split from 813

[edit] Area Code 941

On March 3, 1996, 13 counties south and east of Hillsborough County from Manatee County to Collier County split from the 813 area code after 43 years in the coverage area to form the new 941 area code. Seven of these counties (Polk, Hardee, Desoto, Highlands, Okeechobee, Galdes, and Hendry), in May of 2000, split from 941 area code to form the 863 area code.

[edit] Area Code 727

On February 1, 1999, most of Pinellas County (except Oldsmar) and western Pasco County were the latest areas to date that split from the 813 area code to form Area Code 727 code, leaving the present area. Oldsmar, being right on the county line between Pinellas and Hillsborough stayed 813 despite being in Pinellas because its trunk was wired into the Tampa line and it would cost too much to rewire it.
