Area code 954

From Wikipedia, the free encyclopedia

Area code 954 went into service on September 11, 1995, for all telephones within Broward County, Florida. It replaced Area code 305 for that area. On August 1, 2001, the area code was overlaid with Area code 754.

Notable cities within Area code 954 are Fort Lauderdale, Sunrise, Hollywood, Coral Springs, and Pembroke Pines
