Area code 470

From Wikipedia, the free encyclopedia

Area code 470 is a future telephone area code serving the state of Georgia. It covers metro Atlanta, and overlays with the area codes 404, 678, and 770.

There is some opposition to the proposed new area code. Atlanta phone numbers are commonly identified as 4)xxx-xxxx, 7)xxx-xxxx, and 6)xxx-xxxx, representing the 3 overlapping area codes 404, 770, and 678. A 470 area code would conflict with the 4) and 7) abbreviations currently used by thousands of individuals and businesses.
