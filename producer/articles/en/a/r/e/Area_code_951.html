Area code 951

From Wikipedia, the free encyclopedia

Map of California area codes in blue (and border states) with 951 in red

North American area code 951 is a California telephone area code which covers western Riverside County and was split off from the 909 area code in 2004. It covers Corona, Canyon Lake, Riverside, Temescal Canyon, Woodcrest, Arlington, Mira Loma, Moreno Valley, Perris, Sun City, Lake Elsinore, Murrieta, Temecula, Hemet, Lakeview Nuevo, Banning, Eastvale and Idyllwild.
