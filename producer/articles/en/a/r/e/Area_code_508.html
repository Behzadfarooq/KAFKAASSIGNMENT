Area code 508 was split from area code 617 on July 16, 1988. In addition, northeastern Massachusetts, originally a part of the 508 area code, was given the 978 area code designation in 1998. The 508 area code district is overlaid by area code 774, and services southeastern and southcentral Massachusetts, including the cities of Taunton, Worcester, and Fall River, Cape Cod, and Martha's Vineyard and Nantucket.
