The 262 area code is a telephone area code in Southeastern Wisconsin. It currently covers much of the suburbs of Milwaukee, Wisconsin, but not Milwaukee County itself.

On September 25, 1999 the 262 area code was created as a split from the 414 area code. The 414 area code was retained for the Milwaukee County phone numbers, and 262 area code was assigned to most areas outside of Milwaukee County.

Cities with more than 5,000 residents in the 262 area code include:

  • Brookfield
  • Burlington
  • Cedarburg
  • Delafield
  • Delavan
  • Elm Grove
  • Germantown
  • Grafton
  • Hartford
  • Hartland
  • Kenosha
  • Lake Geneva
  • Mequon
  • Muskego
  • New Berlin
  • Oconomowoc
  • Port Washington
  • Racine
  • Waukesha
  • West Bend
  • Whitewater
