Area code 575 is a proposed area code for the State of New Mexico, intended to serve the remainder of the state outside of the Albuquerque and Santa Fe metropolitan areas, which will remain in Area code 505. The new code is scheduled to be implemented in 2009. As the exact details of the new area code are not yet available, the current map is being shown.
