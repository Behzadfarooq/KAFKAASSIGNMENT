North American area code 830 is a state of Texas telephone area code for numbers in the San Antonio area. It was created July 7, 1997, in a split from area code 210 which this area code completely surrounds.

Counties served by this area code:

Atascosa, Bandera, Bastrop, Blanco, Burnet, Caldwell, Comal, De Witt, Dimmit, Edwards, Frio, Gillespie, Guadalupe, Karnes, Kendall, Kerr, La Salle, Live Oak, Llano, Maverick, Medina, Real, Uvalde, Val Verde, Wilson, and Zavala
