North American area code 603 is the State of New Hampshire's sole area code serving the entire state. Area code 603 was created as one of the original area codes in October, 1947.

State of New Hampshire Area Code: 603
North: 819
West: 802 area code 603 East: 207
South: 351/978
Commonwealth of Massachusetts Area Codes: 339, 351, 413, 508, 617, 774, 781, 857, 978
State of Maine area code: 207
Province of Quebec area codes: 418, 438, 450, 514, 819
State of Vermont Area Code: 802
