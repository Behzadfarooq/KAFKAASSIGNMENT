Area code 269 is the telephone area code serving the southwest portion of Michigan's Lower Peninsula. Cities served by this area code include Kalamazoo/Portage, Battle Creek, and Benton Harbor/St. Joseph. It was created as a split from Area code 616 in July 2002.
