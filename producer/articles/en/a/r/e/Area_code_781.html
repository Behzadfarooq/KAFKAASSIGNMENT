Under the North American Numbering Plan for telephone area codes, area code 781 covers suburban Boston, Massachusetts. In particular, it covers Winchester, Melrose, Saugus, Revere, Malden, Burlington, Norwood, Waltham, Woburn, Medford, Lexington and other Boston suburbs along Route 128. It overlays with area code 339. Area code 781 was created in 1997 by a split from area code 617.
