Area code 973

From Wikipedia, the free encyclopedia

Area code 973 is a North American Numbering Plan area code for Northern New Jersey that covers portions of Bergen, Essex, Hudson, Morris, Passaic and Sussex Counties. It was formed in 1997 when the Area code 201 was split. In 2001, the 862 area code was created to overlay the 973 area code, making 10-digit dialing mandatory.
