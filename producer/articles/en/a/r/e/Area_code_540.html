Area code 540

From Wikipedia, the free encyclopedia

North American telephone area code 540 was established July 15, 1995 as a split from area code 703. It was itself split on September 1, 2001 to create area code 276.

Area code 540 serves the central and north portions of the Commonwealth of Virginia including the communities of Blacksburg, Cave Spring, Fredericksburg, Harrisonburg, Roanoke, Salem, Staunton, and Winchester, as well as most of Stafford County, Spotsylvania County, and Fauquier County.
