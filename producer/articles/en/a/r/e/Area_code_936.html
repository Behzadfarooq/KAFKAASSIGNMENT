North American area code 936 is a state of Texas telephone area code for numbers in the Huntsville area. It was created February 19, 2000, in a split from area code 409.

Counties served by this area code:

Angelina, Brazos, Cherokee, Grimes, Hardin, Harris, Houston, Leon, Liberty, Madison, Montgomery, Nacogdoches, Polk, San Augustine, San Jacinto, Shelby, Trinity, Tyler, Walker and Washington

Towns and cities served by this area code:

Ace, Alto, Anderson, Apple Springs, Batson, Bedias, Broaddus, Center, Chester, Chireno, Coldspring, Conroe, Corrigan, Crockett, Cushing, Daisetta, Dayton, Devers, Diboll, Dobbin, Dodge, Douglass, Etoile, Flynn, Garrison, Goodrich, Grapeland, Groveton, Hardin, Hockley, Hull, Huntington, Huntsville, Iola, Joaquin, Kennard, Latexo, Leggett, Liberty, Livingston, Lovelady, Lufkin, Madisonville, Martinsville, Montgomery, Moscow, Nacogdoches, Navasota, Navasota, New Waverly, Normangee, North Zulch, Oakhurst, Onalaska, Pennington, Plantersville, Pointblank, Pollok, Ratcliff, Raywood, Reklaw, Richards, Riverside, Sacul, San Augustine, Saratoga, Shelbyville, Shepherd, Shiro, Tenaha, Timpson, Trinity, Votaw, Waller, Washington, Wells, Willis, Woden, and Zavalla
