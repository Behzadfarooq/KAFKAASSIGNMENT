Area code 413 is the area code for Western Massachusetts. It is the largest area code area by land area covered in the state, extending from the western state line to the border of Worcester County, though including two Worcester county towns, Hardwick and Warren, while excluding several eastern Franklin County towns, Orange, New Salem, Warwick and Wendell and covering from border to border north to south. As Western Massachusetts has no overlay area code (in other words it has only one area code), it is the only part of Massachusetts which does not require 10 digit dialing. If the rise of extra phone numbers necessitates it, another area code might be added, ending this, unless it is decided to divide the existing 413 territory accordingly.
