Template:Area Code Box 2

From Wikipedia, the free encyclopedia

You have new messages (last change).
{{area_code_{{{State}}}}}
North: {{{N}}}
West: {{{W}}} {{{This}}} East: {{{E}}}
South: {{{S}}}
{{area_code_{{{S1}}}}}
{{area_code_{{{S2}}}}}


Usage: {{Area Code Box n||State=|This=|N=|S=|E=|W=|S1=|S2=}} etc., where n is the number of border states/provinces the area code touches from 1 to 6, e.g. {{Area Code Box 3|parameters}} or {{Area Code Box 5|parameters}} etc. and parameters are the same as this template (explained below), and S1=,S2= are the area code lists for border states/provinces to appear at the bottom, according to the box number. Note that you do not use the phrase parameters in calling any of these transclusion macros.

The parameters are used as follows:

State is the 2-letter abbreviation to include the area code template for this state or Canadian province.
This is this area code, used for the center box.
N=, S=, E= and W= are the area codes (or oceans) that border this area code in those compass directions, and
S1=XX for the first nearby state or province to this area code, where XX is the two letter abbreviation.
S2=XX for the second nearby state,

and so on for as many states as border this area code. The S1 or S2 etc. code should match the number of the particular Area Code Box template used, e.g. for {{Area Code Box 5}} you would want to code S1=XX through S5=XX. Optional Parameter:

Width= to indicate the width of the page the macro should use; it defaults to 80%


For an area code where all surrounding area codes are in the same state/province use {{Area Code Box 0|State=|This=|N=|S=|E=|W=}} where you want the table.
