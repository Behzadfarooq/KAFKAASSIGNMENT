Area code 337

From Wikipedia, the free encyclopedia

Area code 337 is a telephone area code that covers southernwestern Louisiana. The area code was split off from the southern half of the original area code 318, which had covered the western half of Louisiana.

[edit] Cities in the 337 Area Code

  • Lake Charles, Louisiana
  • Lafayette, Louisiana
  • Sulphur, Louisiana
  • Ville Platte, Louisiana
  • Crowley, Louisiana
  • Abbeville, Louisiana
  • Franklin, Louisiana
  • Jeanerette, Louisiana
