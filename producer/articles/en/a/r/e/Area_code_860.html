Area code 860 is a telephone area code that covers the eastern and northwestern parts of Connecticut. The area code stretches along the Massachusetts and Rhode Island borders as well as the northern portion of the New York with the boundary with Area code 203 running along towns such as New Milford, Watertown, and Southington. The area code was created in 1995 to replace 203 when that area code was cut back solely to Fairfield and New Haven counties.

[edit] Major cities and towns within area

  • Bristol
  • Hartford
  • Litchfield
  • Manchester
  • Middletown
  • New Milford
  • New London
  • Storrs
  • Torrington
  • Willimantic

[edit] Proposed area code overlay

The CT Public Utility commission has proposed an area code overlay for the 860 area code. In 1999, a new area code 959, was proposed to overlay 860. This proposal has since been withdrawn.
