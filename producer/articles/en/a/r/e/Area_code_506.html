Area code 506

From Wikipedia, the free encyclopedia

Area code 506 is the telephone area code in the Canadian province of New Brunswick, encompassing the whole province. The code was created sometime during 1955, when New Brunswick and Newfoundland (including Labrador) split from the 902 Area Code; Newfoundland (including Labrador) split to its own 709 Area Code sometime in early 1962.

The incumbent local exchange carrier in 506 is Aliant (part of the Bell Aliant income trust), which was produced from a merger that included NBTEl.In 2005 Eastlink Communications brings competitive local telephone service to New Brunswick. [1]
