Area code 814

From Wikipedia, the free encyclopedia

Area code 814 is a Commonwealth of Pennsylvania telephone area code which serves the central and northwestern region of the state including the cities of Altoona, Erie, State College and Johnstown. 814 is one of the original area codes established in 1947.
