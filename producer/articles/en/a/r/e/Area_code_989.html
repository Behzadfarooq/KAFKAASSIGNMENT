Area code 989

From Wikipedia, the free encyclopedia

Area code 989 is the area code for central Michigan, the northeastern Lower Peninsula, and portions of the Thumb. Major cities include Alpena, Bay City, Midland, Mt. Pleasant, and Saginaw. The area currently served by 989 was formerly part of area code 517.

[edit] Other 989 area codes

  • 989 is an area code for the sending of telexes to Sierra Leone.[1]
  • 989 is also an area code used in Iran for Mobile phones. [2]
