Area code 862 is a North American Numbering Plan area code for Northern New Jersey that covers portions of the Counties of Bergen, Essex, Morris, Passaic and Sussex. It overlays with area code 973 and was formed in 2001.

According to Verizon, area code 862 is used in the state of New Jersey by the following communities:

  • Belleville
  • Bloomfield
  • Byram Township
  • Clifton
  • East Orange
  • Garfield
  • Hanover Township
  • Irvington
  • Livingston
  • Lodi
  • Maplewood
  • Montville Township
  • Montclair
  • Morristown
  • Newark
  • Nutley
  • Orange
  • Parsippany-Troy Hills
  • Passaic
  • Paterson
  • Randolph
  • Sparta
  • Wayne
  • West Milford
  • West Orange
  • Whippany
