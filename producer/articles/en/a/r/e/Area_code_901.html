Area code 901 is an area code in the U.S. state of Tennessee. Originally it was the area code for the entire state, but now is limited to the southwest corner of the state.

[edit] History

[edit] Original assignment

Under the original assignment of area codes in 1947, area code 901 covered the entire state of Tennessee.

[edit] Splits and overlays

[edit] 1954 split

In an area-code split that was a flash-cut sometime during 1954, area code 901 was restricted to the western region of Tennessee;while the Eastern two-thirds of Tennessee were split off and reassigned the 615 area code.

[edit] 2001 split

In an area-code split that became mandatory on September 17, 2001, area code 901 only serves Shelby and Tipton Counties and portions of Fayette, Lauderdale and Hardeman Counties, a district very roughly corresponding with the Tennessee portion of the Memphis metropolitan area. The remaining areas formerly served by 901 were reassigned the 731 area code.
