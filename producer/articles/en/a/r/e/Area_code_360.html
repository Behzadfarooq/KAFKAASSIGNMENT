Area code 360

From Wikipedia, the free encyclopedia

Area code 360 is the area code for western Washington outside of the greater Seattle metropolitan area. It began service on 15 January 1995. The area, which encompasses all of western Washington outside of urban King, Pierce, and Snohomish counties and Bainbridge Island, was previously part of Area code 206. The 360 area code was one of the first two area codes that did not take the N1X or N0X form (i.e., the middle digit is neither 0 nor 1) in the North American Numbering Plan, along with Alabama area code 334, which began service on the same day as 360.

In 1999, an overlay area code of 564 was to be introduced to the entire 360 area; however the implementation has been delayed indefinitely by order of the Washington Utilities and Transportation Commission.
