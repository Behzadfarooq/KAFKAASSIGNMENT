Area code 601

From Wikipedia, the free encyclopedia

Area code 601 is the telephone area code for central Mississippi. Originally, area code 601 covered the entire state. In 1997, area code 228 was created for the Mississippi Gulf Coast area. In 1999, area code 662 was created in the northern half of Mississippi. In 2005, Mississippi's first overlay area code, area code 769, was approved for use to overlay 601. Because of the overlay, 10 digit dialing was made mandatory in the 601 area code.
