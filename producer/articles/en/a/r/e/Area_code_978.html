Area code 978 is a Commonwealth of Massachusetts area code serving the communities of Fitchburg and Peabody as well as northeastern Massachusetts. Area code 978 was created as a split from area code 508 on September 1, 1997; area code 351 was created as an overlay to 978 on May 2, 2001.
