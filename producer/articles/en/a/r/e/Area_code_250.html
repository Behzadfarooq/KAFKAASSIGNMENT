Area code 250

From Wikipedia, the free encyclopedia

Area code 250 is an area code for most of the Canadian province of British Columbia, including Vancouver Island. It was created on October 19, 1996 as a split of Area code 604. [1]

The incumbent local exchange carriers are Telus, Northwestel and the City of Prince Rupert.

[edit] Relief Planning

The projected exhaust date of area code 250 is January 2008. The plan recommended by the Canadian Numbering Administrator is to change the boundaries of area code 778 to include the entire province. [2]

As this solution is an overlay, the current telephone numbers in area code 250 would remain unchanged and new telephone numbers in the region could be assigned area code 778.
