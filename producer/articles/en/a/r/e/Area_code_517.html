Area code 517

From Wikipedia, the free encyclopedia

Area code 517 is the area code which serves the south central portion of the Lower Peninsula of Michigan. Notable cities within 517 include Lansing, Jackson, Howell, Coldwater, and Hillsdale. Previously, 517 also encompassed the places now served by Area code 989.
