North American area code 714 is a California telephone area code which was split from area code 213 as a flash-cut sometime during 1951. It encompasses northern and eastern Orange County, California and a small sliver of Los Angeles County, the City of La Mirada and portions of Chino Hills.

The 714 area code once included most of Southern California, generally south and east of Los Angeles, extending to the Arizona and Nevada state lines to the east, and south as far as the Mexican border (what is essentially area code 760 now). It experienced several splits including moving the San Diego area into area code 619 on January 1, 1982. It was again split, moving eastern Los Angeles and San Bernardino counties into area code 909 on November 14, 1992. The southern and western cities (including most the more affluent municipalities along the southern coast of Orange County) were again split from 714 creating area code 949 on April 18, 1998.

714 now covers Orange County (except for La Habra and Seal Beach and the western portion of La Palma, which have always been in the same area code as Long Beachâ€”currently the 562 area code). This is probably because at the time those splits first occurred, while most cities in Southern California were provided primary local telephone service from what was then Pacific Bell (now AT&T), La Habra and Seal Beach were served by GTE (now Verizon) which is the primary telephone provider for Long Beach. The 714 area code also contains the Carbon Canyon area of Chino Hills which is in San Bernardino County.

Today, four cities remain in both 714 and 949. Costa Mesa, Irvine, Santa Ana, and Tustin
