Area code 402 is the original telephone numbering plan code for Nebraska in the North American Numbering Plan. In 1954, this area code was split: eastern Nebraska retained 402, and western Nebraska was placed in area code 308. The roughly U-shaped 308/402 line generally runs (from south to north): northeast along the Platte River and then northwest along the eastern edge of the Sandhills.
