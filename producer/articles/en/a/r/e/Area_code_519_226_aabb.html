Area code 519 is an area code that covers almost all of southwestern Ontario and was introduced in 1953; portions of area code 416 and area code 613 were combined to form the new code area. In turn, in 1957, parts of 519 and 613 formed 705. It is bounded by area code 905 in the east except for Simcoe County which is bordered by area code 705 in the northeast. It was overlaid with the new area code 226 on October 21, 2006, at which time ten-digit dialing became mandatory. Area code 226 is Canada's 25th geographical area code (not including area code 600).

The incumbent local exchange carriers in 519/226 are Bell Canada, Amtelecom and several independent companies that cover vast tracts of rural Ontario.

[edit] History

The 226 area code was first proposed as a result of an NPA exhaustion study performed in the 1990s. The issue was raised with the CRTC by telecommunications firms and SAIC Canada, and in 1992 a relief plan was approved (Telecom Decision CRTC 2002-25). Although it primarily accepted the industry proposal, the CRTC required a permissive dialling period of four months, with consistent network announcements to consumers, instead of the proposed two month period. On 15 April 2003, the committee recommended delaying the introduction of the 226 area code from 19 February 2005 to 21 October 2006, primarily as a result of updated forecasts of the 519 NPA exhaustion projections.

Area code 519 covers a wide area (over 250 km) with no one city controlling a majority of the available numbers. In theory, therefore, the 226 overlay area code could have been introduced and fully utilised without breaking seven-digit dialing in any of the NPA519 communities. This would however have required exchange numbers to be issued in a specific pattern where (for instance) if 519-999-9999 were a Windsor number, the same number in the new area code, 226-999-9999, be issued in London, Guelph or some other community which is not and will not become a local call to any point in the Windsor area.
