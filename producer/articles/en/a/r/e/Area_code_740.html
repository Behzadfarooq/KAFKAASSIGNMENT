Area code 740 is a telephone area code serving all of central and southeastern Ohio except Columbus, which, with its nearby suburbs, retains the 614 area code. It covers the cities of Athens, Chillicothe, Delaware, Lancaster, Marietta, Marion, Newark, Portsmouth, Steubenville, and Zanesville. Area code 740 split from area code 614 in 1998.
