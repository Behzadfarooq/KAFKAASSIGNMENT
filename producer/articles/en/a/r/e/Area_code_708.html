Area code 708

From Wikipedia, the free encyclopedia

You have new messages (last change).

Area code 708 covers western and southern Cook County and eastern and southern Will County in Illinois. It was split off from area code 312 in 1989, and once covered all of Chicagoland until 1996 when area code 847 and area code 630 were broken off.
