Area code 203 is a telephone area code that covers the southwestern part of Connecticut. The area code stretches from the New York border past the city of New Haven and ends just south of Hartford.

[edit] Proposed area code overlay

The CT Public Utility commission has proposed an area code overlay for the 203 area code. The new area code assigned is 475. A date for implementing the new overlay area code has yet to be assigned.
