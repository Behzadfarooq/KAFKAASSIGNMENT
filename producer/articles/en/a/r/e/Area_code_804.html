Area code 804

From Wikipedia, the free encyclopedia

North American telephone area code 804 was split from area code 703 on Sunday, 24 June 1973, as a flash-cut, with no official permissive dialing period. It was itself split on July 1, 1997 to create area code 757, and again split on June 1, 2001, creating area code 434.

Area code 804 includes the following communities: Chester, Hopewell, Mechanicsville, Petersburg, Richmond, Tuckahoe, and West Point, and regions in eastern Central Virginia the Northern Neck, and the Middle Peninsula. The two southernmost counties on the Middle Peninsula are part of the Hampton Roads metropolitan area, but use 804 instead of the 757 used by the rest of Hampton Roads.
