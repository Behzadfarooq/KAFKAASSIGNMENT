Ares Galaxy is an open source peer-to-peer file sharing program for Microsoft Windows. It is written in Delphi. Proponents for the software claim that it is able to download quickly, has a better and more complete search function than other file-sharing programs, and connects quickly.

[edit] The Ares Network

Ares development began in the middle of 2002 and was originally operating on the Gnutella network. Six months later, it switched to its own network comprising of the leaves-and-supernodes p2p architecture. Its protocol is more difficult to identify than other popular P2P programs. As a result Ares is sometimes the only P2P client that functions on restricted networks, such as some university campuses. However, it is not impossible to block, and many organizations are now doing so. From version 1.9.0, data sharing was enabled between two peers behind a firewall. This may be due to mediating peers.[1] The Ares network is largely free from false and corrupt files unlike others like FastTrack. However, in past versions, some users report that reliability of and difficulty in establishing connections can be erratic - this seems to be fixed in the current versions. From version 1.9.4, Ares included support for the BitTorrent protocol.

[edit] Hash links

Ares has "hashlinks" functionality. Ares is able to search for peers with files pertaining to a hash and download from them. Some sites have begun to post verified Ares hashlinks. Some hashlinks can be decrypted and encrypted online. A website that supports Chat Channel hashlink encryption and Chat channel hashlink decryption can be found on http://www.aresunlimited.net.

[edit] SHOUTcast

From version 1.9.9, Ares Galaxy has an experimental support for the SHOUTcast internet radio

[edit] Standalone chat software

There has been a few releases of third party programs which can act, solely, as a chat client or server on the Ares network. The most famous of these third party chat programs is thought to be Arca Eclipse, an independent Ares Chat Server that doesn't require the original Ares at all. Ares chat servers are thought to have a massive amount of Remote Admins. These remote admins connect to the built in ARCA protocol. This allows users to have moderators other than themselves in the Ares chat room. But as of version 2.0.0, the ARCA protocol is no longer supported leaving all current remote admins rendered useless. This is because of a total rewrite of the ares chat server, with its own built in admin system it can almost manage itself. Although there are still some chat servers out there like Arca Eclipse, some are also being developed. Mars is being developed by HollowLife1987.

There have also been a lot of stand alone chat clients coming out recently. It has been reported that Josh of Ares Unlimited http://www.aresunlimited.net is 3/4 of the way done working on Ares Portable Chat Client [APCC]. A feature rich ares chat client that has a built in channel bot, and gets the channel list, and a lot more. Ryan is reported to still be making Aura, which is a chat client and chat server in one.

[edit] Alternatives

The following stand-alone alternative clients are known to connect with the Ares network:

  • Warez P2P is an adware version of Ares Galaxy with a few minor extras, but there is a Pro version which has even more minor extras and advertising removed.
  • KCeasy connects to the Ares network, in addition to OpenFT and Gnutella.
  • FileCroc connects to the Ares network and the BitTorrent network.
  • Poisoned, a Mac OS X client.
  • Ares Gold, an adware version of Ares, but regarded as being much faster.[citation needed]

In addition, a giFT-Ares plug-in for giFT has been released to provide access the Ares network. However, there has been problems with the plugin regarding certain issues like connecting to the network.
