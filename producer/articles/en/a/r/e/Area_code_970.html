Area code 970

From Wikipedia, the free encyclopedia

The map to the right is clickable; click on an area code to go to the page for that code.

Area code 970 is a telephone area code serving the state of Colorado. It covers Aspen, Durango, Grand Junction, Fort Collins and northern and western Colorado.
