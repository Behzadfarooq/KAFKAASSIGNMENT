Area code 815 is a telephone area code for the northwestern corner and southern Chicagoland suburbs of the state of Illinois. An overlay area code 779 was implemented on March 17 of 2007. A month earlier, on February 17, 1+ten-digits were required to be dialed for all calls in and out of area code 815--in preparation for the 779 overlay.
