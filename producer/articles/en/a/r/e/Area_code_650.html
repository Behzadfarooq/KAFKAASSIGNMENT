Area code 650

From Wikipedia, the free encyclopedia

Map of California area codes in blue (and border states) with 650 in red

North American area code 650 is a California telephone area code which includes most of San Mateo County (except the northernmost portion) and the extreme northwestern portion of Santa Clara County including Palo Alto, Mountain View, and Los Altos.

The 650 area was created by moving prefixes out of the 415 area code. A six month "permissive dialing" period where either area code would work began August 2, 1997. Mandatory dialing of the new area code started on January 31, 1998.
