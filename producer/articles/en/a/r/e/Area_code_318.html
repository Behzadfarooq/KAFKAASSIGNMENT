Area code 318

From Wikipedia, the free encyclopedia

Area code 318 is a telephone area code that covers the central and northern parts of Louisiana. The area code was split off from the original 504 area code, which had covered the entire state previously.

Originally, 318 stretched from the Gulf of Mexico north to the border with Arkansas. However the southern portions of the 318 area code have been split off into separate area codes in recent years.

[edit] Cities in the 318 Area Code

  • Shreveport, Louisiana
  • Monroe, Louisiana
  • Alexandria, Louisiana
  • Fisher, Louisiana
  • Tallulah, Louisiana
  • Rosefield, Louisiana
  • Bossier City, Louisiana
