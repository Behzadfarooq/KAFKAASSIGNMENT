Area code 561 is an area code in Florida, which went into effect on May 13, 1996. It is assigned to Palm Beach County. It used to be part of Area code 407 and prior to that, Area code 305. The northern part of 561 became Area code 772.

The mnemonic, "five from six leaves one" helps one to remember this area code.
