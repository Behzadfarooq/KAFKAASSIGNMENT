Area code 613 is an area code for Ottawa and surrounding Southeastern Ontario. It used to bound 514 before 450 was created, and before 1993, area code 416 before changing to 905.

Ten-digit dialing became mandatory on October 21, 2006. The Ottawa area has local dialing capability with the Gatineau region in area code 819. The Ottawa-Gatineau metropolitan area formerly had "exchange protection", meaning that an exchange assigned in Ottawa could not also be assigned in Gatineau, and vice versa. With the introduction of 10-digit dialing, this protection will be discontinued, although the local dialing area will be maintained.

It is expected that by 2012, a second area code will be added in the region as the area code exhausts; it will most likely be in the form of an area code overlay, but it has not been determined.

The main incumbent local exchange carrier in 613 is Bell Canada, although there are at least two independent companies serving rural exchanges.
