Area code 707

From Wikipedia, the free encyclopedia

Map of California area codes in blue (and border states) with 707 in red

North American area code 707 is a California telephone area code which covers Napa, Vallejo, Benicia, Fairfield, Santa Rosa, Fort Bragg, Crescent City, Eureka, Ukiah, and northwestern California.
