Area code 609

From Wikipedia, the free encyclopedia

Area code 609 once covered the entire South Jersey region and reached up into the north. Now, it covers only the central part of the state, and also the entire southern part of the 'Jersey Shore'. This area code was split in 1997 and West Jersey used the area code 856. 609 is still used for cell phones all over the state, and many people who live in the 856 area have a cell phone in the area code 609. When ordering a new cell phone, customers are given the option between area codes.
