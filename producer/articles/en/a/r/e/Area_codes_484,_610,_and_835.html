Area codes 484, 610, and 835

Area codes 484, 610 and 835 are Commonwealth of Pennsylvania telephone area codes which serve the Southeast region of the state including the cities of Allentown and Bethlehem. Area code 610 was split from area code 215 on January 8, 1994. Overlay code 484 was created June 5, 1999, and overlay code 835 is pending but has not yet been established.
