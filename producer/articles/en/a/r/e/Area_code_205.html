Area code 205

From Wikipedia, the free encyclopedia

Area code 205 is a telephone numbering plan code covering central and western Alabama including the cities of Birmingham and Tuscaloosa. Until 1995, this area code served the entire state of Alabama.

[edit] Cities located in Area code 205

  • Birmingham
  • Tuscaloosa
  • Hoover
  • Northport
  • Jasper
  • Oneonta
  • Clanton

[edit] Trivia

  • All of the telephone exchanges in area code 205 went down on January 26, 1983 following the death of legendary University of Alabama football coach Paul "Bear" Bryant from a heart attack in Tuscaloosa.
  • 2003 American Idol winner Ruben Studdard, a native of Birmingham, often performs wearing shirts with "205" emblazoned on the front.
