Area code 207 is the State of Maine's sole area code, except for a tiny number portion which uses the Quebec area code 418. Area code 207 was created as one of the original area codes in 1947. The small hamlet of Escourt Station uses Quebec area code 418 rather than Maine area code 207.

North: 418, 506
West: 603, 819 area code 207 East: 506
South: Atlantic Ocean
Province of New Brunswick Area code: 506
