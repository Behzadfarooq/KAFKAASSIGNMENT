Area codes 604 and 778

Area code 604 is a telephone area code encompassing the Lower Mainland, Sunshine Coast, Howe Sound and lower Fraser Valley regions of Southwestern British Columbia, Canada including the City of Vancouver. Initially 604 was the area code for the entire province until a geographic split was implemented in 1997 in which the remainder of the province received the new area code: 250.

A portion of the region within area code 604 has an additional area code implemented as a concentrated overlay: 778. Area code 778 was announced in NANPA planning letter PL-246 with an effective date of November 3, 2001. Area code 778 encompasses the Greater Vancouver Regional District as well as the Abbotsford and Mission exchanges. The remainder of the region is served only by area code 604. The addition of area code 778 required the implementation of 10-digit dialing within both the 604/778 concentrated overlay and the remainder of the 604 region.

The incumbent local exchange carrier in 604/778 is Telus.
