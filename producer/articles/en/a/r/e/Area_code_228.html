Area code 228

From Wikipedia, the free encyclopedia

Area code 228 is the telephone area code for the Mississippi Gulf Coast. It was the second Mississippi area code, created in 1997 when it was split off from the original area code 601, which covered the entire state.

Since then, the 601 area code was further reduced in size when area code 662 was created in 1999 in the northern half of Mississippi.
