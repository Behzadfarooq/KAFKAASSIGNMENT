Area code 418 is a telephone area code in the Canadian province of Quebec, encompassing the eastern portion of the province. When created in 1947, its boundaries were depicted as generally the eastern half of Quebec, but in 1957, northwestern Quebec was added; the area code attained its present boundaries in the late 1970s. It is expected that by 2015 area code 418 will become exhausted and a second area code will be needed, most likely in the form of an area code split, with Quebec City keeping the existing 418 area code.

Communities served by the 418 area code include Quebec City, Saguenay, Rivière-du-Loup and the Gaspé Peninsula and Côte-Nord regions. 418 also serves a tiny hamlet in the US state of Maine.

The incumbent local exchange carriers in 418 are Bell Canada and Telus (formerly Quebec-Telephone]], plus some independent companies.
