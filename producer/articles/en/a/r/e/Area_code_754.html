Area code 754 is the second area code for Broward County, Florida. In November 2000, the Florida Public Service Commission approved the use of an overlay area code to service Broward County, after a telecommunications industry group sought alternatives to relieve the strain on the county. It was projected that area code 954 was to be exhausted by third quarter of 2002. The North American Numbering Plan Administration announced on December 4, 2000 that number 754 was assigned to Broward County.[1] The area code was activated on August 1, 2001[2] and began use on April 1, 2002.[3] The second area code made ten-digit dialing mandatory.

Notable cities within Area code 954 are Fort Lauderdale, Sunrise, Hollywood, Pompano Beach, Coral Springs, and Pembroke Pines
