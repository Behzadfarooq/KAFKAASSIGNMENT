[edit] Documentation

Discussions -- usually held on talk pages -- often grow so lengthy that they must be archived to preserve their utility. There are several methods for doing this but all suggest that a link to the archive remain on the original page.

Insert {{arbox}} near the top of a page you wish to archive. This will generate a box that links to an archive subpage and transcludes its entire contents. Follow the link and, on the subpage, add links to all archived versions of the original page. Do not copy an entire archived version itself to this new subpage! The entire page will be transcluded, defeating the purpose of archiving.

[edit] Usage

{{Arbox}}

This template takes no parameter. It may be substituted or transcluded.

[edit] Example

Code Result
{{arbox}}
Archive:
  • [1] Dummy archive only
