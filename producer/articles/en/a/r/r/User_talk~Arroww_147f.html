[edit] Image copyright problem with Image:Kr20106mj.jpg

Thanks for uploading Image:Kr20106mj.jpg. However, the image may soon be deleted unless we can determine the copyright holder and copyright status. The Wikimedia Foundation is very careful about the images included in Wikipedia because of copyright law (see Wikipedia's Copyright policy).

The copyright holder is usually the creator, the creator's employer, or the last person who was transferred ownership rights. Copyright information on images on Wikipedia is signified using copyright templates. The three basic license types on Wikipedia are open content, public domain, and fair use. Find the appropriate template in Wikipedia:Image copyright tags and place it on the image page like this: {{TemplateName}}.

Please signify the copyright information on any other images you have uploaded or will upload. Remember that images without this important information can be deleted by an administrator. You can get help on image copyright tagging from Wikipedia talk:Image copyright tags. -- Carnildo 09:05, 1 February 2006 (UTC)

[edit] Infobox Automobile on Peugeot 309

I see you're having a lot of trouble trying to insert an image into the Infobox Automobile template.

First of all, an image is added by using the syntax [[Image:Imagename.ext]]. Additional parameters can be specified, such as the image's size, position (left/right), and description/alternative text. The syntax you were (unsuccessfully) trying to use ([[Imagename.ext]]) is only to be used when linking to the article named "Imagename.ext".

In addition, I'd like to inform you that if you're having trouble with an Infobox, you may need to check the description/talk page thereof. In your case, the relative talk page contains a detailed example of syntax which you should have looked up when you needed it.

Thank you for your time. --Emc² (contact me) 15:42, 13 January 2007 (UTC)
