Abyss Web Server is a web server for Microsoft Windows, Linux, Mac OS X and FreeBSD developed by Aprelium Technologies. It is claimed that it has low usage of system resources and stability.

[edit] Releases

Abyss Web Server is distributed as two different forms: X1 and X2.

[edit] X1

X1 is freeware with features which include:

  • HTTP 0.9, 1.0 and 1.1 support
  • CGI-Scripting (Scripting such as PHP, ASP, ASP.NET, Python etc.)
  • Fast-CGI
  • Custom Headers
  • Built-in password protection
  • Statistics
  • Bandwidth throttling (Per IP-address or in total)
  • IP-restrictions
  • Remote/local administration (GUI)
  • Anti-hack protection
  • Anti-Leeching
  • Custom Error Pages (404, 403, 401, etc.)
  • URL Rewriting
  • No nag-screens, no limitations
  • And more ... (See external links for other features)

[edit] X2

X2 is a commercial version of the webserver, providing the same functionality as X1, but also several enhancements:

  • Virtual hosting (Multiple hosts/domains on one server)
  • Extended bandwidth control, allows the user to set a limit on a specific file or path
  • Faster support
  • Update protection (first year free, then $24.95/year or $39.95/two years)
  • Upfront Costs: $59.95
  • Compare both versions
