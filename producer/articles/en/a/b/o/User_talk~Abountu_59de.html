anyone knows how can I use the same account in different languages? I had to register in another language the same account... i see that it's odd.

I need to use one image that is already on wikipedia in another article how can I do that cleanly? Abountu 00:58, 9 September 2006 (UTC)

another question, how to I propose merging two articles?

Unfortunately, the current MediaWiki system does not support universal accounts. However this support is in the works, meaning that they are working on making an account on Wikipedia able to work on any language's Wikipedia.
As to how to use images in articles, find the file name, then type [[Image:filename.jpg]]], where filename is the name of the image file on Wikipedia.
As far as proposing the merger of two articles, Wikipedia:Merging and moving pages contains all the information you need to know. --Willy No1lakersfan (Talk - Contribs) 01:22, 9 September 2006 (UTC)

[edit] How can I create a redirection ?

I was wondering how can I create a redirection, so when people search for a word (aaa) they'll fall on an article that is named (bbb)

Create an article, and put #redirect [[article name here]] in the edit box. Alternatively, you can use the button on the toolbar that has #R on it, and replace the text with the article name. J Ditalk 21:37, 13 September 2006 (UTC)

That's the way. Thanks for the note and whatever you do, always feel free to ask me for help. - Lucky 6.9 22:26, 13 September 2006 (UTC)
