[edit] PPS Information Services/PPSnet abuse - an idea

To anyone investigating/concerned about this abuse: I have an idea on one possible solution to this problem; I don't know if it will work. As every student and staff member that uses a school district computer has their own user account, it might be possible for a school administrator to block wikipedia - completely - on for some school-account users. For example, say the student with the school account "xy123456" has been verified as vandalizing wikipedia, not logged in to wikipedia, and doing so from the PPS/IS IP address 159.191.12.24. Instead of blocking that IP, which would affect thousands of students, a PPS admin could block wikipedia on that user's account. Type in www.wikipedia.org and they'd get something saying that page was unavailable. Although it would take slightly more work on both sides, and probably tick off some students, it could solve some problems. Of course, I also don't know if this is possible with our current networking software. It would be tragic if wikipedia had to block all PPS students and staff, just because some people think it's funny to vandalize. 50% of the students at my middle school, if not more, used wikipedia for research; some teachers even recommended it. Lets just hope this problem gets fixed...  :( --Kormerant 22:50, 16 July 2006 (UTC)
