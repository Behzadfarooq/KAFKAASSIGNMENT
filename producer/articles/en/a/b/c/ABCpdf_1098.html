ABCpdf
Developer: WebSupergoo
Latest release: 6.0 / January 2007
OS: Microsoft Windows
Use: PDF Library
License: Proprietary
Website: www.websupergoo.com

ABCpdf is a proprietary Portable Document Format (PDF) library developed by WebSupergoo.

ABCpdf is available in both .NET and COM formats.

It includes functionality for PDF creation, manipulation, rendering and printing.

It allows core PDF functionality like adding text and graphics, appending, reorganizing and annotating PDF documents.

It supports the import and export of image media in a variety of formats (for example, JPEG, TIFF, JPEG 2000, Encapsulated PostScript, Scalable Vector Graphics) and color spaces (for example, RGB, CMYK, Grayscale).

It allows the conversion of HTML to PDF using built in conversion routines. It allows the import of Microsoft Office Formats like Microsoft Word and Microsoft Excel via OpenOffice.org.
