Thanks for uploading Image:Cruzeiro Fans.jpg. Wikipedia gets hundreds of images uploaded every day, and in order to verify that the images can be legally used on Wikipedia, the source and copyright status must be indicated. Images need to have an image tag applied to the image description page indicating the copyright status of the image. This uniform and easy-to-understand method of indicating the license status allows potential re-users of the images to know what they are allowed to do with the images.

[edit] Image

Hi Abreuzinho,

Thanks for asking me about image copyright tags. If a website releases all rights and allows a picture to be used anywhere, then you should leave the licensing selection as "None selected," but add the tag {{No rights reserved}} or {{PD-release}} to the end of the image description. Only use PD-release if the website explicitly says that it is dedicated to the public domain or that there is no copyright, otherwise use No rights reserved. Also, make sure you include links to the page where you got the image and the page with the copyright information. This is very important if you did not create the images yourself.
