AOLserver Dynamic Pages

In computer science, AOLserver Dynamic Pages (or ADP) is the name of the server-side technology employed by AOLserver to dynamically create web pages. The web developer exploits ADP by embedding ADP-extended Tcl commands in HTML source code. When the page is called for by the user agent, the web server executes the embedded Tcl commands in the HTML source, generating a custom web page.
