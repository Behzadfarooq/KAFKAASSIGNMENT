AOL OpenRide
Developer: AOL LLC
Latest release: 1.5
OS: Windows
Use: Web browser
License: Proprietary
Website: aol.com


AOL OpenRide is an Internet application suite made by AOL that combines e-mail, instant-messaging, a web browser and a media player in one window. Anyone can download it for free, but to make maximum use of the software you need to sign in with an AOL or AIM screenname.

The People pane will enable users to sign into the AOL Instant Messenger service to display the user's buddy list and ongoing conversations. The E-mail pane is a standard email client, and is not just restricted to AOL email, but allows users to add email accounts from other providers. The browser pane utilizes AOL Explorer, which uses the Trident rendering engine but 'enhances' it with tabbed browsing and other features. The media pane provides access to AOL's collection of videos, the Radio@AOL service, and features a media player for files on the user's local hard drive.

Currently, the AOL Beta offices are overseeing its development, similar to the way that they are doing AOL 9.0 VR, the version of AOL that is said to be the only version to support Windows Vista. The AwayView Screensaver is currently only on AOL Beta Central and is said to be included with a special version of AOL OpenRide, known as AOL OpenRide with AwayView.
