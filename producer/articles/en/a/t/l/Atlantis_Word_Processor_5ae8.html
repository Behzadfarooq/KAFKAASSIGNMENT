[edit] Features

[edit] Standard Features

Among the standard features of Atlantis Word Processor you will find bookmarks and hyperlinks, sections, newspaper columns, headers and footers, footnotes and endnotes, fields, styles, bulleted and numbered lists.

Atlantis has a Backup Files feature. When this feature is enabled, Atlantis creates automatic and regular backups of all open documents. Successive versions of each document are saved to a special folder. In this way, you will always be able to go back to earlier versions of any document, for example when you made undesirable changes to a document, and saved it.

Atlantis features a typing assist - Power Type. Working in close cooperation with the Atlantis spellcheckers, Power Type collects statistical information about text that you type, creating an ordered database of the words that you use most. When you start typing the same words again, a small window pops up with a warning sound, displaying a list of words beginning with the same characters. Very often, you will only have to press Enter to insert the suggestion highlighted by Power Type. In other instances, you will need to select the appropriate word from the list first, then press Enter to insert the highlighted word. If you are a beginner or slow typist, the Atlantis Power Type will help you type documents in less time with less effort. But Power Type will also prove useful to advanced users: Power Type can be configured to suggest not only individual words but entire collocations and phrases. Users can register custom sets of phrases and collocations with Power Type. The Power Type suggestions will be all the more appropriate.

[edit] Printing

Atlantis also offers a whole array of advanced printing capabilities. Documents can be printed with a specific zooming factor even when the document page size and the actual paper format do not match. Multiple-page documents can be printed on one single sheet of paper. Documents can be printed in two-sided mode on any printer, even on printers which do not have such built-in capability. This will save paper and print more compact versions of your documents. And you can even create and print various types of booklets, including pocket-size booklets.

[edit] Spellcheck and Auto Correct

Of course, Atlantis has the advanced spellchecker. It will detect and report any potential misspellings in your documents. Spellchecking can be run either at your request after typing, or, most conveniently, As-You-Type. The Atlantis AutoCorrect feature will automatically take care of punctuation signs, typical misspellings, common syntax mistakes. Finally, Atlantis has tools for fully automatic or semi-automatic hyphenation of the document end-of-lines.

[edit] Privacy

Any confidential information, whether private or professional that you might have, can be securely protected using the encryption abilities of Atlantis. Atlantis uses 256-bit keys to create documents in a proprietary encrypted COD file format.

[edit] Shareware

Atlantis Word Processor is shareware (ie "try-before-you-buy"). You can install Atlantis to your computer, and evaluate it for up to 30 days for free.
