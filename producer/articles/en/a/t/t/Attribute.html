The word attribute can refer to:

  • An attribute in philosophy, an abstraction of a characteristic of an entity or substance
  • An attribute in art, an object that identifies a figure, most commonly referring to objects held by saints - see emblem
  • An attribute in social sciences, a characteristic of a variable
  • An attribute, a deity's aspect; see Negative theology

In computing:

  • Attribute (computing), a factor of an object or other kind of entity
  • Attribute (network management), a property of a managed object that has a value
  • An attribute, a property inherent in a database entity or associated with that entity for database purposes
  • An attribute, a parameter of an element in SGML-based markup languages like HTML and XML

In gaming:

  • Attribute (role-playing games), a type of statistic for a fictional character
  • Attribute (game), a board game
