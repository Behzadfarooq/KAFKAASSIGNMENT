[edit] Categorize Tagging

Hello. First I would like to apoligize if I offended you by tagging your article with the categorize template. However it is a cornerstone of new page patrolling and can be very helpful. It is common for authors to simply forget to add a category and tagging it will remind. In addition if the author never returns to the page it will put it into the category so others can find it and add categories later. At any rate if I caused you any grief I am sorry.
