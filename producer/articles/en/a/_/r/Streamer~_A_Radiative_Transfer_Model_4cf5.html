Streamer: A Radiative Transfer Model

Streamer is a radiative transfer code (Key and Schweiger, 1988) to calculate radiances (intensities) or irradiances in the atmosphere.

The code uses N-strem approximation to the radiative transfer equations (Stamnes et al. 1988) and allows for flexible choice of bands. The code can be used both for satellite radiance applications and estimates of heating rates in both cloudy and non-cloudy atmosphere. One can specify surface reflectivity. Streamer is written in FORTRAN.

[edit] FluxNet: A Neural Network Version of Streamer

FluxNet, the neural network version of Streamer, calculates upwelling and downwelling surface flux in either shortwave or longwave. It is less flexible than Streamer but is two to four times faster.
