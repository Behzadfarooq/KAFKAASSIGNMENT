The "proof" includes a step where

A(A-A) = (A+A)(A-A)

is reduced to

A = A + A

This is done by dividing both sides of the equation by (A-A). The catch is that A - A = 0 - if you subtract any number from itself, you get zero. This means that the step described above in fact consists of dividing both sides of the equation by zero, which is not a valid move (see division by zero for more information).

What the equation actually resolves to (as is probably easier to see when it is represented as A² - A² = A² - A²) is the completely uncontroversial

0 = 0

Does that help?
