[edit] Your question at the new contributors' help page

According to Wikipedia's style guide for external links, Wikipedia articles should never link to sites that illegally reproduce copyrighted material, so you should remove any links to such sites and leave a {{spam1}} template on the talk page of the user who added the link. If the user continues to add the link or similar illegal links, use {{spam2}}, then {{spam3}}, and finally {{spam4}}. If the user continues after four warnings, you should list him at Wikipedia:Administrator intervention against vandalism. Let us know if you have any more questions. Thanks. --TantalumTelluride 18:42, 29 June 2006 (UTC)

P.S. When leaving warning templates on user talk pages, you should always substitute them. For example, you would type {{subst:spam1}}, which would produce a message asking the user to stop adding links to commercial sites. For general vandalism, you should type {{subst:test1}}, which produces a message directing test edits to the sandbox. For more information about dealing with various types of vandalism, you can check out Wikipedia:Vandalism. --TantalumTelluride 18:53, 29 June 2006 (UTC)
