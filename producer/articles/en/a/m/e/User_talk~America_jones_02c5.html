(edit conflict)I agree that Wikipedia's conventions can be daunting for a beginner, but keep trying and ask for help -- there are lots of friendly folks willing to help. The reason I marked this as a possible copyright violation was because of the license that it is labeled under at flickr. Specifically, this license is not the same as this one which you labeled the image here. The problem with that is the the copyright holder specified that the license is Non-commercial and no derivative works are allowed, which is incompatible with the license you gave, and further not compatible with Wikipedia's goals. It could be possible to use this image under fair use because it is copyrighted, but the person would have to be dead (which it doesn't appear he is) or the image has to be unrepeatable, like the picture of the winning touchdown in the Superbowl, which this image also appears to just be describing him. When you search flickr.com in the future, you should go under the "advanced search" and click all three boxes under "Creative Commons", just above the search button. The images that come up in that search you can use on Wikipedia, and better yet, you can upload them to Commons! I did this search for "Amiri Baraka" and it came up empty, so it appears that there is nothing at flickr for Wikipedia. The next best option is to try and contact the subject and ask them for a free image. See Wikipedia:Requesting copyright permission for help with that. The next option is to search the internet, and if you find an image, see if it is free, and if it's not try and contact that user to see if they will release it free. I've actually had success with these options.
