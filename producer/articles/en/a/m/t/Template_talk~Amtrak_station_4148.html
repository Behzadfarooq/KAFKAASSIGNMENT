[edit] Using this template

Parameters:

  • name – Station name, or "[[Image:...]]<br/>Station name"
  • address – Street address<br/>City, State
  • lines – list of services
  • other – bus services, etc.
  • code – 3-letter station code
  • owned – station owner name
  • lat, long – latitude and longitude params, see {{geolinks-US-streetscale-neutered}}
