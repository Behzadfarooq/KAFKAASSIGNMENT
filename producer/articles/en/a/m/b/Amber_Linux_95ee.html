Amber Linux

From Wikipedia, the free encyclopedia

The official logo of Amber Линук

Amber Linux is a Latvian Linux distribution based on Debian. It aims at being the first business desktop Linux distribution that is tailored specifically to the needs of Latvian users.

Latest version: 2.0 released on December 2004.

Some features:

  • Automatic hardware detection and storage device mounting.
  • GNOME as the default desktop environment.
  • OpenOffice.org as the default office suite.
  • Epiphany, X-Chat IRC client, Gaim Instant Messaging client, Gftp FTP client
  • Totem video player
  • WINE Windows compatibility layer
  • GIMP image editor
  • Hansa Financials accounting software (on demand)
  • Amber Linux Control Center software
  • Automatic updates
