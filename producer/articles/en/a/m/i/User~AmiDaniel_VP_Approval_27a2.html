This page should only be used to request permission to use VandalProof. If you would like to report a bug, make a suggestion, or otherwise discuss the program, please go to one of the pages listed above.

If you would like to use VandalProof, please add your username to the bottom of the Awaiting Approval section. Moderators will approve a user based upon sufficient evidence that he or she intends to use the tool in a constructive manner. While in no way a strict rule, new users with fewer than 250 mainspace edits, users who have recently been involved in edit wars, and users with a history of vandalism will generally not be accepted.

[edit] Awaiting Approval

If you would like to use VandalProof, please add your name to this list. As the list is not moderated by hand, please be sure to use the following format: * {{user2|YOUR_USERNAME}}, and replace any spaces with underscores (_). If your name is removed from the list and you do not receive confirmation that you have / have not been approved, please resubmit your name.
