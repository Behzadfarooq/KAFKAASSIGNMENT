VandalProof 1.36 coming soon...

VandalProof 1.36 is coming soon! This release will feature the following updates and fixes:

  • The 'Notify Contributors' feature is fixed, so if you ever wanted to make use of that but couldn't, you'll soon be able to.
  • Slight speed improvements in the retrieval of the RSS feed should mean fewer crashes and lock-ups.
  • The RC filter is working much more smoothly now, so you'll get WAY fewer false positives and might even be able to actually take notice of it *shock*.
  • The custom rollback feature has been totally revamped, so look forward to all the new warnings as well as several that were missing.

And a number of other behind-the-scenes changes as well. Version 1.36 should be with us in a few days.

What's New - 1.3

There are some exciting new features now present in this version

  • RSS based Recent Changes: This adds the ability to
  1. show the amount of text added or removed from articles (page blanking)
  2. show any watched words present on the Bad word list
  3. give different user groups/edits different colors (ie Blacklisted users show in red, whilelisted green etc). These are fully customizable
  • Tabbed based browsing: Simply hold Ctrl when clicking a link and it will open in a new tab (up to five can be displayed)
  • Display incoming links: One click shows all incoming links of a page split by their respective wikispaces
  • Article log: New function shows all log entries for a user/article
