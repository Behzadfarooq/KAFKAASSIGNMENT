AMiLDA

AMiLDA Linux is a Linux distribution suitable for embedded systems, especially routers. Currently it runs on the ADM5120 chipset. There are other distributions for this chipset, such as midge.

[edit] Features

  • Stateful firewall
  • DHCP server support
  • Support for connections using static IP, dynamic IP, and PPPoE (although this is quite experimental).
  • Expandable
  • Web configuration interface

[edit] Benefits

The distribution uses a System V initialization scheme. For the advanced users, shell access is provided so they can tweak the system.

Development is open to anyone, with a section for contributed hacks.
