[edit] Decline users button

Hi Daniel

I don't know if you have seen the decline users template I quickly wrote (apologies if you've already done one! - and please feel free to amend it here) but I was wondering if I could add it to my custom buttons... problem is I would need it to pop up with a "reason" box (see the main mod page for the format). So essentially I'd add something like


<TALK 1>
{{DeclineVP}}
{{subst:DeclineVP|{{{1}}} }}
VandalProof decline

Where {{1}} (or whatever needs to go there) is the reason - so it matches up with Template:DeclineVP

Any thoughts? - Glen TC (Stollery) 08:36, 16 April 2006 (UTC)

Hopefully one of these days I'll get around to actually creating a help file... lol. To add it to your buttons, simply add:

<TALK 1>
{{DeclineVP}}
{{subst:DeclineVP|%P}}
Notify user that his/her request to use VP was not granted

(Hope you don't mind that I changed your edit summary; feel free to change it to whatever you want). The %P will cause VandalProof to prompt you for an article name to replace it. AmiDaniel (Talk) 08:42, 16 April 2006 (UTC)
