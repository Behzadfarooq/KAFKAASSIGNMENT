Version 1.3.1

I sincerely, sincerely apologize to all you VandalProof users out there for my absense from Wikipedia over the past few months, and I understand that many of you are having problems launching VandalProof at all. This problem is caused by the sheer size of the userlist (currently >900 users), which the former auth system was not designed to handle. Therefore, I'm going to do my absolute best to get VP v1.31 (which uses a different login system, along with many other nifty features) out as soon as possible. You can expect to see it available for download around Sunday, the 12th of Novemeber. Sorry for the delay!!! AmiDaniel (talk) 10:16, 7 November 2006 (UTC)

IRC Channel

VandalProof now has its own IRC channel, where you can find out all about the upcoming release, get help using the tool, report bugs, and make suggestions. Check it out at #vandalproof.

Version 1.2.2 Patch

Due to a couple of changes in the MediaWiki software, I've had to urgently release a minor revision to VandalProof. Please download the version 1.2.2 executable here. There's nothing to dramatically different in this release, as I simply loaded up the old v1.2.1 source code and repaired the following:

  • VandalProof now recognizes the &lmt; character recently added by the devs to each item in Special:Watchlist and Special:Recentchanges, which VP formerly rendered as a question mark, making these features useless as a result of the change.
  • The bugs with ampersands and forward slashes when reverting have been repaired.
  • The problem with terminal question marks in page titles has been repaired.

Common Problems

  1. Rollback is not working or VandalProof is not leaving edit summaries. Update to VandalProof v1.2 and all your problems go "poof"!
  2. Annoying error messages every time VP loads a page. Try loading your monobook.js within Internet Explorer and doing a hard refresh (Ctrl+F5). If that does not work, see Temporary Fix for Javascript Error Messages.(Should be fixed in v1.2 which can be downloaded now!)
  3. Other annoying problems not mentioned above Try looking in the new FAQ/Troubleshooting guide---If the solution is not there, than your problem is probably a bug, go to the Bug Report Page for more infomation, but before doing so, make sure you have the Latest version of VandalProof Installed

Known Bugs

  1. Annoying: though non-fatal Internet Explorer Script Messages. See Temporary Fix for Javascript Error Messages.
  2. Persistent crash on start-up. See Crash on Startup.
