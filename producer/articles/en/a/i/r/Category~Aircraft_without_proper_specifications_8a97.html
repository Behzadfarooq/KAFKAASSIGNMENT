 This aircraft article has not been updated to WikiProject aircraft's current standards. Please see this for more details.

If you have come here from an article, it is probably because you have come from an aircraft article that hasn't been updated to the specifications standards mandated by WikiProject aircraft. Current standards call for a template-based specifications section; older standards used table- or text-based specifications. Depending on the status of the article, there are a number of ways to perform the update. If the article is still in table format (it has a blue table in the upper left corner), you should substitute the template code by typing the following text directly above the External links (==External links==): if it is an American aircraft, type {{subst:airspec-imp}}; otherwise type {{subst:airspec-met}}. You can also copy and paste the text at #Template code, though this is less efficient.

If the article has already been converted, substituting the template code is still probably the best solution. In this case, substitute it at the beginning of the specifications section.

Once the template code has been added, you can copy and paste the information originally provided into the template. Some uncommon fields will be removed if you leave them blank. If you need help, please visit WT:Air, User talk:Ingoolemo, User talk:Ericg, or User talk:Emt147. Because of the complicated nature of the template, any bungles with the template will be forgiven.
