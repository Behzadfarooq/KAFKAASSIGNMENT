Template talk:Airnd

From Wikipedia, the free encyclopedia

This is a navigational template for military aircraft categories. 'ntd' stands for 'nation', 'type', and 'decade', the three characteristics used to sort aircraft categories that this template is applied to. Similar templates include Template:Airntd, Template:Airnt, Template:Airnd, Template:Milairnd, Template:Civairnd, Template:Milairtd, Template:Civairtd. It uses {{qif}} statements to deal with problematic nations such as Russia, which do not always have the same name in every decade. It is meant to templatise the category scheme at WP:Air/C.

[edit] How to use the template

It should be used only for [[Category:{{{nation}}} aircraft {{{1}}}0-{{{1}}}9]], never for {{{nation}}} military aircraft or {{{nation}}} civil aircraft; for these use Template:Milairnd and Template:Civairnd. You must specify the following parametres:

  • {{{nation}}}: Must be exactly the same as the adjective in the category title.
  • {{{1}}}: The first three digits of the associated decade; if the 1990s, use 199 for this parametre.
