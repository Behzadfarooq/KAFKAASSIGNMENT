This is an attempt to index edit summaries used on reverts. These are all genuine edit summaries I noticed on Recent Changes; the user mentioned is pretty much a matter of chance and who was online at the time. I have not checked any of these reverts to see if they were genuine and/or useful, just looked for revert-like summaries on Recent Changes pages by eye. I'm collecting this information to try to improve my edit counter. Edits marked 'rm' aren't counted, as that's a common abbreviation for non-revert removal of information. I may have made mistakes on this list. The reverts are written in copy-and-paste plain text, not in wikimarkup, and diffs are provided (for verfication and to provide GFDL history; it would be a bit silly to cause a copyvio by illegally copying an edit summary!)

[edit] Looking for reverts done by particular tools

These reverts were found by checking the contribs of the tool's creators, in most cases. (It's not entirely clear who came up with admin rollback!) For instance, User:AmiDaniel's contribs were checked to find a VandalProof revert. The tools are listed in alphabetical order. If you know of a vandal-fighting tool not listed here, feel free to add a sample revert-summary to the list. Most of these tools have a link in their edit summaries, and in these cases the link is added here to show which tool was used.
