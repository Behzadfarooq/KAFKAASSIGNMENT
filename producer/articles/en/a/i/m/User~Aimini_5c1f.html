aimini is an all-in-one peer-to-peer file sharing application which support the global position of PozID for the Internet, person to person, this program offers direct connect up in between users, File Direct-Transfer, peer-to-peer (P2P) file-sharing, making for faster transfers and searches of all types.

The aimini P2P software provides audio (voice) and video transmission of high quality for talks, you can also send customizing text messages and pictures, etc.

The current stable version is 1.8.0.1 as of May 5, 2006, [1].

[edit] Features

Aimini includes the following features which make it stand out (features which appear in most peer to peer clients are not listed):

  • Direct connect up in between users.
  • Searching and sharing resources use of the Peer to Peer.
  • File Direct-Transfer.
  • Talking by voice, video and rich text.
  • Built-in An Address Book.
  • Monitors multi-task status.
