Agar (software)

From Wikipedia, the free encyclopedia

Agar
Developer: CubeSoft Communications, Inc
Latest release: 1.1 / June 8, 2006
OS: Linux, Unix, Macintosh, Windows
Use: API
License: BSD (revised)
Website: http://agar.csoft.org/

Agar is a set of high-level computer graphics libraries used by many applications including 2D/3D computer games, visualization tools, graphical editors and simulations. Agar applications are portable to a variety of operating systems thanks to the Simple DirectMedia Layer (SDL) library. Agar is mainly known for Agar-GUI, a graphical user interface library which is compatible with both OpenGL and direct-video (SDL) displays.

Agar is written in the C programming language and is open source, released under a revised (two-clause) BSD license. The project has been in development since early 2002. The first production release of the core libraries was made in October 2005.

Libraries included in the official Agar distribution include:

  • Agar-Core (generic object persistence, memory management, dependencies and such).
  • Agar-GUI (graphical user interface library)
  • Agar-VG (vector graphics routines)
  • Agar-RG (pixmap-based graphics routines)

The Agar project also develops multiple libraries for use with Agar provided as separate distributions, such as:

  • Agar-SG (3D scene graph library)
  • Agar-SC (linear algebra and scientific computing routines)
  • Agar-Game (game level system)
