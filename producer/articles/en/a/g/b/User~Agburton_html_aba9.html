[edit] Hello world program

[edit] From Wikipedia, the free encyclopedia

(Redirected from <a href="/w/index.php?title=Hello_world&redirect=no" title="Hello world">Hello world</a>)
Jump to: <a href="#column-one">navigation</a>, <a href="#searchInput">search</a>

A "hello world" program is a <a href="/wiki/Computer_program" title="Computer program">software program</a> that prints out "Hello, world!" on a <a href="/wiki/Display_device" title="Display device">display device</a>. It is used in many introductory tutorials for teaching a <a href="/wiki/Programming_language" title="Programming language">programming language</a> and many students use it as their first programming experience in a language.

Such a program is typically one of the simplest programs possible in a computer language. Some, however, are surprisingly complex, especially in some <a href="/wiki/Graphical_user_interface" title="Graphical user interface">graphical user interface</a> (GUI) contexts; but most are very simple, especially those which rely heavily on a particular <a href="/wiki/Command_line_interpreter" title="Command line interpreter">command line interpreter ("shell")</a> to perform the actual output. In many <a href="/wiki/Embedded_system" title="Embedded system">embedded systems</a>, the text may be sent to a one or two-line <a href="/wiki/Liquid_crystal_display" title="Liquid crystal display">liquid crystal display</a> (LCD) (and in yet other systems, a simple <a href="/wiki/Light-emitting_diode" title="Light-emitting diode">LED</a> being turned on may substitute for "Hello world!").

A "hello world" program can be a useful <a href="/wiki/Sanity_test" title="Sanity test">sanity test</a> to make sure that a language's <a href="/wiki/Compiler" title="Compiler">compiler</a>, <a href="/wiki/Development_environment" title="Development environment">development environment</a>, and <a href="/wiki/Run-time_environment" title="Run-time environment">run-time environment</a> are correctly installed. Configuring a complete programming <a href="/wiki/Toolchain" title="Toolchain">toolchain</a> from scratch to the point where even trivial programs can be compiled and run may involve substantial amounts of work. For this reason, a simple program is used first when testing a new tool chain.

While small test programs existed since the development of programmable <a href="/wiki/Computer" title="Computer">computers</a>, the tradition of using the phrase "Hello world!" as the test message was influenced by an example program in the book <a href="/wiki/The_C_Programming_Language_%28book%29" title="The C Programming Language (book)">The C Programming Language</a>, by <a href="/wiki/Brian_Kernighan" title="Brian Kernighan">Brian Kernighan</a> and <a href="/wiki/Dennis_Ritchie" title="Dennis Ritchie">Dennis Ritchie</a>, published in <a href="/wiki/1978" title="1978">1978</a>. The example program from that book prints "hello, world" (i.e., no capital letters, no <a href="/wiki/Exclamation_sign" title="Exclamation sign">exclamation sign</a>; those entered the tradition later). The book had inherited the program from a <a href="/wiki/1974" title="1974">1974</a> <a href="/wiki/Bell_Labs" title="Bell Labs">Bell Laboratories</a> internal memorandum by Kernighan —Programming in C: A Tutorial— which shows the first known version of the program:

 main() {
    printf("hello, world");
 }

However, the first known instance of the usage of the words "hello" and "world" together in computer literature occurred in A Tutorial Introduction to the Language B, by Brian Kernighan, <a href="/wiki/1972" title="1972">1972</a> <a href="http://cm.bell-labs.com/cm/cs/who/dmr/bintro.html" class="external autonumber" title="http://cm.bell-labs.com/cm/cs/who/dmr/bintro.html">[1]</a>, with the following code:

 main( ) {
   extrn a, b, c;
   putchar(a); putchar(b); putchar(c); putchar('!*n');
 }
 a 'hell';
 b 'o, w';
 c 'orld';

There are many variations on the punctuation and casing of the phrase, and the examples on this page print out several of these variations. Variations include the presence or lack of the comma and exclamation mark, and the capitalisation of the 'H', both 'H' and 'W', or neither. The most commonly accepted variation, despite not being the original text, is "Hello, world!" Some languages are forced to implement different forms, such as "HELLO WORLD!," on systems that only support capital letters, while many <a href="/wiki/Hello_world_program_in_esoteric_languages" title="Hello world program in esoteric languages">"hello world" programs in esoteric languages</a> print out a slightly modified string. For example, the <a href="/wiki/Malbolge" title="Malbolge">Malbolge</a> program prints "HEllO WORld", this having been determined to be "good enough." "Hello world" programs also normally print a <a href="/wiki/Newline" title="Newline">newline</a> (<a href="/wiki/ASCII" title="ASCII">ASCII</a> character 10, character 13, or both).
