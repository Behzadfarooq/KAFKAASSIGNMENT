Alfresco is an open source, open standards, enterprise scale content management system that includes a modern content repository, an out-of-the-box web portal framework for managing and using content designed to work with standard portals, and a CIFS interface that provides Microsoft Windows file system compatibility. It is built by an experienced content management team that includes the co-founder of Documentum, and principal engineers from Documentum, Interwoven, Oracle and Vignette.[1]

The Alfresco product has a modular component architecture. The Alfresco system is developed using Java technology.

It was licensed under the terms of the Mozilla Public License (MPL). The Alfresco CMS recently switched to the GNU General Public License. Due to the switch, the company wanted to ensure that the code would stay open - or as Marketing-Manager Matt Asay said, "We want that the code is bigger than the company." For now the company is keeping the product under the "GPL2 only" version of the license. A change to the GPLv3, currently under development, is later possible, said Alfresco.[1]
