[edit] Link on Logo

You edited an external link in logo design article that is [www.logoblog.org]. Can you tell me what spamy thing did you find on this page. Its just a logo design resource portal.Aly pirani 05:40, 20 December 2006 (UTC) (copied from User_talk:Jkatzen)

If you look through the Talk page of the Logo article, you'll see that it, over time, gets swamped with so many links (spam-like and not) that any new link that might come up is probably going to need to undergo considerable discussion among the frequent editors. Basically, any link that doesn't provide considerably more value than what could be included in the article is presumptively (though not necessarily ultimately) inappropriate. Likewise, external sites that generally only link to other sites (link directories) will probably not pass muster. As far as the "blog" elements of the site, there are dozens, if not hundreds, of blogs on the Internet that cover similar topics, and in my opinion, if any were to be included, it would not be this one, as many of the others are written by established and published typographic experts and involve far more analysis than this one. If you don't think my opinion is valid (and maybe I'm wrong about removing the link), post a question on the discussion page there asking about your link and see what the response is. Jkatzen 07:00, 20 December 2006 (UTC)
I just looked over the Talk page there, and I see that you did post there first (thanks), and I apologize for not checking it first. That being said, no one's really said anything one way or another about it, which suggests that it's not catching traction, but who knows. Personally, I would wait a month or so from when you first post it to the Talk page to see if anyone responds. But, I'm not the God of the Logo page, so do as you like. If you add it again as a link now (or after waiting a bit), I won't remove it. If it doesn't get removed by anyone else, I guess it'd be fine, though I would be surprised if it lasts. I urge you, if you add it again, to carefully craft the description next to the link to explain to everyone what it really is and its value so that it doesn't appear to be a spamlink. Jkatzen 07:10, 20 December 2006 (UTC)
