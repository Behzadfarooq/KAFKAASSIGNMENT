Alamo Game Engine is a game engine developed by Petroglyph, which they have used to create the real-time strategy games Star Wars: Empire at War, and Star Wars: Empire at War: Forces of Corruption. A modified version of the Alamo Game Engine is being used for Petroglyph's third title, Universe at War: Earth Assault[1] .

[edit] Features

The Alamo Game Engine features 3D gameplay and a real time strategic view of a map on which the game is taking place.

[edit] Games using the Alamo Engine

  • Star Wars: Empire at War (2006), the first game to use the engine
    • Star Wars: Empire at War: Forces of Corruption (2006), expansion pack
  • Universe at War: Earth Assault (2007) [2]
