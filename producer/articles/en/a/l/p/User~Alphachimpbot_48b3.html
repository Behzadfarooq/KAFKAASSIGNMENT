About Me

This is a bot account regularly operated by Alphachimp. This bot runs using pywikipediabot and WP:AWB; contact me for any relevant bot files (I have uploaded those I think may be helpful). It is currently approved to perform the following functions (with bot flag):

  • Subst'ing user talk templates
    • tempsubst
  • Application of cleanup-by-month tags within the cleanup category
  • Moving userboxes to userspace.
  • Moving categories that have been renamed or will be deleted.
  • Correcting errors in links.
  • Addition of Wikiproject templates (e.g. {{Architecture}}) to talk pages
  • Implementation of merge-by-month categories. Check out my approval for more information.
    • mergefix

If you have any non-critical queries, leave a message at User talk:Alphachimp. In an emergency, simply block the bot.
This user account is a bot operated by Alphachimp (talk).

It is not a sock puppet, but rather an automated or semi-automated account for making repetitive edits that would be extremely tedious to do manually.
Administrators: if this bot is malfunctioning or causing harm, please block it.
