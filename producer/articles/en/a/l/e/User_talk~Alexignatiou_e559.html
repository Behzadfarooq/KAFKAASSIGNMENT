[edit] Helpme

Is it possible to illustrate an article with images that are not uploaded on wikipedia? If yes, is there an article that this happens? --Alexignatiou 11:15, 13 April 2006 (UTC)

No, all images must be uploaded to Wikipedia (or the Commons). Wikipedia:Images has some info on this.--Commander Keane 11:17, 13 April 2006 (UTC)

Oh, and something else.. What about images that have been uploaded on other wikipedias? --Alexignatiou 11:20, 13 April 2006 (UTC)

Images uploaded to individual Wikipedias can't be used on other Wikipedias. However, if an image is uploaded to the Commons (it will say on the image description page) then it can be used in any Wikipedia, using the exact same file name.--Commander Keane 11:26, 13 April 2006 (UTC)

As I see that you are a native Greek speaker who is interested in a variaty of subjects I was wondering if you would consider helping me. Needless to say, I don't speak Greek. However, I think it would be a shame if my thesis were to reflect only the views and topics covered in the countries such as Austria, Britain, France, Germany, the Netherlands etc (in short, the countries whose languages I understand).
