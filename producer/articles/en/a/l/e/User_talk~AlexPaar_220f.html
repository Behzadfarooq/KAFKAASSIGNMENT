[edit] Welcome

Hello, welcome to Wikipedia, Alexander!!

Nice to welcome a fellow German here! I hope you don't mind that i have put up Alexander Paar for speedy deletion. You have found the correct place for an user page already, as i see. I haven't marked the pictures (the actual one and the one from the history) for deletion because you can use them on your user page if you want. If you don't want to use them any longer, please mark them for deletion yourself. I've put the usual welcome blurb below, there are some useful links in it.

I hope you like this place and have fun editing.
We always like to meet new Wikipedians!

You might find these links helpful in creating new pages or helping with the above tasks: How to edit a page, How to write a great article, Naming conventions, Manual of Style. You should read our policies at some point too.

If you want get to know more people here, list yourself at Wikipedia:New user log, and go back there sometimes to see if you find people matching your intersts. Go to Wikipedia:Community Portal to learn more about wikipedia and ways to participate.

If you have any questions, see the help pages, or add a question to the village pump. You also can leave me a message at User talk:Lady Tenar, but it may take a few days bevore i see the message and can respond.

Here are some more tips:

  • If you made any edits before you got an account, you might be interested in assigning those to your username.
  • You can sign your name using three tildes, like this: ~~~. If you use four, you can add a datestamp too.
  • If you ever think a page or image should be deleted, please list it at the votes for deletion page. There is also a votes for undeletion page if you want to retrieve something that you think should not have been deleted.

Again, welcome! Lady Tenar 11:01, 16 Nov 2004 (UTC)
