[edit] User login statistics

I'm not sure it's possible to get such statistics, though you could ask on the MediaWiki mailing list. There is a variable called something like "last touched" which might tell you when someone last logged in, so perhaps you could derive something from that by querying the database. Otherwise, you would need to check the server logs, which, if you know each person's IP, you could find out who was reading each page and when, unless the pages are cached.
