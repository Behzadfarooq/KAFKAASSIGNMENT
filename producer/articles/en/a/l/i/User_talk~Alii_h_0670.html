[edit] Reference Atlético de Madrid

The club website itself uses Atlético. However the club originally used Athletic. The name was changed in 1941 when a decree issued by Franco banned the use of non-Spanish language names. If you read the main article this is explained in more detail. The situation is complicated further by the fact that Athletic Bilbao use the English spelling. I think this is were the mix up occurs. The version you used seems to be a strange combination of both. I would recommend using Atlético in reference to the Madrid club and Athletic when referring to the Bilbao club.
