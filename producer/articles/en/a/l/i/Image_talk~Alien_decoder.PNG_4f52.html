I wanted to ask what copyright this image would fall under. I created the image myself, but the font is most likely copyrighted by Futurama, and possibly the site at which I got the font, [1]. I would appreciate if someone could explain what copyright law this may fall under. Thank you for your time. --WillMak050389 11:34, 7 June 2006 (UTC)

If it's a screen shot, it's not under the font's copyright. You can choose GFDL - self created. Waitak 11:43, 7 June 2006 (UTC)
But it is not a screenshot. I downloaded this font and used it for the image. Is the image copyrighted if the font is not mine? --WillMak050389 11:58, 7 June 2006 (UTC)
How did you create the image, exactly? Waitak 12:07, 7 June 2006 (UTC)
I used Microsoft paint, typed the alphabet in TNR and then typed the alphabet using the alien language font. I believe the font itself is copyrighted, that is what I'm worried about, because the font is not my own and I don't want to say it is. --WillMak050389 12:12, 7 June 2006 (UTC)

If you used Microsoft Paint, then it's a self created image. What the font's copyright prevents you from doing is creating another font that's identical, and calling it yours. The image that you created in Microsoft Paint isn't the font. It's a picture, that you created, that has characters from the font rendered in it. Waitak 12:22, 7 June 2006 (UTC)

Thank you for the info. I will change the copyright info now. --WillMak050389 12:24, 7 June 2006 (UTC)

Still, I think it would be appropriate to at least credit the font authors (Darrell Johnson & Leandro Pardini, as credited from http://www.futurama-madhouse.com.ar/fonts/index.shtml), since the picture is specifically about the font. -- RoninBK T C 05:53, 10 March 2007 (UTC)

I have done so. I would have thought I did this before, but I must have forgotten, thanks for pointing it out. --WillMak050389 05:59, 10 March 2007 (UTC)
