[edit] ASCII

This project is about collecting a list of ASCII art with mrWoot.com written out.

[edit] Headers

This was a simple project which started with just a simple image changer. As time grew, the image collection grew into the hundreds. Most photos are taken by me. Some are stock pictures.

[edit] Quote Yourself

Created using PHP and MySQL. It was suppose to be a replica of Bash.org, but never completed. Quotes are not moderated so some might be NSFW.

[edit] Yahoo Webcam

Created for the use of Yahoo! webcams. All the user has to do is enter the webcam user's id and if that user is online and is allowing anonymous users to view, then the person using the page may view the webcam anywhere, without installing Yahoo! messenger.
