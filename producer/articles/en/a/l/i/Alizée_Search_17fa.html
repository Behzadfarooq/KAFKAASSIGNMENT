Alizée Search is "an internet search engine exclusively designed for the promotion of the French singer Alizée Jacotey."[1] It was created in 2006 to obtain better search results for the keyword "Alizée Jacotey" compared to other search engines.[2] It was the first internet search engine to include favicons in its Web search results in 2006.

[edit] Technical details

An internet search engine is an algorithmic information retrieval system to help find information on the World Wide Web. One of the first internet search engines was Aliweb, which was released in 1993. Although the name Alizée Search is similar, it is not known to be affiliated with Aliweb.

Alizée Search retrieves information about Alizée from the Web in the form of Web pages, images, and news articles. It is a specialized search engine based on a focused crawler for only the single keyword "Alizée Jacotey".[2] Opposed to general search engines, specialized search engines try to obtain better search results by focusing on a particular topic. Most specialized search engines employ focused crawlers, which are Web crawlers that attempt to download pages that are similar to each other. A Web crawler is an automated Web browser that retrieves and analyses Web pages. It starts with a list of URLs, which are called seeds. The Web crawler visits these URLs and adds the hyperlinks in these pages to the list of URLs to visit, known as the crawl frontier. Alizée Search does not rely on a general Web search engine for providing starting points,[2] as focused crawlers usually do.

The Web search results of Alizée Search are sorted algorithmically based on a link popularity measure of relevance.[2] They are displayed on the search engine results pages using the title, description tag, URL, and the favicon of the domain.[2] Usually the importance of an item is determined using a link analysis algorithm, which assigns a numerical weighting to each page. It is based on the idea that good or desirable pages are linked to more. An example of such a link popularity importance measure is PageRank of Google.

[edit] Alizée Jacotey

Main article: Alizée

Alizée Jacotey is a French singer born on August 21st 1984 in Ajaccio on Corsica known by her stage name Alizée. She has released two studio albums which were composed by Laurent Boutonnat and written by Mylène Farmer. In the year 2000, Alizée made her debut with the album "Gourmandises". Her first album was a success both in France and abroad, which made her the highest-selling female French singer in 2001.[3] The album featured her most successful single "Moi... Lolita" which reached the number one spot in music charts of most countries in Europe and east Asia in which it was released. In 2003 she returned with a second album called "Mes Courants Électriques". The album was accompagnied by a concert tour "Alizée En Concert", of which a live album and DVD were released in 2004. Since then Alizée has taken a break from singing, but she is rumored to return with a new album in 2007.[4]
