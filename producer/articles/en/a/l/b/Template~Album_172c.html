[view] [edit] Template documentation

Usage

{{Album
|class=
|importance=
|attention=
|needs-infobox=
|auto=
}}

All parameters are optional.

  • For information on the values of class and importance see Wikipedia:WikiProject Albums/Assessment. Album assessment worklists: Index · Statistics · Log
  • needs-infobox should be set to yes if the article needs an album infobox; set to no or remove if an infobox is present or not needed
  • attention: Answer yes if the page needs immediate attention, otherwise remove this line.
  • auto: This parameter is for the use of bots. It calls the {{stubclass}} template.
