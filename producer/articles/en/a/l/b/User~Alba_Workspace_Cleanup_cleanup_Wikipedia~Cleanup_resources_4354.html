[edit] How to mark articles for cleanup

  1. Make sure this is the page you should be using. (See the section "Related pages" below.)
  2. Find the appropriate tag(s) in the table below.
  3. Add the tags to the top of the article.
  4. Add an explanatory note to the article's talk page and/or list the article on Wikipedia:Cleanup.

By doing this, you will put the article into one or more categories where other Wikipedians can find and fix it. The categories are linked from the tables below.

If you don't have time to do anything more than tag an article as needing cleanup, that's fine and a very useful thing to do. (Reading and evaluating articles does indeed take a lot of time and is certainly real work.) If you do have the time to do so, it would be helpful to fix one article already on the list for every one that you add. This helps ensure that at least some fix-it work actually does get done, and helps keep the size of this collection from growing without bound.

[edit] Short list of tags

This article needs cleanup (past this point) (since 04 2007) because:

... it needs wikifying.

  • from the CIA World Factbook.
  • from the Congressional Biographies.

... it is confusing:

  • it has too many abbreviations.
  • it has too many buzzwords.
  • it doesn't provide context (on a book / comic / school / taxon).
  • it contradicts itself or another article.
  • it doesn't explain importance.
    • ... of a music article.
  • it goes off-topic, or to another topic.
  • it reads like an ad /a story / an essay / a resume / spam.
  • it is fiction, or confuses it with fact.
  • it makes prophecies.

... it needs more:

  • text, in a section or the whole article:
    • that is more specific.
    • that has a more global viewpoint.
    • that explains technical details.
  • infobox(es) added / replaced / for a song).
  • diagram(s) / sections / categories / links / citation(s).

... it needs less:

    • of an intro.
    • of a plot summary.
    • trivia.

... it needs rearranging:

    • a complete rewrite.
    • sections to be merged or restructured.

... it needs copyediting:

  • of its inappropriate use of person.
  • of its inappropriate tone.
  • to change units to SI.
  • of its external links.
  • of its recent major revision.

... it needs image editing:

  • for focus, or overexposure, or underexposure, or image size.

... it needs merging: (finish later)

... it needs splitting: (finish later)

... translation is incomplete:

  • unstarted / rough only / passages remain / needs polishing / needs phonetics.

... it has movable material:

  • How-tos to go to Wikibooks.
  • Gaming material to go to a gaming wiki.
  • Vanity pages to go to user space.
  • Quote lists to go to Wikiquote.
  • Definitions to go to Wiktionary.
  • Source material to go to Wikisource.
  • Recipes to go to Wikibooks Cookbook.
  • Media files to go to Wikimedia Commons. [was moved.]
  • Species data to go to Wikispecies.
  • Its content, to a different page (among options). [was moved.]

... an expert should:

  • verify its accuracy.

... AFD suggested it.

... its facts:

  • are out of date / disputed / unverified / unreferenced / unreliable.
  • need their citations added / checked / edited.

... it is a list (without a lead) / an image / a disambiguation page.

[edit] De-listing cleaned articles

  • Remove the {{cleanup}}, {{cleanup-date|April 2006}}, etc. tag from the article itself, if there is one.
  • Remove the listing from Wikipedia:Cleanup, if there is one. Please don't just strikethrough; these will just accumulate in an unsustainable fashion.

[edit] Related pages

For images that need to be improved, see Wikipedia:Images for cleanup.

To request an image or map be added to an article, please see Wikipedia:Requested pictures or Wikipedia:WikiProject Maps/Requested and orphan maps.

For categories that need to be improved, see Category:Wikipedia categories in need of attention.

For repetitive cleanup that would benefit from automation, see Wikipedia:Bot requests.

[edit] Collaboration coordination

Some editors prefer that helpers work on leftovers and older articles, just to deal with them and get them out of the way. But whether your preference is new or old; all help is appreciated. If you work on cleanup regularly, you can add your name and a description of what you do to the Wikipedia:Cleaning department page.
