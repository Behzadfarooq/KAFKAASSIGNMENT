Welcome to the Cleanup Bucket! You probably reached this page by clicking on link from a page marked for Cleanup. We're doing our best to clean up Wikipedia, as part of the general effort to maintain Wikipedia. To do so, we need your help to fix articles.

If you can easily fix the problems on a page, we encourage you to register, be bold, and edit it yourself -- on Wikipedia, everyone is an editor!

Otherwise, please help by taking a few moments to sort out the problem with the page. It's easy! Click on a description in the table of contents, then find the best description(s) of the page's problem. There will be a word or phrase in {{brackets}}. Copy that (the brackets and everything in them); click "edit this article" on the page with a problem, and paste it into the top of the page. Then click "Save page". Voila! The page is now on a list of pages with similar problems, and will get the wikilove it deserves.

For pages needing topic-specific attention, you can add the page to Wikipedia:Pages needing attention, which uses a slightly different system -- see below.

This page is a shorter version of Wikipedia:Cleanup resources, which displays each template in full and describes exact guidelines for using templates.

[edit] This doesn't look like a Wikipedia page

  • Its content is not encyclopedic:
    • it's an {{advert}}isement.
    • it's {{PotentialVanity}}.
    • it belongs elsewhere:
      • as a dictionary defintion, {{Move to Wiktionary}}.
      • as a quote collection, {{Move to Wikiquote}}.
      • as a source text, {{Move to Wikisource}}.
      • as textbook material, {{Move to Wikibooks}}.
      • as a recipe, {{Move to Wikibooks Cookbook}}.
      • as a media collection, {{Move to Wikimedia Commons}}.
  • It needs translation:
    • it's completely {{notenglish}}.
    • a dually-fluent person should {{cleanup-translation}}.
    • it's only a {{RoughTranslation}}.
    • it's oddly pronounced, and needs {{cleanup-ipa}}.
  • It needs to be {{wikify}}ed:
    • it needs {{Sections}}.
    • there is a {{diagram needed}}.
  • It's too short: it's a {{stub}}.
    • Stubs are sorted; if you know the right stub category, help out by using a specific stub.

[edit] This page needs editing help

  • it is {{confusing}}:
    • it has insufficient {{context}}.
    • it's hard to tell fact from {{fiction}}.
    • it is {{contradict}}ory.
    • it doesn't discuss its {{importance}}.
      • it doesn't assert {{music-importance}}.
    • a section is {{off-topic}}.
      • there's another topic it discusses: {{off-topic-other}}
  • it is poorly {{copyedit}}ed:
    • the writing uses {{inappropriate person}}.
    • it uses {{inappropriate tone}}.
    • it uses too many {{abbreviations}}.
    • it has a large {{intro length}}.
    • it has poor {{citation style}}.
    • it needs to be {{globalize}}d.
  • it has a problem relating to its facts:
    • its facts are {{not verified}}.
    • it needs a {{citecheck}}.
    • it has poor {{citation style}}.
    • it's out of date and needs an {{Update}}.

[edit] Some (or all) of this article doesn't belong here

  • it doesn't belong here:
    • and we should {{move}} it.
      • it's been {{moved}}.
    • and it should be {{split}}.
      • just {{splitsection}}, with a new title {{splitsection}}.
    • and we should {{merge}}.
      • on the destination page, {{mergefrom}}.
        • just merge a section: {{mergesection}}.
      • on the source page, {{mergeto}}.
      • and the merger is in progress: {{merging}}.
      • and someone objected! {{mergedisputed}}.

[edit] How to request topic-specific attention to a page

To list an article for Pages needing attention, follow this two-step process, replacing PageName with the name of the page to be attended to.

I.
Add the article to the Pages needing attention category.

  Click the link below corresponding to the correct Pages needing attention/Category page.
  Edit the page, and copy this text into it:

{{subst:attention1 | pg=PageName | text=Reason why the page needs attention}} ~~~~

Replace PageName with the name of the page to be fixed, and include a reason why you think the page needs attention, where indicated. Check the "Watch this page" box if you would like to follow the editing process in your watchlist. Please describe fully what you think is wrong with the article; the more you explain, the more likely the problems are to be fixed, and the faster the fixes will happen. Save the page.
II.
Put the attention tag on the article.

  Edit the article, and put the following tag at the top of the page
  (You can use your web browser to copy and paste the text):

{{subst:attention2|CategoryName}}

Replace CategoryName with the name of the same category you added the page to in step one. This adds PageName to the automatic attention page lists, and brings it to the attention of the knowledgeable Wikipedia editors. Save the page.

[edit] This page is OK, but needs an expert to make it better

If the page looks clean, with few or no editing problems, but is still too short or incomplete, you can replace the {{cleanup}} tag with {{Expert}}. This says to any interested Wikipedians, I want an {{Expert}} to add more and better content.

Note: don't add this tag if the article still has serious editing problems! We want our experts to add quality material to already encyclopedic articles, not to be fixing major style problems themselves. If you think the page needs both editing help and topic-specific attention, add editing tags and list it on WP:PNA, as shown above.
