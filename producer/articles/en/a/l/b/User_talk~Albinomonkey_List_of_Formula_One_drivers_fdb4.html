[edit] dns

  1. ↑  Amati failed to qualify on all 3 attempts, and so never started a Formula One race.
  2. ↑  Bartels failed to qualify on all 4 attempts, and so never started a Formula One race.
  3. ↑  Bayardo qualified for, but did not start the only race he entered.
  4. ↑  Bertaggia failed to pre-qualify in all 6 events he entered, and so never started a Formula One race.
  5. ↑  Gary Brabham failed to pre-qualify in both events he entered, and so never started a Formula One race.
  6. ↑  Ernesto Brambilla failed to qualify in the 1963 Italian Grand Prix, and did not start (his car was used by Pedro Rodriguez) the 1969 Italian Grand Prix, as so never started a Formula One race.
  7. ↑  Brancatelli failed to pre-qualify in the 1979 Monaco Grand Prix, and did not qualify in the other two events he entered, and so never started a Formula One race.
  8. ↑  Chaves failed to pre-qualify in all 13 events he entered, and so never started a Formula One race.
  9. ↑  Cogan failed to qualify in both events he entered, and so never started a Formula One race.
  10. ↑  Colombo failed to pre-qualify in the 1978 Italian Grand Prix, and failed to qualify in the other two events he entered, and so never started a Formula One race.
  11. ↑  Crespo failed to qualify in the only Grand Prix he entered, and so never started a Formula One race.
  12. ↑  de Changy failed to qualify in the only Grand Prix he entered, and so never started a Formula One race.
  13. ↑  de Dryver failed to qualify in the 1977 Belgian Grand Prix and failed to pre-qualify in the 1978 Belgian Grand Prix, and so never started a Formula One race.
  14. ↑  de Riu failed to start the only Grand Prix he entered, and so never started a Formula One race.
  15. ↑  Dochnal failed to qualify for the 1963 Mexican Grand Prix, and so never started a Formula One race.
  16. ↑  Dusio failed to qualify for the 1952 Italian Grand Prix, and so never started a Formula One race.
  17. ↑  Ecclestone failed to qualify for the 1958 Monaco Grand Prix, and failed to start the 1958 British Grand Prix, and so never started a Formula One race.
  18. ↑  Zapico failed to qualify in the only Grand Prix he entered, and so never started a Formula One race.
