[edit] Cleanup

Many cleanup templates listed below need their very own minor cleanup. They add tagged pages to a category, unfortunately also lists with examples. To disable this replace
...<includeonly>[[Category:etc.]]</includeonly> by
...<includeonly>{{{category|[[Category:etc.]]}}}</includeonly>.

{| border="1" cellspacing="0" cellpadding="3" width="100%" !Tag !Text that will be shown (and usage notes) {{tlrow|cleanup|category=Wikipedia cleanup}} * This applies to general problems not addressed by other tags. * '''In the future, please use cleanup-date, below, instead.''' {{tlrow|cleanup-date|{{CURRENTMONTHNAME}} {{CURRENTYEAR}}|category=Cleanup by month}} * This applies to general problems not addressed by other tags. * Please do '''not''' use this in combination with the [[#Cleanup_.E2.80.94_specific_issues|more specific]] cleanup tags. {{tlrow|cleanup-remainder|{{CURRENTMONTHNAME}} {{CURRENTYEAR}}|category=Cleanup by month}} * Use this tag where the remaining sections of an article need cleanup. {{tlrow|CleanupConfusing|{{CURRENTMONTHNAME}} {{CURRENTYEAR}}|category=Cleanup by month}} * Use this tag in articles that need broad cleanup to eliminate confusion and discrepancies. {{tlrow|cleanup-list|category=Wikipedia list cleanup}} * Use this tag where a list needs cleanup. {{tlrow|cleanup-image|category=Images for cleanup}} * This should be used for images that have artifacts, poor definition, poor lighting, or other visual flaws. * Many of these images may need to be replaced entirely. Try listing on [[Wikipedia:Image recreation requests]]. {{tlrow|disambig-cleanup|category=Disambiguation pages in need of cleanup}} * Use this for disambiguation pages not written according to the style in [[MoS:DP]]. * Replace {{Lts|disambig}} with this. {{tlrow|cleanup-afd|category=Wikipedia cleanup after AFD}} * Use this tag where an [[Wikipedia:Articles for deletion|AFD]] discussion closes with a consensus of "keep and cleanup." |}{{-}}
