 This is the template documentation page for Template:Allmusicguide.
This page is not necessarily intended to be viewed directly. If it uses variables, some links may appear broken. Do not replace these variables with hardcoded page names or URLs.

[edit] Purpose

To provide a convenient link in the External links section of a music group to All Music Guide's entry (if available).

[edit] Usage

{{allmusicguide | id= | label= }}

where

  • id is the id found in the URL of the external, immediately after sql=; for example, this is highlighted in http://www.allmusic.com/cg/amg.dll?p=amg&sql=11:2o220r2at48j for Deerhoof
  • label is an optional label to display the link, which defaults to {{PAGENAME}}, or the name of the article where this template is used

[edit] Examples

On the Stereolab article, you can simply use the id (the label will be automatically be formatted from the article name:

*{{allmusicguide | id=11:t09ds31ba3zg }}

results in:

  • Stereolab at All Music Guide


A label other than the article name can be specified, for example in the Faust (band) article:

*{{allmusicguide | id=11:bl5a8qxtbtv4 | label=Faust }}

results in:

  • Faust at All Music Guide
