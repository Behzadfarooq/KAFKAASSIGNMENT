AllPeers is a open-source, freeware extension for the Mozilla Firefox browser. The extension allows building a social network and sharing files on a P2P basis. It uses a darknet style of peer-to-peer communication; files and information shared between users are only accessible as long as the users have each other on their respective access lists - their 'trusted private network'. AllPeers beta version was launched on August 24, 2006, and is still currently in development. It works on Windows, Linux and Mac, which, once downloaded, becomes a toolbar in Mozilla Firefox.

Even though it is closed-source, it uses open source BitTorrent technology.[1] The extension does not require any ports be opened. AllPeers encrypts its communication using standard protocols like SSL so as to protect the user against 3rd party intervention listening in.[2] AllPeers at the start its beta launch was 200,000 lines of C++ and JavaScript code.[1]
