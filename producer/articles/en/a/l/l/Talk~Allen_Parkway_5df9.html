[edit] Topozone source

The topozone source requires some counting and some patience. As can be verified by enlarging the map and looking elsewhere in the Houston area, each line respresents 5 feet. However, contour labels and elevation points are scarce on this particular map. Counting lines, though, one can see that there are 7 lines between the bayou and Allen Parkway (plus a bit of extra elevation on each side of countour 1 and countour 7). This is not original research, as it just requires good eyesight and some simple counting. If anyone can find a prettier map, please add it as a footnote. Thanks. Ufwuct 16:16, 23 September 2006 (UTC)

[edit] USGS source

The data comes from the only USGS streamgage on Buffalo Bayou that is adjacent to Allen Parkway (where Allen Parkway ends at Shepherd Drive). The interactive Google Earth map of Texas streamgages is available at WaterWatch on the USGS website. Moving downstream, the next streamgage is where White Oak Bayou joins Buffalo Bayou. Although this is probably more representative of Allison's flooding, it is past Allen Parkway. Also, the source only provides daily average streamgage heights. Given that the flooding on many of these roads was gone within a day, the peak streamgage height could have been much higher. I hope this helps. Ufwuct 16:33, 23 September 2006 (UTC)
