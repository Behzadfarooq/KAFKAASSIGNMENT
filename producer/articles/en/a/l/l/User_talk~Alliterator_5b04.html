[edit] Veronica Mars

Thanks for being bold and creating the new Veronica Mars episode articles, as well as the List of Veronica Mars episodes. I wanted to call your attention to a number of changes I've been making to the articles, based on Wikipedia and Wikiquote style practices:

  • I've changed the article titles to follow UPN's list and WP style guidelines, so some of the articles have slightly different titles than the way you started them.
  • I changed the italicization of episode titles to quotation marks.
  • I've eliminated the section heading capitalization, which is a very common mistake. (In fact, based on the capitalized headings you used as a template, I'm guessing you got the structure from Buffy the Vampire Slayer or Angel, which also have had that problem.)
  • I reformatted quotes to follow Wikiquote's style practices for dialog and added a WQ box link in the "Quotes and trivia" sections.
  • I created two new categories:
    • Category:Veronica Mars episodes for the episodes themselves, which itself belongs to…
    • Category:Veronica Mars, which should eventually include all things VM. This I put in Category:Drama television series for starters.
  • I also added some non-printing spacing in the articles to make it easier on editors. Far too often, we cram everything into spaceless text that looks more like C programming than encyclopedia text.

Before you start your next batch of article creation, please look over these changes so that you can incorporate them in your excellent work. Thank you. ~ Jeff Q (talk) 03:38, 2 February 2006 (UTC)

Thanks for editing the List of Veronica Mars episodes that I made. I'm new to making articles in Wikipedia. And you're right, I did copy-and-paste from Angel episode pages. And thanks for setting up categories, which I'm still unsure about how to make. alliterator 10:37, 01 February, 2006

You're off to a great start! Don't worry too much about making mistakes; based on Angel, you can see that even veteran editors don't always have their ducks in a row. ☺ A key tenet in wiki philosophy is that someone will eventually come along and fix things. In the process, of course, we all learn about those fine points from watching our favorite articles get edited.

One aid to this process is the edit summary — the input box just below the edit window — where you should always include a few words about the purpose of your edit. Click on the "history" tab of the VM articles to see how much easier it is to follow the changes when the edit summary is used. (Sorry about the soapbox, but I get annoyed when old timers ignore this critical feature, so I've made it a mission to educate new editors before they learn bad habits. ☺)

Categories are indeed a complicated feature. Here are a few tips for working with them:

  • Categories are started simply by adding a category tag to an article, but they aren't fully created until you click on the red link, add some text describing the contents of the category, and add the category itself to a parent category. Edit Category:Veronica Mars to see what the fully created category page contains.
  • If you aren't sure about what parent to add a category to (a very common problem), use a similar article as a guideline. (I checked Buffy the Vampire Slayer, Rupert Giles, and one of the Buffy episode articles to see how to set up corresponding VM categories. I noticed that someone else had already started Category:Veronica Mars characters, and just added some articles that had fallen through the cracks.
  • To add a category, the syntax looks like [[Category:Veronica Mars episodes]]. To talk about a category and provide a link to it, like I'm doing here, the syntax is [[:Category:Veronica Mars episodes]] (note the preceding colon).

I hope you find this information useful. ~ Jeff Q (talk) 16:03, 2 February 2006 (UTC)
