[edit] Image copyright

Alagappan, if you've taken the clock tower image yourself and you don't have any problems in open sourcing it under GFDL, can you please go to the image page (Image:Nitt.adblock.jpg), click edit and add {{GFDL}} to it? Otherwise it can't be used in articles. Thanks for the Cities template and image. -- Sundar (talk • contribs) 08:59, Apr 6, 2005 (UTC)

Unfortunately, we can't add copyrighted images to Wikipedia. Some companies like Google, Yahoo! etc will have standard logos that can be used in appropriate places. Also, some screen shots are permitted. Most preferred are the ones hand drawn or photographed by Wikipedians. We also have a number of public domain sources for images. Btw, if you have GFDL images, add it to commons. It can be accessed from other Wikipedias like Tamil Wikipedia -- Sundar (talk • contribs) 11:09, Apr 6, 2005 (UTC)
