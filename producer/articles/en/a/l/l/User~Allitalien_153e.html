[edit] Why I Contribute To Wikipedia

When I first used Wikipedia I thought that it was a great resource, I later found out that it is completely user created and that no expertise is necessary to contribute to Wikipedia. This led me to view Wikipedia as a joke.

A friend suggested going on and creating my own page. So I did; I learned all the HTML and Wikipedia tags that I need to know and I spent about 6 hours doing a biography on myself.

I woke up the next morning to see that my page had already been deleted for "csd a7"

I didn't know who checked resources or citations, I just knew that I was pissed. I later found out that there are hundreds of admins patroling Wikipedia and I decided I wanted to give them a hand... If I wasn't going to have my own page I didn't want anyone to have one.

So anyway, each day I learn a little more about tagging on Wikipedia, and recently I updated the "Paolo" page and listed all notable Paolo's on wikipedia, hoping that one day I would be able to add myself to that list.

I also recently created this User section. I figured this was a good start to getting my name onto wikipedia!
