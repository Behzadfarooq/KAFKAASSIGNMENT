[edit] Error on Multilingual error message

Thank you for pointing this error out to me, as I never would have noticed. It happens because the URLs for the Japanese and Polish donations pages which I had pulled used a "%C9" style of coding instead of actual UTF-8 characters, and the error details are inserted using codes like "%C". I've corrected the error and the fixed version should go live as soon as the developers get around to it. The new version will also automatically show Japanese when viewing ja.wikipedia.org (and the same goes for the other languages). Previously, it would only default to Japanese if the user had their operating system set to Japanese by default. Anyway, once again, thank you. :) - Mark 04:58, 6 December 2005 (UTC)

My pleasure. Cool work, Mark! 朝彦 (Asahiko) 06:13, 6 December 2005 (UTC)
