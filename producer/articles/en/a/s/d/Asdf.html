This article is about asdf in computing culture. For other uses, see ASDF.

"asdf" is the sequence of letters that appear on the first four keys on the home row of a QWERTY or QWERTZ keyboard. The string can be easily typed with the fingers of the left hand. They are often used as a sample or test case or as random, meaningless nonsense. It is also a common learning tool for keyboard classes, since all four keys are located on the home row.

asdf is a default password for some systems. Therefore the word is used in many dictionary crackers, since there is always the possibility that some user has left their password as asdf by default.

Many internet users who are uninterested in choosing something more personalized use asdf as their username and password pair so they can obtain quick access to the information or links that have been restricted. This has led to many websites specifically invalidating asdf@asdf.com as an email address. Similarly, some online games have restricted using asdf or asdfg as a password, as it is easy to guess[citation needed] .

Some users choose to set up asdf as the command to switch from QWERTY to Dvorak, with aoeu as the reverse. [1]

asdf, being a very common sequence of letters, has led to various sarcastic jocularities analyzing its meaning. One such example is asdf.com.
