While it might seem reasonable to assume that such files can be freely used on Wikipedia, this is in fact not the case. Please do not upload any more files with this restriction on them, because content on Wikipedia needs to be compatible with the GNU Free Documentation License, which allows anyone to use it for any purpose, commercial or non-commercial.

If you created this image (i.e. you shot the photo or drew the image yourself) and want it to be available on Wikipedia, you may re-upload the image, choosing from the Licensing dropdown menu "GFDL (self made)", "Creative Commons Attribution-ShareAlike 2.5", or "You created this yourself and release it into the public domain." Be aware that every one of these options allows other people to legally modify or re-use your image for commercial or non-educational purposes without asking further permission. If the image has already been deleted, you may re-upload it, choosing a correct license from the dropdown menu.

If you did not create this image but want it to be used on Wikipedia, there are two ways to proceed. The first way is to re-upload the image, adding to the image's description page a fair use tag from this list if you believe one of those fair use rationales applies to the file. Every fair use image which is not used in an article, or does not name its copyright owner, or does not have a rationale for its use will be deleted. The second way is to contact the copyright holder and request that they make the media available under a free license.
