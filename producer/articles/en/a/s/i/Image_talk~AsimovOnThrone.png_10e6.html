[edit] Changed image tags to indicate invariant sections

The traditional Template:GFDL template includes a statement that the document contains no Invariant Sections or Cover Texts. Since this image includes an Invariant Section, that template seemed inappropriate. Thus, I created a new template, Template:GFDL-invariants, and changed this image to use that.
