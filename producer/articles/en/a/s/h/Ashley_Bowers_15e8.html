Ashley Bowers

[edit] Description

Ashley Bowers is a graduate of MIT and resides in Nashville Tennessee and is currently working on web programming and is developing several mobile websites that help people learn about web standards.She also has a blog on her webiste that covers topics ranging from nano technolgy to web standards.
