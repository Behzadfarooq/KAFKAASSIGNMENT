[edit] re: newspaper scan

Hi Ashley, the easiest thing to do in that situation is remove the license info altogether, so the image will get automatically deleted in a week. You could upload it to some site like imageshack.us for review by the admins, if you need it up longer than a week. Uploading to Wikipedia is (mostly) only for images for use in encyclopedia articles, so they either have to be free-licensed or have a fair use rationale for inclusion in an article. Wikipedia is getting more and more paranoid about copyright infringement so they've become very strict about this. For fair use info see Wikipedia:Fair use criteria but it's tedious and getting worse. Phr (talk) 17:00, 10 August 2006 (UTC)
