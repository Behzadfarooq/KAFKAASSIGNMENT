The Acer Aspire 5000 is a laptop based on the AMD Turion 64. It is based on the SiS M760GX chipset.

Along with the included Windows XP SP2 operating system these notebooks will also run alternative x86 and x86 64 operating systems such as Linux and FreeBSD.

[edit] Specifications

[edit] Aspire 5002 NWLMi/WLMi

  • Processor: AMD Turion 64 ML-30 ( 1.6 GHz , 1 MB L2 Cache , 35W TDP )
  • Memory: 256/512/1024 MB PC2700 DDR RAM
  • Hard Drive: 60/80/100 GB HD
  • Optical Drive: Dual Layer, Multi-Format DVD Writer
  • Monitor: 15.4" WXGA+ 1280x800
  • Operating System: Windows XP Home w/ SP2 or Windows XP Professional w/SP2
  • Video: SiS M760GX
  • Weight: 6.6 lb
  • Battery Life: 2.5 to 3 Hours
  • Modem: 56k Modem
  • USB 2.0
  • PCMCIA Slot
  • Broadcom 4318 80211b/g

[edit] Aspire 5003 WLMi

  • Processor: AMD Turion 64 ML-32 ( 1.8 GHz , 512 KB L2 Cache )
  • Memory: 512 MB DDR333
  • Hard Drive: 80 GB HD or 100 GB HD
  • Optical Drive: Dual Layer, Multi-Format DVD Writer
  • Monitor: 15.4" WXGA+ 1280x800
  • Operating System: Windows XP Home w/ SP2
  • Video: SiS M760GX
  • Weight: 6.2 lb
  • Battery Life: 2.5 to 3 Hours
  • Modem: 56K ITU V.92 Modem
  • USB 2.0 x 3
  • PCMCIA Slot
  • Broadcom 4318 80211b/g

[edit] Aspire 5004 NWLMi/WLMi

  • Processor: AMD Turion 64 ML-34 ( 1.8 GHz , 1 MB L2 Cache , 35W TDP )
  • Memory: 512/1024 MB PC2700 DDR RAM
  • Hard Drive: 100 GB/120 GB HD
  • Optical Drive: Dual Layer, Multi-Format DVD Writer
  • Monitor: 15.4" WXGA+ 1280x800
  • Operating System: Windows XP Home w/ SP2 or Windows XP Professional w/SP2
  • Video: SiS M760GX
  • Weight: 6.6 lb
  • Battery Life: 2.5 to 3 Hours
  • Modem: 56k Modem
  • USB 2.0

  • Broadcom 4318 80211b/g
