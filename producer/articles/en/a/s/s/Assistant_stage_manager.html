Assistant stage manager

An ASM or assistant stage manager is a theater technician who is responsible for keeping track of cast members, sets, and miscellaneous details during rehearsals and production of a stage performance. During the running of a show, they are usually located backstage. They can communicate with performers and technicians locally as well as other technicians in the running crew using a headset. They usually report to the Production Stage Manager.

Some possible duties and responsibilities of an ASM

  1. Attend all rehearsals as needed by the Stage Manager
  2. Assist with the running of props and costume pieces
  3. Follow the "book" and prompt
  4. Run lines with actors when asked
  5. Prepare the rehearsal stage for work each day (sweep, mop, set furniture, etc.)
  6. Run errands for the director and stage manager
  7. Participate in strike
  8. Assist the stage manager in any assignments given.
  9. Attend all production meetings.
  10. Supervise all crews backstage during tech week and performance.
  11. Inspect the stage before each performance.
  12. Inspect and sweep the rehearsal space before each rehearsal.
  13. Supervise the proper storage and placement of all props, furniture, movable set pieces, and costumes before and after each rehearsal and performance.
  14. Be in contact with the stage manager during rehearsal period and all performances.
