This page is for processing advocate verifications ofr the AMA IRC channel (which is #AMA-Wikipedia). We would encourage all advocates to join the channel, so that it can be used as a forum for help and discussion. If a non-advocate seeking help joins the channel, they will, at first, be unable to speak, however it is likey that an op in the channel will voice you temporarily. Currently, the ops are Martinp23 and Royalguard11. We would invite anyone with any questions to join the channel The requirement for voice makes sure, for us, that we can keep the channel secure by identifying members as opposed to other users. To further reiterate our point - We need you to join the channel to make it a success!

Please add your wikipedia username below (using ~~~ (three tildes), followed by your IRC username, so you can be verified on the channel (which is #AMA-Wikipedia) and voiced. If you have no idea what we're talking about, see Wikipedia:IRC tutorial. Thanks.

IMPORTANT: There is a discussion about this at the current meeting. This can be used instead of downloading an actual client. Just set the server to irc.freenode.net, and the channel to #AMA-Wikipedia. /nickserv register <password> can be used to register your nickname, and /nickserv identify <password> can be used to identify yourself after.
