[edit] Documentation

{{actionon}} can be put on templates within an <includeonly> to say that some action (e.g. deletion) is to take place a certain number of days[1] from the specified date. The page will go into Category:Action to be taken on month day. The category will be used for all actions to be done on that day.

[edit] Usage

{{Actionon|month|day|later}} - An action on the page is scheduled for later[1] days after month day

[edit] Example

Code Result
{{Actionon|4|16}} [[Category:Action to be taken on {{subst:day+x|4|16|7}}]]
{{Actionon|4|16|5}} [[Category:Action to be taken on {{subst:day+x|4|16|7}}]]
