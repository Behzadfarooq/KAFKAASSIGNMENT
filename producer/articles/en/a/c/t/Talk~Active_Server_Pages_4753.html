I didn't understand whether ASP or ASP.net could run on a non-windows server. I've been to two different web hosts where one says they are going to offer ASP support while the other host says ASP is impossible to have on non-windows servers. ASP support comes installed only on IIS(Windows) servers. This doesn't mean that it will not run on others, but it does require that you install specialized software such as Sun Java System Active Server Pages (for linux): http://www.sun.com/software/chilisoft/ (for example) Bryantn 20:51, 6 February 2007 (UTC)
