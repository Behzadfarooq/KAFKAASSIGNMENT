[edit] Pictures

Some things I've noticed:

  • Please removed attribution from articles and leave it on the image's page
  • Name them something more descriptive than "WIKI XXXX.jpg"
  • Please don't use all caps in names: there's no need for them
  • Not all pictures have to reside in a <gallery>
  • If you do use a gallery, please add captions after the pipe "|" (see User:Cburnett#Cameras for an example)
  • Please use spaces instead of underscores in file names (they are auto-converted)

Please let me know if you have any questions! Cburnett 16:18, July 25, 2005 (UTC)
