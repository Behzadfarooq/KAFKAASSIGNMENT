Action Replay DS

Action Replay DS is a Nintendo DS game hacking device by Datel and a part of their popular Action Replay product line. Released worldwide in August 2006, it enables users to use predefined codes to edit in game behaviour.

Unlike other Action Replay devices, codes are not entered by the user, but are stored on the cartridge after being downloaded via a subscription in the Action Replay client software. The subscription is a simple XML file that contains the cheat data for a specific game. At the time of writing the only real source for subscriptions is the Codejunkies website, which is run by Datel. It can also be used as a PassMe. A recent firmware update makes it possible to edit codes using the Action Replay itself when plugged into the DS console, which was previously impossible.

One inconvenience consumers have noticed is that if the DS lid is closed while the AR software is running, the DS will not enter sleep mode, which can waste battery power. Note that this problem only applies to the AR software itself. Once playing a game, even with AR codes in place, closing the lid has the normal DS function. Another inconvenience about the action replay ds is that if too many codes are uploaded the action replay will not accept any more or in a rare case, crash the whole system. A way to fix this crash is to hold A + B then turn on DS, when the "This product is not liscensed or endorsed by Nintendo" HOLD A + B + Select + Start, when it reaches in the menu check the lefthand corner to see if it says 0.

A recent update in the software now allows users to enter their own codes through the PC application, which can then be transferred to the DS cart. You make your own codes by buying a hacking system called "Trainer Toolkit."(Warning! The Trainer Toolkit should be handled by a pro in hacking. You may mess up your computer and or Nintendo DS) As of AR DS v1.52, the trainer toolkit for AR DS is no use, it now includes code adding in the device itself, still it is a little glitchy.

The codelist editing is ran by XML files. Many releasers of codes edit main XML codelists and then release them.

This product should not be confused with the similar Action Replay Duo, a save game device for DS games. The Action Replay Duo is for DS and Gameboy Advance. Action Replay DS is sold in stores all around for the price of $24.99. Note: This product may corrupt the saved progress files on the ds cartridge you are using. Ex: Mario Kart Ds will take away 'Rob' as a player if the 'All character unlocked' box is selected, even if they are all there in the first place.
