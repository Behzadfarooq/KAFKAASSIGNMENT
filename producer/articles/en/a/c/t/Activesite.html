Activesite is a web content management system. It uses the Model-view-controller and Open standard based on technologies like Java Platform basically J2SE and J2EE along with RDBMS and XML. The core J2EE technologies used are Servlet API 2.3, JSP 1.2, JNDI 1.2, JTA 1.0.1, JavaMail 1.3, and JDBC. Other technologies used are Jakarta Struts, Hibernate and Spring framework.

The system has pluggable modules such as an e-commerce module, webshop module etc.

Companies such as Crompton Greaves and Tata Share Registry Ltd, Banasthali Vidyapith University are using Acitvesite for their web publishing requirements.
