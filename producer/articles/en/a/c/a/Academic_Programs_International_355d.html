Academic Programs International

API, Academic Programs International, is an independent study abroad provider based in the US. Its locations include Costa Rica, France, Hungary, Ireland, Italy, Mexico, Poland, Spain, and the United Kingdom. API procures housing and educational arrangements as well as excursions in each program.

[edit] Fees

Average tuition costs range from $7500 to $14,000/semester. Year programs can go as high as $25,000. These fees are flat-rate (do not vary with how many credit hours are taken in that period, which is variable) and include everything from housing, meals, travel, medical insurance, etc.

[edit] Excursions

Each program typically offers 4-9 extracurricular excursions (depending on the selected destination) included in the flat-rate fee. They last a few days and travel is done with other students in the API group. For any other excursions on the student's behalf, Europe's Eurail is a popular and convenient way to get around (see Eurail on Wikitravel).

[edit] Other Information

  • API provides a scholarship of $250-1000. It is partially based on financial need and part on the student's merits (essays/recommendations).
  • Housing is recommended to be a host family, though student apartment alternatives are available. For a true immersion experience, a host family is a must.
  • Both passports and visas are necessary to participate, so it's always a good idea to get those as soon as possible.
