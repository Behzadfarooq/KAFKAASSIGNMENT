Academically priced software

From Wikipedia, the free encyclopedia

Academically priced software is a software package for resale to students and teachers and non-profit organisations at a substantialy discounted price.

Academically priced software packages can come with little or no printed instruction manual with the understanding that academics are going to be taught how to use the software by professionals. Also, this saves on cost for the manufacturer, thus passing the savings to the customer. However with microsoft software they provide the full instuctions, and the product will be packaged the same way.

Academically priced software is usually not eligible for upgrade and crossgrade discounts. Academically priced software is usually not eligible for customer support - again, because there is an understanding that the academics are going to be taught how to use the software. However microsoft again includes all features. [1]
