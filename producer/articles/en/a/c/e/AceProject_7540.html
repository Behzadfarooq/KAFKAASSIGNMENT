AceProject

Developer: Websystems, Inc
Use:
Website: AceProject.com

AceProject is a web-based project management tool, created by Websystems, Inc. in 2001. This software allows multiple project management, creating Gantt Charts, generating time and task reports, and time tracking.

Its web-based structure allows one to use the software from any computer, with no need to install anything. The source code is available for purchase.

AceProject is an alternative to major project management software. In addition to being a web-based tool, it can be installed locally and even customized. Other AceProject features include: Project templates, task reminder, international characters support, calendars, and more.
