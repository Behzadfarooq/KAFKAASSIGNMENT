Acer N50

From Wikipedia, the free encyclopedia

Image:Acer n50 in cradle.jpg
The Acer n50 in its USB cradle

The Acer n50 is a PDA produced by Acer. It is part of the Acer N Series of PDAs.

[edit] Features

[edit] Operating System

The Acer n50 was one of the last PDAs produced to use the Windows Mobile 2003 SE operating system.

[edit] Memory

The Acer n50 has 64MB of system memory (RAM) and 64MB of Flash Memory. It has a SD slot and a Compact Flash slot, providing much expandability. It has a USB host port, for use of Flash Drives and similar USB mass storage devices.

[edit] Processor

The relatively fast 312Mhz processor is easily enough to allow applications to run. It minimises power usage to provide 8 hour battery life.

[edit] Display

The 3.5" TFT LCD display with QVGA resolution (320 x 240)gives the user a clear view of the programs that are running, and its ability to display 65,536 colours allows for any application to use its full colour potential.

[edit] Connectivity

The integrated IrDA, Wi-Fi and Bluetooth allows for connections to the internet and any other devices with IrDA and Bluetooth. The Bluetooth connection allows such things as Bluetooth GPS and Bluetooth Keyboards to be used wirelessly by the Acer n50.
