Accessibility Inspector

Accessibility Inspector inspecting a Mozilla Firefox window showing the Wikipedia main page.
Developer: Apple, Inc.
Latest release: 1.0 / 2004
OS: Mac OS X
Use: Software utility
Website: http://www.apple.com/macosx

Accessibility Inspector is a software utility by Apple, Inc. made to verify the accessibility of Mac OS X applications. It displays information about the GUI element that is currently under the cursor.
