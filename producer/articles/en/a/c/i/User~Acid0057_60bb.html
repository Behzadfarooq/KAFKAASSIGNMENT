How to create a link to another article

When you are editing an article in Wikipedia, and you want to create a link to another Wikipedia article, place double square brackets around the topic you want to link to. For example:

[[orange]]

produces this link:

orange

For the plural write this:

[[orange]]s

which will look like this:

oranges
