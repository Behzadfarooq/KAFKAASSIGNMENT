[edit] Documentation

This category contains novel articles graded according to the Wikipedia 1.0 Assessment Scale.

Do not put items in this category. Put them in the appropriate category by adding or editing the film's {{NovelsWikiProject}} Talk page tag, as follows:

  • {{NovelsWikiProject}} — for an initial article that has yet to be assessed. If unsure use this!

If you know an article to be of a particular class assign it to one using the following:-

  • {{NovelsWikiProject|class=FA}} — for Feature Article class articles
  • {{NovelsWikiProject|class=A}} — for A class articles
  • {{NovelsWikiProject|class=GA}} — for Good Article class articles
  • {{NovelsWikiProject|class=B}} — for B class articles
  • {{NovelsWikiProject|class=Start}} — for Start class articles
  • {{NovelsWikiProject|class=Stub}} — for Stub class articles

Note: You should not assign any article GA, A, or FA grades arbitrarily. These grades must pass through official Wikipedia channels.
