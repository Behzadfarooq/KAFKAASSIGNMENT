[edit] Documentation

This category contains Gilbert and Sullivan articles graded according to the Wikipedia 1.0 Assessment Scale.

Do not put items in this category. Put them in the appropriate category by adding or editing the film's {{G&S-project}} Talk page tag, as follows:

  • {{G&S-project}} — for an initial article that has yet to be assessed. If unsure use this!

If you know an article to be of a particular class assign it to one using the following:-

  • {{G&S-project|class=FA}} — for Feature Article class articles
  • {{G&S-project|class=A}} — for A class articles
  • {{G&S-project|class=GA}} — for Good Article class articles
  • {{G&S-project|class=B}} — for B class articles
  • {{G&S-project|class=Start}} — for Start class articles
  • {{G&S-project|class=Stub}} — for Stub class articles

Note: You should not assign any article GA, A, or FA grades arbitrarily. These grades must pass through official Wikipedia channels.
