[edit] Using {{fb start}} and {{fb end}}

This navigational box related to football (soccer) uses {{fb start}} and {{fb end}}. Place this template between {{fb start}} and {{fb end}} in the article, like this:

{{fb start}}
{{AFC Leagues}}
{{fb end}}

If there already are other navigational boxes in the article using this standardization, place this template between {{fb start}} and {{fb end}} in the article, like this:

{{fb start}}
{{AFC Leagues}}
{{Another football template}}
{{fb end}}
