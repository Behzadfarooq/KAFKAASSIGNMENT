The Aukyla Platform, previously known as the Aukyla PHP Framework, is a platform for creating, managing and deploying web-based PHP5 applications.

The platform provides an Administration Area for user and configuration management and for installing, upgrading and removing Aukyla applications.

The Aukyla Platform provides its own Aukyla Application Package Format[1] which is used for deploying applications.

It also features a framework for building web-applications. Among other things, the framework includes widgets for building (X)HTML interfaces, and provides facilities for creating and working with Local URI namespaces, which are a virtual filesystem-like mechanism for accessing data across multiple applications. It also contains document viewers and a built-in search database (using PostgreSQL), including viewers and indexers for OpenDocument, HTML and plain text files.

[edit] Applications based on the Aukyla Platform

  • The Aukyla Document Management System (ADMS) is a light-weight document management system intended for use in intranet environments.
  • Aukyla SiteManager is a content management system (CMS) for websites.
