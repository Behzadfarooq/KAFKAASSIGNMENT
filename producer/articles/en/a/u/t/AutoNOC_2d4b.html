AutoNOC is a production network management platform for use on Microsoft Windows and Linux systems.

[edit] Features

AutoNOC has a scalable historical database architecture that allows archiving of years of network data. The network crawler can automatically detect and adapt to network and device changes without intervention by the operator. The operations core can poll millions of probes in high density networks.

[edit] Grid Computing

AutoNOC supports a computational model that allows many servers (called grid nodes) to be deployed. These grid nodes work together to form a large internetworked computer that shares operations tasks across all servers.
