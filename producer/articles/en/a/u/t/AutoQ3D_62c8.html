AutoQ3D is an open source OpenGL based software used for 3D modeling.

It is a very capable and easy to use software with many features and tools.

It is written in C++ using Trolltech QT Libraries and is currently available for download for free under a GPL license.

[edit] Software Features

  • Graphics Tools ( UCS icon, Flip Triangles, Zoom, Pan, Rectangle, Circle, Polygon, Move, Rotate, Scale, Line, Move, Stretch, Mirror, Snap To Objects, Revolve, Array)
  • Rendering (Normal Shading, Wireframe, Shade Hide Lines Option)
  • Color and Texture (Multiple materials for any model, Materials can be solid colors or image-map textures, Simple texture coordinate editing, Texture UV)
  • Multiple Image File Formats Support
  • Command line support
  • Supported 3D File Formats. (Import DXF File Format DXF 2004, 2000, R14. Export DXF File Format (DXF 2000). Import MD2 Frames File Format (Quake File Format).
  • Extras. (Predefined views options, UNDO command).
