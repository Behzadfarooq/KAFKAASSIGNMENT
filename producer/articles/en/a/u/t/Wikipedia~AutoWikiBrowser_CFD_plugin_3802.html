WP:AWB/CFD

CFD plugin for AutoWikiBrowser is a tool for speedy recategorisation per results of discussions at WP:CFD and WP:UCFD. It is capable of eliminating a single day's backlog in a single run. It is shipped with AWB starting with version 3.0.6.2.

[edit] Manual

Plugins are included into main AWB distribution archive, however they are placed in the \Plugins subdirectory, so if you need to use a particular plugin, copy it to the main AWB directory.

  1. Open the plugin window by selecting Plugins → Recategorise per CFD main menu item.
  2. Open a section of WP:CFD/W or WP:CFD/WU for editing and paste its contents into the plugin window. The table below will display recategorisations to be made (empty "To" field means that category will be removed. Lines with non-whitespace characters between asterisks and first opening square bracket will be ignored because these characters are "NO BOTS" typically:)
  3. Press OK button. The plugin will load contents of categories and close its window.
  4. Enter an appropriate edit summary on the (3) Start tab of the main window. Please give a link to the day's deletion/recategorisation discussion.
  5. Press Start button.
