This template is designed to replace {{AutoProject}}, and will be used specifically to mark which categories should be included in Wikipedia:WikiProject Automobiles/Articles.

Use suggestions:

  • If an automobile category is created or renamed, please go ahead and tag the new category.
  • DO NOT SUBST this template, because this template's backlinks are the critical data that the bot uses.
  • Some categories do not contain 100% articles related to the automobile wikiproject (for example, Category:Electric vehicles). This is okay, as we have a great many articles, and so a small amount of inaccuracy is acceptable. If a category has <50% automobile-related articles though, the tag may not be welcomed on the category.

A separate template is being used, because at some point, it's likely that we'll need to split the list into two due to wikimedia article size constraints (currently, there are 8000+ links on the page), and we'll need to mark the second topic slightly differently.

Currently, this template is not used for motorsports articles, because of the link-limit issue. Motorsports will be added to another list in the future. --Interiot 01:54, 1 August 2006 (UTC)
