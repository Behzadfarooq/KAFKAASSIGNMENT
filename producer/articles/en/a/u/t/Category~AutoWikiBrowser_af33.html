AutoWikiBrowser is a semi-automated Wikipedia editor designed to make tedious repetitive tasks quicker and easier. It is essentially a browser that automatically opens up a new page when the last is saved. When set to do so, it suggests some changes (typically formatting) which are generally meant to be incidental to the main change.
