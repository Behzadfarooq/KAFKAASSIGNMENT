This template takes the following input:

{{authoronlinesource2007|section={Section}
| author={Author}
| title={Title of press article}
| org={Media outlet/organization}
| url={URL to press article}
| date={Date} }}

[edit] Usage examples

[edit] Basic example

Adding double brackets to the string authoronlinesource2007|author=Joe Blow|date=16 January 2007|url=http://google.com|title=Something Nifty This Way Comes|org=Slashdot|section=January produces:


This page has been cited as a source by a media organization. See the 2007 press source article for details.

The citation is in: Joe Blow. "Something Nifty This Way Comes", Slashdot, 16 January 2007.

[edit] Author unknown example

Alternatively, Template:onlinesource2007 does the same thing except it leaves out the author field in case the author of the press article is unknown.

[edit] Optional unnamed parameter example

An optional unnamed parameter allows you to customize the text describing the citation:


This page has been cited as a source by a media organization. See the 2007 press source article for details.

The citation is in: Joe Blow. "Something Nifty This Way Comes", Slashdot, 16 January 2007.
