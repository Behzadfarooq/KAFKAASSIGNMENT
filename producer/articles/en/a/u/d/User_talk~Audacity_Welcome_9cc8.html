This template is designed as a more personalized replacement to {{welcome}} and its deviants. It should be used when you notice a good contribution to an article by a user or anonymous editor who has not yet been welcome. To use, first create a page here, copying this template and changing the name and userpage link. Then type

{{subst:<the page you created>
|me=<your username>
|type=<editor type (user or anon)>
|article=<article name>
|topic=<general topic>
|portal=<portal name>
|project=<WikiProject name>
}}

changing the appropriate <variables>.
