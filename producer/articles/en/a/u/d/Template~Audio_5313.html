This high-risk template has been protected from editing to prevent vandalism. (protection log). Please discuss changes on the talk page or request unprotection. You may use {{editprotected}} on the talk page to ask for an administrator to make an edit for you.

Purpose

This template links to an audio file for quick listening. It is meant to be used inline with text, for pronunciations and the like.

Usage

{{Audio|name of sound file|text to use as link to soundfile}}

Example:

'''Bordeaux''' ({{Audio|Fr-Bordeaux.ogg|pronunciation}}) is a [[Seaport|port]] city in...

gives this:

Bordeaux (pronunciation (help·info)) is a port city in...

See also

  • {{Audio-nohelp}}, a variant of this template with no help links.
  • {{Audio-IPA}}, a variant of this template to be used with IPA notation.
    • {{Audio-IPA-nohelp}}, a variant with no help links.
  • {{Spoken}}, which adds only the small icon with no links.
  • {{Listen}}, which sets an audio clip off from the main text.
