Need to transfer all this to Wikipedia:WikiProject snooker...

Wikipedia is sadly lacking in articles on snooker, a state of affairs I am trying to redress.

The main article Snooker needs some attention. In particular, there's nothing about how the game is governed and how the ranking system works. Unfortunately, I know little about this, so any pointers would be welcomed!

I'd like to link all snooker articles together somehow, probably something like what has been done for United Kingdom politics. This will enable readers to see easily what snooker resources are available, and navigate between them. I'm planning a list of all snooker-related articles, and a list of all players. Prototype here.

I've expanded and moved the article on Embassy World Snooker Championship to World Snooker Championship, as there was a championship long before Embassy started sponsoring it. Really ought to fix pages which link to Embassy World Snooker Championship. The other major tournaments need a greater mention - a page to list all the major tournaments (ranking and non-ranking), and maybe even a page for each tournament, with a list of past winners.

I've done a page on the world rankings - the history, how they work and past records.

One problem with all this, of course, is that articles, especially player biographies, will require updating after each tournament and season. A list of "continuous updates required" would be useful, which could then be ticked off as completed, so we know where we are - maybe this could go on the main snooker talk page. I wonder how this is done for other sports?

I've created a new article for Pot Black.

An article on maximum snooker break would be nice.

Done an article on Snooker Loopy(!). Might even write a few words on Chas & Dave.

Done dozens of player biographies. See Wikipedia:WikiProject snooker for what's left to do.
