Austal Ships (ASX: ASB) is a shipbuilder located in Henderson, Western Australia. The company was started in 1988 and has since built high-speed craft for commercial, pleasure and defence purposes.

Austal has produced vessels such as:

  • Auto Express 45 Class
  • Auto Express 48 Class
  • Auto Express 56 Class
  • Auto Express 60 Class
  • Auto Express 66 Class
  • Auto Express 68 Class
  • Auto Express 78 Class
  • Auto Express 82 Class
  • Auto Express 81 Class
  • Auto Express 86 Class
  • Auto Express 92 Class
  • Auto Express 101 Class
  • Auto Express 107 Class
  • Auto Express 127 Class trimaran
  • Lake Express (Auto Express 58 Class)
  • Spirit of Ontario I (Auto Express 86 Class)
  • Armidale class patrol boat
  • catamarans for P&O Ferries in Europe
  • Defence ships for the United States and Australia

Austal is listed on the S&P/ASX 300 index[1] of the Australian Stock Exchange.[2]
