Auslogics is a software house that specialize in Utility programs. Created in 1998, Auslogics claimed that they have provided computer optimization software for more than 350,000 users. Auslogics aims on giving their users easy to use, reliable software that keeps computers running smoothly, error free and greatly improving system performance.

[edit] Products

  • AusLogics BoostSpeed, a system tweaking suite trialware.
  • Auslogics Disk Defrag, a disk defragmentation freeware. It concentrates mainly on eliminating file fragments instead off consolidating free space. Hence, it provided a very fast although less comprehensive defragmentation.
  • Auslogics Visual Styler, a GUI tweaking suite trialware.
  • Emergency Recovery, a file recovery trialware.
