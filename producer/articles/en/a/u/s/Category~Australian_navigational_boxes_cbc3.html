This is a collection of Australia-related Navigation templates.

To add a template, use the following code:

<noinclude>
[[Category:Australian navigational boxes|{{PAGENAME}}]]
</noinclude>

If you add this code at the end of the template, please make sure that the opening <noinclude> starts on the same line as the last char of the template code. Experiments revealed that some templates became seriously defunct when that rule was violated.
